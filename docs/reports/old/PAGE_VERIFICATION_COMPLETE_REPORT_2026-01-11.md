## 3. API 검증

### ✅ 제품 검색 API (`/api/products/search`)
- **메서드**: GET
- **필수 파라미터**: `keyword`
- **상태**: 정상 작동
- **검증 결과**:
  ```bash
  # keyword 없음 - 400 에러 (예상됨)
  curl "/api/products/search" → {"error": "Keyword parameter is required"}

  # keyword 있음 - 200 성공
  curl "/api/products/search?keyword=パウチ" → {"success": true, "data": [...]}
  ```

### ✅ 제품 필터 API (`/api/products/filter`)
- **메서드**: POST
- **상태**: **수정 완료** (2026-01-11)
- **수정 내용**:
  - `execute_sql` RPC 함수 의존성 제거
  - Supabase Query Builder로 리팩토링
  - 가격 범위 필터 일시 비활성화 (JSONB 복잡도)
- **검증 결과**:
  ```bash
  # 빈 필터 - 모든 제품 반환
  curl -X POST "/api/products/filter/" -d "{}" → {"success": true, "data": [5 products]}

  # 카테고리 필터 - 1개 제품 반환
  curl -X POST "/api/products/filter/" -d '{"category":"stand_up"}' → {"success": true, "count": 1}
  ```
