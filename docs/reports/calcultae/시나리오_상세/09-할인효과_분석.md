## 9-3. 연간 주문 시뮬레이션

### 시나리오: 월 2,000개 × 12개월 = 24,000개

**옵션 1: 매월 1열 생산 (2,000개 × 12회)**
```
월별 주문: 2,000개 × 545,976엔 = 545,976엔
연간 총액: 545,976엔 × 12개월 = 6,551,712엔
```

**옵션 2: 분기별 2열 생산 (4,000개 × 6회)**
```
회별 주문: 4,000개 × 545,976엔 = 545,976엔
연간 총액: 545,976엔 × 6회 = 3,275,856엔
절감액: 6,551,712엔 - 3,275,856엔 = 3,275,856엔 (50% 절감)
```

**옵션 3: 반기별 2열 생산 (8,000개 × 3회)**
```
회별 주문: 8,000개 × 1,091,952엔 = 1,091,952엔
연간 총액: 1,091,952엔 × 3회 = 3,275,856엔
절감액: 6,551,712엔 - 3,275,856엔 = 3,275,856엔 (50% 절감)
```

**결론**:
- 2열 생산으로 연간 327만 6천엔 절감
- 재고 관리 비용 절감 추가 효과
- 배송비 12회 → 3회로 75% 절감
