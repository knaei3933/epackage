## 9-3. 연간 주문 시뮬레이션

### 시나리오: 월 2,000개 × 12개월 = 24,000개

**옵션 1: 매월 1열 생산 (2,000개 × 12회)**
```
월별 주문: 2,000개 × 343,814엔 = 343,814엔
연간 총액: 343,814엔 × 12개월 = 4,125,768엔
```

**옵션 2: 분기별 2열 생산 (4,000개 × 6회)**
```
회별 주문: 4,000개 × 343,814엔 = 343,814엔
연간 총액: 343,814엔 × 6회 = 2,062,884엔
절감액: 4,125,768엔 - 2,062,884엔 = 2,062,884엔 (50% 절감)
```

**옵션 3: 반기별 2열 생산 (8,000개 × 3회)**
```
회별 주문: 8,000개 × 687,628엔 = 687,628엔
연간 총액: 687,628엔 × 3회 = 2,062,884엔
절감액: 4,125,768엔 - 2,062,884엔 = 2,062,884엔 (50% 절감)
```

**결론**:
- 2열 생산으로 연간 206만 2천엔 절감
- 재고 관리 비용 절감 추가 효과
- 배송비 12회 → 3회로 75% 절감
