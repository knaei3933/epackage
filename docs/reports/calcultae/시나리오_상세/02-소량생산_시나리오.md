# 소량 생산 시나리오

## 2-1. 500개 생산 (평백 80×120mm)

### 1열 생산

```
【500개 1열 생산 계산】

기본 사양:
- 치수: 80×120mm (평백)
- 횡(W): 120mm = 12cm
- 수량: 500개
- 피치: 120mm
- 1m당 생산 수: 8.33개/m

필름 계산:
- 이론 미터: 60m (500개 ÷ 8.33개/m)
- 확보량: 60m (파우치는 최소 확보량 없음, 1m 단위)
- 로스: 400m (고정)
- 총 필름: 460m
- 필름 폭: 281mm → 590mm 원반 (실제 폭: 0.59m)

【원재료비 계산】(4층 구조: PET 12μ + AL 7μ + PET 12μ + LLDPE 90μ)
- PET (12μ): 0.012 × 0.59 × 460 × 1.40 = 4.56kg × 2층 = 9.12kg
- AL (7μ): 0.007 × 0.59 × 460 × 2.71 = 5.15kg
- LLDPE (90μ): 0.090 × 0.59 × 460 × 0.92 = 22.42kg
- 총 중량: 36.69kg

원재료비 = (9.12kg × 2,800원) + (5.15kg × 7,800원) + (22.42kg × 2,800원)
         = 25,536원 + 40,170원 + 62,776원
         = 128,482원

【인쇄비】(항상 1m로 계산)
= 1m × 460m × 475원/m² = 218,500원

【라미네이트비】(AL 있음: 75원/m, 라미 회수 3회)
= 0.59m × 460m × 75원 × 3회 = 61,005원

【슬리터비】
= MAX(30,000원, 460m × 10원) = 30,000원

【파우치 가공비】(평백 12cm, 최소 200,000원)
= 12cm × 0.4원 × 500개 = 2,400원
= MAX(2,400원, 200,000원) = 200,000원

【기초 원가】
= 128,482 + 218,500 + 61,005 + 30,000 + 200,000
= 637,987원

【마진 및 비용】
- 제조자 마진 (40%): 637,987 × 0.4 = 255,195원
- 제조자 가격: 637,987 + 255,195 = 893,182원
- 관세 (5%): 893,182 × 0.05 = 44,659원
- 관세 포함 가격: 937,841원

【배송비】
- 총 중량: 약 37kg → 2박스 (29kg/박스)
- 배송비: 2 × 127,980원 = 255,960원

【최종 판매가】
- 수입원가: 937,841 + 255,960 = 1,193,801원
- 판매자 마진 (20%): 1,193,801 × 0.2 = 238,760원
- 최종 판매가(원화): 1,432,561원
- 최종 판매가(엔화): 1,432,561 × 0.12 = 171,907엔
- 1개당 단가: 344엔
```

### 2열 생산 (1,000개)

```
【1,000개 2열 생산 계산】

기본 사양:
- 치수: 80×120mm (평백)
- 횡(W): 120mm = 12cm
- 수량: 1,000개
- 피치: 120mm
- 1m당 생산 수: 16.67개/m (2열)

필름 계산:
- 이론 미터: 60m (1,000개 ÷ 16.67개/m)
- 확보량: 60m (파우치는 최소 확보량 없음, 1m 단위)
- 로스: 400m (고정)
- 총 필름: 460m
- 필름 폭: 551mm → 590mm 원반 (실제 폭: 0.59m)

【원재료비】(동일: 128,482원)
【인쇄비】(동일: 218,500원)
【라미네이트비】(동일: 61,005원)
【슬리터비】(동일: 30,000원)

【파우치 가공비】(1,000개)
= 12cm × 0.4원 × 1,000개 = 4,800원
= MAX(4,800원, 200,000원) = 200,000원

【기초 원가】(동일)
= 637,987원

【마진 및 비용】(동일)
- 최종 판매가(원화): 1,432,561원
- 최종 판매가(엔화): 171,907엔
- 1개당 단가: 172엔

비교:
- 1열 500개: 344엔/개
- 2열 1,000개: 172엔/개 (50% 절감)
```
