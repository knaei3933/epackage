# 01. 계산 예시: 평백 (Flat Pouch)

> **대상**: 실제 제품의 완전한 원가 계산 예시 (기본형)
> **참고**: 원본 07-계산_예시.md의 번역 및 업데이트 버전

---

## 제품 정보

| 항목 | 값 |
|------|-----|
| **제품 타입** | 평백 (3방) |
| **사이즈** | H=160mm × W=100mm |
| **수량** | 10,000개 |
| **필름 구성** | PET 12μ + AL 7μ + PET 12μ + LLDPE 90μ (4층) |
| **인쇄** | 디지털 인쇄 (도수 무관) |
| **후가공** | 라미네이트, 슬리터, 제대 |
| **포장** | 골판지 박스 (29kg 이하) |

---

## 1. 열수 판정 (Columns)

```
1열 필름 폭 = (160 × 2) + 41 = 361mm
2열 필름 폭 = (160 × 4) + 71 = 711mm

판정: 711mm ≤ 740mm → **2열 채택 (2-up)**
사용 원반: 760mm 원반
유효 폭: 740mm
```

---

## 2. 미터수 계산

| 항목 | 계산식 | 결과 |
|------|--------|------|
| **피치(Pitch)** | W = 100mm | 평백은 폭(W) 방향으로 진행 |
| **1m당 생산** | 1,000mm ÷ 100mm × 2열 = 20개 | - |
| **이론 미터** | 10,000개 ÷ 20개/m = 500m | - |
| **최소 확보** | 500m (1SKU 최소량) | - |
| **로스** | 400m (고정) | - |
| **실제 발주** | 500m + 400m = **900m** | - |

---

## 3. 원재료비 계산 (760mm 원반)

**중요**: 4층 구조(PET 12 + AL 7 + PET 12 + LLDPE 90)로 PET가 2회 사용됨.

| 재질 | 두께 | 폭 | 길이 | 비중 | 중량(kg) | 단가(원/kg) | 금액(원) |
|------|------|----|------|------|----------|-------------|----------|
| PET (1) | 12μ | 0.76 | 900 | 1.40 | 11.5 | 2,800 | 32,200 |
| AL | 7μ | 0.76 | 900 | 2.71 | 13.0 | 7,800 | 101,400 |
| PET (2) | 12μ | 0.76 | 900 | 1.40 | 11.5 | 2,800 | 32,200 |
| LLDPE | 90μ | 0.76 | 900 | 0.92 | 56.6 | 2,800 | 158,480 |
| **합계** | - | - | - | - | **92.6kg** | - | **324,280원** |

---

## 4. 인쇄 및 가공비

### 4-1. 인쇄비
- **기준**: 항상 1m 폭으로 계산 (760mm 원반이어도 1m 비용 청구)
- 1m × 900m × 475원 = **427,500원**

### 4-2. 라미네이트비 (AL 포함 75원)
- 0.76m × 900m × 75원 × 3회 = **153,900원**

### 4-3. 슬리터비
- MAX(30,000, 900m × 10원) = **30,000원**

### 4-4. 제대비 (평백 기본)
- W 10cm × 0.4원 × 10,000개 = 40,000원
- 최소 비용 적용 = **200,000원**

### 4-5. 기초 원가 합계
- 324,280 + 427,500 + 153,900 + 30,000 + 200,000 = **1,135,680원**

---

## 5. 최종 가격 도출

| 단계 | 항목 | 계산 | 금액(원) |
|------|------|------|----------|
| **1** | **기초 원가** | - | 1,135,680 |
| **2** | **제조 마진** | 40% | 454,272 |
| **3** | **제조가** | ① + ② | 1,589,952 |
| **4** | **관세** | 5% | 79,498 |
| **5** | **배송비** | 92.6kg → 4박스 | 511,920 |
| **6** | **수입 원가** | ③ + ④ + ⑤ | 2,181,370 |
| **7** | **판매 마진** | 20% | 436,274 |
| **8** | **최종 판가** | ⑥ + ⑦ | **2,617,644** |
| **9** | **엔화 환산** | × 0.12 | **314,117엔** |
| **10** | **개당 단가** | ÷ 10,000 | **31.4엔** |

---

## 6. 요약

- **원반 효율**: 760mm 원반에서 711mm 사용 (93.5% 효율)
- **생산 효율**: 2열 생산으로 인쇄/라미 거리 단축
- **최종 단가**: 개당 **31.4엔** (매우 경쟁력 있음)
