# 대량 생산 시나리오

## 4-1. 50,000개 생산 (평백 120×180mm)

### 1열 생산

```
【50,000개 1열 생산 계산】

기본 사양:
- 치수: 120×180mm (평백)
- 횡(W): 180mm = 18cm
- 수량: 50,000개
- 피치: 180mm
- 1m당 생산 수: 5.56개/m

필름 계산:
- 이론 미터: 9,000m (50,000개 ÷ 5.56개/m)
- 확보량: 9,000m
- 로스: 400m
- 총 필름: 9,400m
- 필름 폭: 421mm → 590mm 원반 (불가) → 760mm 원반 (실제 폭: 0.76m)

【원재료비】(4층 구조, 9,400m, 0.76m 폭)
- PET (12μ): 0.012 × 0.76 × 9,400 × 1.40 = 120.16kg × 2층 = 240.32kg
- AL (7μ): 0.007 × 0.76 × 9,400 × 2.71 = 135.38kg
- LLDPE (90μ): 0.090 × 0.76 × 9,400 × 0.92 = 589.10kg
- 총 중량: 964.80kg

원재료비 = (240.32kg × 2,800원) + (135.38kg × 7,800원) + (589.10kg × 2,800원)
         = 672,896원 + 1,055,964원 + 1,649,480원
         = 3,378,340원

【인쇄비】
= 1m × 9,400m × 475원/m² = 4,465,000원

【라미네이트비】
= 0.76m × 9,400m × 75원 × 3회 = 1,607,400원

【슬리터비】
= MAX(30,000원, 9,400m × 10원) = 94,000원

【파우치 가공비】(평백 18cm, 50,000개)
= 18cm × 0.4원 × 50,000개 = 360,000원
= MAX(360,000원, 200,000원) = 360,000원

【기초 원가】
= 3,378,340 + 4,465,000 + 1,607,400 + 94,000 + 360,000
= 9,904,740원

【마진 및 비용】
- 제조자 마진 (40%): 3,961,896원
- 제조자 가격: 13,866,636원
- 관세 (5%): 693,332원
- 관세 포함 가격: 14,559,968원

【배송비】
- 총 중량: 약 965kg → 34박스
- 배송비: 34 × 127,980원 = 4,351,320원

【최종 판매가】
- 수입원가: 14,559,968 + 4,351,320 = 18,911,288원
- 판매자 마진 (20%): 3,782,258원
- 최종 판매가(원화): 22,693,546원
- 최종 판매가(엔화): 2,723,226엔
- 1개당 단가: 54엔
```

### 2열 생산 (100,000개)

```
【100,000개 2열 생산 계산】

기본 사양:
- 치수: 120×180mm (평백)
- 횡(W): 180mm = 18cm
- 수량: 100,000개
- 피치: 180mm
- 1m당 생산 수: 11.11개/m (2열)

필름 계산:
- 이론 미터: 9,000m (100,000개 ÷ 11.11개/m)
- 확보량: 9,000m
- 로스: 400m
- 총 필름: 9,400m
- 필름 폭: 831mm → 760mm 원반 초과 → **2열 생산 불가**

⚠️ 주의사항:
120×180mm 평백은 2열 필름 폭이 831mm로
760mm 원반을 초과하여 2열 생산이 불가합니다.

대안:
- 1열 생산으로 진행
- 여러 종류의 SKU를 그룹화하여 로스 공유
- 각 그룹별 최소 300m 확보
```

## 4-2. 대량 생산 비교표

| 수량 | 1열 총액(엔) | 1열 단가(엔) | 2열 가능 여부 | 비고 |
|------|-------------|-------------|-------------|------|
| 50,000개 | 2,723,226엔 | 54엔 | ❌ 불가 | 760mm 원반 초과 |
| 100,000개 | 5,446,452엔 | 54엔 | ❌ 불가 | 760mm 원반 초과 |

**대량 생산 시 추천**:
- 2열 생산 불가 시 1열 생산으로 진행
- 여러 SKU를 그룹화하여 로스 공유로 효율화
- 주문량 50,000개 이상 시 별도 협상
