# 박스파우치 시나리오 (Box Pouch)

## 6-1. 5,000개 3종 합계 15,000개 생산 (박스 100×200×60mm)

박스파우치는 구조상 2열 생산이 불가능합니다. 따라서 **'멀티롤(Multi-roll) 병렬 생산'**을 통해 원가를 절감해야 합니다.

**기본 조건**:
- 제품 규격: 100(W)×200(H)×60(G)mm
- 주문 수량: A, B, C 디자인 각 5,000개 (총 15,000개)
- 필름 구조: 4층 구조

### 1. 생산 방식 비교

| 구분 | 개별 생산 (비효율) | 멀티롤 병렬 생산 (추천) | 비고 |
|-----|-------------------|----------------------|------|
| **단위 필름 폭** | 352mm | 352mm | 공식: (G+W)×2+32 |
| **사용 원반** | **590mm** × 3회 | **760mm** × 1회 | 590mm에 1개(352mm) vs 760mm에 2개(704mm) |
| **병렬 본수** | 1본 (1-up) | 2본 (2-up) | 760mm ÷ 352mm = 2.1 → 2본 가능 |
| **총 생산 거리** | 3,600m + 로스 1,200m | 1,200m + 로스 400m | 3종을 2열로 배치하여 생산 |
| **원단 효율** | 60% (352/590) | 93% (704/760) | **원단 낭비 최소화** |

### 2. 상세 원가 계산 (총 15,000개 기준)

| 항목 | 개별 생산 (총 4,800m) | 멀티롤 생산 (총 1,600m) | 절감 내역 |
|------|---------------------|-----------------------|----------|
| **① 원재료비** | **1,341,336원** | **572,773원** | 원단 효율 60%→93% 상승 + 로스 공유로 **57% 절감** |
| **② 인쇄비** | **2,280,000원** | **760,000원** | 3회 세팅 → 1회 통합으로 **67% 절감** |
| **③ 라미네이트** | 637,200원 | 273,600원 | - |
| **④ 슬리터** | 90,000원 | 30,000원 | - |
| **⑤ 제대비** | 1,320,000원 | 1,320,000원 | 개당 단가는 동일 (44만원 × 3종) |
| **기초 원가 합계** | **5,668,536원** | **2,956,373원** | **약 48% 절감** |

> **원가 절감 핵심**:
> 1.  **인쇄비**: 디자인 3종을 각각 찍으면 4800m를 찍어야 하지만, 2열로 묶어서 찍으면 거리가 대폭 단축됨
> 2.  **원단비**: 590mm 원반을 쓸 때 버려지는 238mm 폭을, 760mm 원반에서 2개를 뽑으며 최소화(56mm)

### 3. 최종 납품가

| 항목 | 개별 생산 (단가) | 멀티롤 생산 (단가) | 비고 |
|------|-----------------|-------------------|------|
| **기초 원가** | 5,668,536원 | 2,956,373원 | - |
| **최종 판매가** | **12,289,386원** | **6,409,417원** | 마진, 관세, 배송비 포함 |
| **엔화 단가** | **98.3엔/개** | **51.3엔/개** | **약 48% 인하 효과** |

> **제안**: "3가지 맛을 각각 하시면 개당 98엔이지만, 묶어서 하시면 개당 51엔에 가능합니다."
