## 1. 理論メートル数計算

### 1-1. 基本公式

```
1mあたり生産可能数 = 1,000mm ÷ ピッチ（またはフィルム長/mm）

理論メートル数 = 数量 ÷ 1mあたり生産可能数
```

### 1-2. 計算例

**例**: 120×180mmパウチ、注文数量500個

```
ピッチ = 120mm（横方向）
1mあたり生産可能数 = 1,000 ÷ 120 = 8.33個/m
理論メートル数 = 500 ÷ 8.33 = 60m
```

## 3. 実際発注量決定

### 3-1. 商品タイプ別ルール

> **重要**: ロスは**同じピッチ（パウチ）または同じ幅（ロール）のSKUグループごとに400m固定**

#### パウチ商品

| 項目 | 設定値 |
|------|--------|
| **最小確保量** | **なし** |
| **切り上げ単位** | 1m単位 |
| **ロス** | 同じピッチのSKUグループごとに400m |

#### ロールフィルム商品

| 項目 | 設定値 |
|------|--------|
| **最小確保量（1SKU）** | 500m |
| **最小確保量（2+SKU）** | 各300m |
| **切り上げ単位** | 50m単位 |
| **ロス** | 同じ幅のSKUグループごとに400m |

### 3-2. 確保量決定関数

#### パウチ商品

```
// 同じピッチのSKUをグループ化
FOR EACH ピッチグループ DO
  // 理論メートル数を合計
  理論メートル数合計 = Σ(各SKU数量 ÷ (1000mm ÷ ピッチ))

  // 1m単位で切り上げ（最小確保量なし）
  確保量 = CEILING(理論メートル数合計, 1m)

  // ロス追加（グループごとに400m）
  グループ総発注量 = 確保量 + 400m
END FOR
```

#### ロールフィルム商品

```
// 同じ幅のSKUをグループ化
FOR EACH 幅グループ DO
  // 理論メートル数を合計
  理論メートル数合計 = Σ(各SKU数量 ÷ (1000mm ÷ ピッチ))

  // 最小確保量適用
  IF SKU数 = 1 THEN
    最小確保量 = 500m
  ELSE
    最小確保量 = 300m × SKU数
  END IF

  // 確保量決定（50m単位で切り上げ）
  IF 理論メートル数合計 ≤ 最小確保量 THEN
    確保量 = 最小確保量
  ELSE
    確保量 = CEILING(理論メートル数合計, 50m)
  END IF

  // ロス追加（グループごとに400m）
  グループ総発注量 = 確保量 + 400m
END FOR
```

### 3-3. 計算例

#### ケース1: パウチ 1SKU、注文500個

| 項目 | 計算 | 結果 |
|------|------|------|
| ピッチ | W = 120mm | - |
| 1mあたり生産数 | 1000 ÷ 120 | 8.33個/m |
| 理論メートル数 | 500 ÷ 8.33 | 60m |
| 確保量 | CEILING(60, 1) | **60m**（最小制限なし） |
| ロス | 同一ピッチグループ | **400m** |
| **総発注量** | 60 + 400 | **460m** |

#### ケース2: パウチ 2SKU（同じピッチ）、各500個

| 項目 | SKU 1 | SKU 2 | 合計 |
|------|-------|-------|------|
| 数量 | 500個 | 500個 | 1000個 |
| ピッチ | 120mm | 120mm | 同一グループ |
| 理論メートル数 | 60m | 60m | 120m |
| 確保量 | - | - | **120m** |
| ロス | - | - | **400m**（グループごと） |
| **総発注量** | - | - | **520m** |

#### ケース3: ロールフィルム 1SKU、注文500m

| 項目 | 計算 | 結果 |
|------|------|------|
| 注文メートル数 | 500m | - |
| 最小確保量 | 1SKU | 500m |
| 確保量 | MAX(500, 500) | **500m** |
| ロス | 固定 | **400m** |
| **総発注量** | 500 + 400 | **900m** |

#### ケース4: ロールフィルム 2SKU（同じ幅）、各500m

| 項目 | SKU 1 | SKU 2 | 合計 |
|------|-------|-------|------|
| 注文メートル数 | 500m | 500m | 1000m |
| 各最小確保量 | 300m | 300m | 600m |
| 確保量 | - | - | **600m** |
| ロス | - | - | **400m**（グループごと） |
| **総発注量** | - | - | **1000m** |

## 4. 原材原価計算

### 4-1. 原反重量計算

```
原反重量(g) = 厚み(μm) ÷ 1,000 × 幅(m) × メートル数 × 比重
```

> **重要**: 厚さはミクロン(μm)で、60μm = 0.060mm

### 4-2. 重量計算例（LLDPE 90μm、幅0.76m、メートル数900m）

```
LLDPE重量 = 90 ÷ 1,000 × 0.76m × 900m × 0.92
         = 0.090 × 0.76 × 900 × 0.92
         = 56.61kg
```

### 4-3. 原反価格計算

```
原反価格(ウォン) = 原反重量(kg) × 単価(ウォン/kg)
```
