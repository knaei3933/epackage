## 1. 理論メートル数計算

### 1-1. 基本公式

```
1mあたり生産可能数 = 1,000mm ÷ ピッチ（またはフィルム長/mm）

理論メートル数 = 数量 ÷ 1mあたり生産可能数
```

### 1-2. 計算例

**例**: 120×180mmパウチ、注文数量500個

```
ピッチ = 120mm（横方向）
1mあたり生産可能数 = 1,000 ÷ 120 = 8.33個/m
理論メートル数 = 500 ÷ 8.33 = 60m
```

## 2. 各パウチタイプのピッチ

### 2-1. ピッチ一覧（2026-03-07訂正）

| パウチタイプ | ピッチ | 説明 |
|-------------|----------|------|
| 平袋（三封） | **W** | 横方向が進行方向 |
| スタンドパウチ | **W** | 横方向が進行方向 |
| 合掌袋（T封） | **H** | **縦方向が進行方向**（訂正） |
| ガゼットパウチ（M封） | **H** | **縦方向が進行方向**（訂正） |
| スパウトパウチ | **W** | スタンドパウチ準用 |

> **重要（2026-03-07訂正）**: 合掌袋（T封）とガゼットパウチ（M封）は展開図基準で生産するため、パウチの**長さ（H）**が進行方向（ピッチ）になります。

### 2-2. ピッチ計算例

| パウチタイプ | サイズ | ピッチ | 1mあたり生産数 |
|-------------|--------|----------|----------------|
| 平袋 | 100×120mm | 100mm | 10.0個/m |
| 平袋 | 120×180mm | 120mm | 8.33個/m |
| 平袋 | 160×200mm | 160mm | 6.25個/m |
| スタンド | 130×130mm (G=30) | 130mm | 7.69個/m |
| **合掌袋** | W=60×H=150mm | **150mm** | 6.67個/m |
| **ガゼット** | W=50×H=100×G=90 | **100mm** | 10.0個/m |

### 2-3. 展開図基準パウチの詳細

展開図基準で生産する合掌袋（T封）とガゼットパウチ（M封）は、パウチの縦方向（H）が進行方向になります。

```
展開図のレイアウト:
┌────────────────────────┐
│  ┌──┐  ┌──┐  ┌──┐     │ ← 進行方向（ピッチ = H）
│  └──┘  └──┘  └──┘     │
│    W                   │
└────────────────────────┘
     ↑
   フィルム幅 = W または W+G
```

> **重要**: 合掌袋とガゼットパウチのメートル数計算時は、H（縦）をピッチとして使用します。

## 3. 実際発注量決定

### 3-1. 商品タイプ別ルール

> **重要**: ロスは**同じピッチ（パウチ）または同じ幅（ロール）のSKUグループごとに素材別ロス固定**

#### パウチ商品

| 項目 | 設定値 |
|------|--------|
| **最小確保量** | **なし** |
| **切り上げ単位** | 1m単位 |
| **ロス** | **AL/クラフト素材**: 400m / **それ以外**: 300m |

#### ロールフィルム商品

| 項目 | 設定値 |
|------|--------|
| **最小確保量（通常）** | 500m |
| **最小確保量（クラフト）** | **1000m** |
| **最小確保量（2+SKU）** | 各300m |
| **切り上げ単位** | 50m単位 |
| **ロス** | **AL/クラフト素材**: 400m / **それ以外**: 300m |

> **重要（2026-03-07追加）**: クラフト材料を含む場合、最小確保量は1000m

### 3-2. 確保量決定関数

#### パウチ商品

```
// 同じピッチのSKUをグループ化
FOR EACH ピッチグループ DO
  // 理論メートル数を合計
  理論メートル数合計 = Σ(各SKU数量 ÷ (1000mm ÷ ピッチ))

  // 1m単位で切り上げ（最小確保量なし）
  確保量 = CEILING(理論メートル数合計, 1m)

  // 素材判定（ALまたはクラフトを含むか）
  hasALorKraft = (素材構造に 'AL' または 'KRAFT' が含まれる)

  // ロス追加（グループごとに素材別ロス）
  IF hasALorKraft THEN
    グループ総発注量 = 確保量 + 400m
  ELSE
    グループ総発注量 = 確保量 + 300m
  END IF
END FOR
```

#### ロールフィルム商品

```
// 同じ幅のSKUをグループ化
FOR EACH 幅グループ DO
  // 理論メートル数を合計
  理論メートル数合計 = Σ(各SKU数量 ÷ (1000mm ÷ ピッチ))

  // クラフト材料判定
  usesKraft = 材料にKRAFTが含まれる

  // 最小確保量適用
  IF usesKraft THEN
    最小確保量 = 1000m
  ELSE IF SKU数 = 1 THEN
    最小確保量 = 500m
  ELSE
    最小確保量 = 300m × SKU数
  END IF

  // 確保量決定（50m単位で切り上げ）
  IF 理論メートル数合計 ≤ 最小確保量 THEN
    確保量 = 最小確保量
  ELSE
    確保量 = CEILING(理論メートル数合計, 50m)
  END IF

  // ロス追加（グループごとに400m）
  グループ総発注量 = 確保量 + 400m
END FOR
```

## 4. 原材原価計算

### 4-1. 原反重量計算

```
原反重量(g) = 厚み(μm) ÷ 1,000 × 幅(m) × メートル数 × 比重
```

> **重要**: 厚さはミクロン(μm)で、60μm = 0.060mm

### 4-2. 重量計算例（LLDPE 90μm、幅0.76m、メートル数900m）

```
LLDPE重量 = 90 ÷ 1,000 × 0.76m × 900m × 0.92
         = 0.090 × 0.76 × 900 × 0.92
         = 56.61kg
```

### 4-3. 原反価格計算

```
原反価格(ウォン) = 原反重量(kg) × 単価(ウォン/kg)
```

| **合計** | - | - | - | - | - | **55.89kg** | **187,510ウォン** |

> **MOQ**: 500M（通常材料扱い）

### 6-2. Kraft+PET+LLDPE 構造

**構成**: KRAFT 50g/m² + PET 12μ + LLDPE（50/70/90/100/110μ）

#### グラメールから厚さへの変換

クラフト紙は「グラメール（g/m²）」単位ですが、重量計算時には直接グラメール値を使用します。

**正確な重量計算式**:
```
原反重量(kg) = (グラメール_g/m² × 幅_m × メートル数) ÷ 1,000
```

> **重要**: クラフト紙の計算では、比重を含めたグラメール値（g/m²）を直接使用します。これはすでに密度を考慮した単位です。

#### 計算例（KRAFT 50g/m² + PET 12μ + LLDPE 70μ、幅0.76m、メートル数1000m）

| 材料 | 厚さ/グラメール | 幅(m) | 長さ(m) | 比重 | 単価(ウォン/kg) | 重量(kg) | 価格(ウォン) |
|------|---------------|--------|--------|------|----------------|----------|-------------|
| KRAFT | 50g/m² | 0.76 | 1000 | - | 10,000 | **38.00** | 380,000 |
| PET | 12μm | 0.76 | 1000 | 1.40 | 2,800 | 12.77 | 35,756 |
| LLDPE | 70μm | 0.76 | 1000 | 0.92 | 2,800 | 48.85 | 136,780 |
| **合計** | - | - | - | - | - | **99.62kg** | **552,536ウォン** |

**クラフト紙重量の詳細計算**:
```
KRAFT重量 = (50g/m² × 0.76m × 1000m) ÷ 1,000
          = 38,000g ÷ 1,000
          = 38.00kg
```

> **MOQ**: 1000M（クラフト材料特別）

### 6-3. グラメール材料の計算式まとめ

#### プラスチック材料（PET, AL, NY, LLDPEなど）
```
重量(kg) = 厚さ(μm) ÷ 1,000 × 幅(m) × メートル数 × 比重
```

#### クラフト紙（KRAFT）
```
重量(kg) = グラメール(g/m²) × 幅(m) × メートル数 ÷ 1,000
```

> **重要**: クラフト紙はグラメール（g/m²）が「そのまま重量密度」として機能するため、比重を乗算する必要がありません。

### 6-4. 新材料の価格比較

| 構造 | 総重量(kg/1000m) | 総価格(ウォン/1000m) | 単価(ウォン/m) |
|------|-----------------|---------------------|---------------|
| PET+AL+PET+LLDPE | 143.83kg | 503,974ウォン | 504ウォン |
| **NY+LLDPE** | **55.89kg** | **187,510ウォン** | **188ウォン** |
| **Kraft+PET+LLDPE** | **99.62kg** | **552,536ウォン** | **553ウォン** |

> **観察**: NY+LLDPEは最も安価ですが、アルミなしのためバリア性能が低いです。Kraft+PET+LLDPEは自然素材のプレミアム感がありますが、単価は高めです。
