## 1. 印刷費用

### 1-1. 基本印刷料

```
印刷費用(ウォン) = 1m × 使用メートル数 × 475ウォン/m²
```

> **最も重要**: 印刷費は**フィルム幅に関係なく、常に1m**で計算します！

### 1-2. 計算例

| メートル数 | 計算式 | 印刷費 |
|-----------|--------|--------|
| 900m | 1m × 900m × 475ウォン | 427,500ウォン |
| 1,000m | 1m × 1,000m × 475ウォン | 475,000ウォン |
| 1,400m | 1m × 1,400m × 475ウォン | 665,000ウォン |

### 1-3. マット印刷（マット仕上げ）

マット仕上げを選択すると、印刷フィルム幅に合わせて**別途マット印刷**を行う必要があります。

#### 計算式

```
マット印刷追加費(ウォン) = フィルム幅(m) × 使用メートル数 × 40ウォン/m
```

> **重要**: マット印刷はフィルム幅に比例して費用がかかります。

#### 計算例

| フィルム幅 | メートル数 | 計算式 | 追加費 |
|-----------|-----------|--------|--------|
| 0.361m | 900m | 0.361 × 900 × 40 | 12,996ウォン |
| 0.711m | 1,000m | 0.711 × 1,000 × 40 | 28,440ウォン |
| 0.760m | 1,400m | 0.76 × 1,400 × 40 | 42,560ウォン |

## 2. ラミネート加工費

### 2-1. 基本公式

```
ラミネート費(ウォン) = 実際使用幅(m) × 使用メートル数 × ラミ単価 × ラミ回数
```

### 2-2. ラミ単価（素材別）

| 条件 | 単価 |
|------|------|
| **AL素材あり** | 75ウォン/m |
| **AL素材なし** | 65ウォン/m |

### 2-3. ラミ回数決定ルール

```
ラミ回数 = 層数 - 1
```

| 層数 | ラミ回数 | 説明 |
|------|----------|------|
| 1層 | 0回 | ラミネートなし |
| 2層 | 1回 | PET + LLDPE など |
| 3層 | 2回 | PET + NY + LLDPE など |
| 4層 | 3回 | PET + AL + PET + LLDPE など |
| 5層 | 4回 | 複雑な多層構造 |

### 2-4. 計算例

#### ケース1: 4層構造、ALあり、幅0.76m、メートル数1,400m

```
ラミ回数 = 4層 - 1 = 3回
ラミネート費 = 0.76m × 1,400m × 75ウォン × 3回
            = 239,400ウォン
```

#### ケース2: 3層構造、ALなし、幅0.59m、メートル数900m

```
ラミ回数 = 3層 - 1 = 2回
ラミネート費 = 0.59m × 900m × 65ウォン × 2回
            = 69,030ウォン
```

## 3. スリッター加工費

### 3-1. 基本公式

```
スリッター費(ウォン) = MAX(30,000ウォン, 使用メートル数 × 10ウォン)
```

### 3-2. 計算例

| メートル数 | 計算式 | スリッター費 |
|-----------|--------|-------------|
| 900m | MAX(30,000, 900 × 10) | **30,000ウォン**（最小単価適用） |
| 1,000m | MAX(30,000, 1,000 × 10) | **30,000ウォン**（最小単価適用） |
| 4,000m | MAX(30,000, 4,000 × 10) | **40,000ウォン** |
| 10,000m | MAX(30,000, 10,000 × 10) | **100,000ウォン** |

> **注意**: 3,000m以下の場合は最小単価30,000ウォンが適用されます。

## 4. 製袋加工費

### 4-1. 基本公式

```
基本加工費(ウォン) = パウチ横幅(cm) × 単価(ウォン/cm) × 数量
製袋加工費(ウォン) = MAX(基本加工費, 最小単価)
```

### 4-2. パウチタイプ別単価と最小単価

| パウチタイプ | 単価(ウォン/cm) | 最小単価（ウォン） |
|--------------|-----------------|-------------------|
| **平袋（三封）** | 0.4ウォン | 200,000ウォン |
| **合掌袋（T封）** | 1.2ウォン | 440,000ウォン |
| **スタンドパウチ** | 1.2ウォン | 250,000ウォン |
| **平袋（ジッパー）** | 1.2ウォン | 250,000ウォン |
| **ジッパー付きスタンド** | 1.7ウォン | 280,000ウォン |
| **ガゼットパウチ（M封）** | 1.2ウォン | 440,000ウォン |

> **重要**: 合掌袋（T封）とガゼットパウチ（M封）は外注処理が必要なため、**別途往復配送料150,000ウォン**がかかります。（加工費とは別に計算）

### 4-3. 計算例

#### ケース1: 平袋 10cm、10,000個

```
基本加工費 = 10cm × 0.4ウォン × 10,000個 = 40,000ウォン
最小単価 = 200,000ウォン
製袋加工費 = MAX(40,000, 200,000) = 200,000ウォン
```

#### ケース2: スタンドパウチ 13cm、5,000個

```
基本加工費 = 13cm × 1.2ウォン × 5,000個 = 78,000ウォン
最小単価 = 250,000ウォン
製袋加工費 = MAX(78,000, 250,000) = 250,000ウォン
```

#### ケース3: 平袋 10cm、100,000個

```
基本加工費 = 10cm × 0.4ウォン × 100,000個 = 400,000ウォン
最小単価 = 200,000ウォン
製袋加工費 = MAX(400,000, 200,000) = 400,000ウォン
```

## 5. 後加工費合計

### 5-1. 後加工費の構成

```
後加工費 = ラミネート費 + スリッター費 + 製袋加工費 + オプション費
```

### 5-2. 計算例

#### マットなしの場合

| 項目 | 金額 |
|------|------|
| ラミネート費 | 239,400ウォン |
| スリッター費 | 30,000ウォン |
| 製袋加工費 | 200,000ウォン |
| **合計（マットなし）** | **469,400ウォン** |

#### マットありの場合

| 項目 | 金額 |
|------|------|
| ラミネート費 | 239,400ウォン |
| スリッター費 | 30,000ウォン |
| 製袋加工費 | 200,000ウォン |
| マット印刷追加費 | 42,560ウォン |
| **合計（マットあり）** | **511,960ウォン** |

## 6. 重要ポイントのまとめ

| 項目 | 設定値 | 説明 |
|------|--------|------|
| **印刷費** | 常に **1m**で計算 | フィルム幅と無関係 |
| **印刷単価** | 475ウォン/m² | 固定単価 |
| **マット印刷追加費** | 40ウォン/m | フィルム幅比例 |
| **ラミ単価（ALあり）** | 75ウォン/m | AL素材使用時 |
| **ラミ単価（ALなし）** | 65ウォン/m | AL素材なしの場合 |
| **ラミ回数** | 層数 - 1 | 4層 = 3回 |
| **スリッター最小単価** | 30,000ウォン | 3,000m以下で適用 |
| **平袋製袋単価** | 0.4ウォン/cm | 最小200,000ウォン |
| **平袋（ジッパー）製袋単価** | 1.2ウォン/cm | 最小250,000ウォン |
| **ジッパー付きスタンド単価** | 1.7ウォン/cm | 最小280,000ウォン |

## 6. スパウトパウチ加工費

### 6-1. 基本公式

スパウトパウチは特殊な後加工が必要で、スパウト（キャップ）の取り付けに特別なコストがかかります。

```
スパウト加工費(ウォン) = (スパウト単価 × 数量) + 往復配送料
```

> **重要**: スパウトパウチの**最小注文数量は5,000個**です。

### 6-2. スパウトサイズ別単価

| スパウトサイズ | 単価（ウォン/個） | 説明 |
|--------------|-----------------|------|
| **9パイ（φ9mm）** | 70ウォン | 小型スパウト |
| **15パイ（φ15mm）** | 80ウォン | 標準小型スパウト |
| **18パイ（φ18mm）** | 110ウォン | 標準スパウト |
| **22パイ（φ22mm）** | 130ウォン | 大型スパウト |
| **28パイ（φ28mm）** | 200ウォン | 超大型スパウト |

> **注**: スクリュータイプのスパウトで、サイズ以外の構造は同じです。顧客がサイズを選択できます。

### 6-3. 往復配送料

| 項目 | 金額 |
|------|------|
| **往復配送料** | **150,000ウォン**（固定） |

> **重要**: スパウトパウチも外注処理のため、別途往復配送料がかかります。

### 6-4. 計算例

#### ケース1: 9パイスパウト、最小数量5,000個

```
スパウト単価 = 70ウォン
数量 = 5,000個（最小数量）
往復配送料 = 150,000ウォン

スパウト加工費 = (70ウォン × 5,000個) + 150,000ウォン
             = 350,000ウォン + 150,000ウォン
             = 500,000ウォン
```

#### ケース2: 15パイスパウト、10,000個

```
スパウト単価 = 80ウォン
数量 = 10,000個
往復配送料 = 150,000ウォン

スパウト加工費 = (80ウォン × 10,000個) + 150,000ウォン
             = 800,000ウォン + 150,000ウォン
             = 950,000ウォン
```

#### ケース3: 28パイスパウト、5,000個

```
スパウト単価 = 200ウォン
数量 = 5,000個
往復配送料 = 150,000ウォン

スパウト加工費 = (200ウォン × 5,000個) + 150,000ウォン
             = 1,000,000ウォン + 150,000ウォン
             = 1,150,000ウォン
```

### 6-5. スパウトパウチの原価構成

スパウトパウチの最終原価は以下の要素で構成されます：

```
スパウトパウチ原価 = 原材料費 + 印刷費 + ラミネート費 + スリッター費
                  + パウチ加工費（基本）+ マット印刷費（オプション）
                  + スパウト加工費 + 往復配送料
```

> **重要**: スパウト加工費と往復配送料は、通常のパウチ加工費とは**別に計算**されます。
