## 1. 가격 정책 기본 원칙

### 1-1. 마진 구조 유지

```
【기존 마진 구조(수입원가 20% 마진 기준)】

(기초원가 × 1.4 × 1.05 + 배송비) × 1.2 × 0.12
    (제조자 40%)   (관세 5%)    (마진 20%)     (환율)

수입원가 = 기초원가 × 1.4 × 1.05 + 배송비
최종 판매가 = 수입원가 × 1.2 × 0.12

→ 수입원가에서 최소 20% 마진을 유지하며 병렬 생산 할인은 제조자 마진에서 흡수
```

### 1-2. 병렬 생산 할인의 적용 범위

```
【할인율 결정의 3가지 원칙】

원칙1: 효율 90% 이상 → 40% OFF (최우)
원칙2: 효율 80-89% → 30-35% OFF (우수)
원칙3: 효율 70-79% → 20% OFF (표준)
원칙4: 효율 60-69% → 할인 없음 (비추천)

→ 단순히 병렬 수가 많다고 할인하는 것이 아니라, 필름 폭 활용 효율을 고려하여 적용
```

## 3. 가격 책정의 핵심 로직

### 3-1. 1열 vs 2열 생산 가격 비교

```
【평Bag 80mm × 120mm × 10,000개 분석】

1열 생산:
  필름 폭: 201mm (1열) → 590mm 원반 사용 가능
  총 필름: 1,350m (950m + 400m 로스)
  기초원가: ¥806,150
  최종 판매가: ¥158,270 (단가 ¥15.83/개)

2열 생산:
  필름 폭: 401mm (2열) → 590mm 원반 불가, 760mm 사용
  총 필름: 900m (500m + 400m 로스)
  기초원가: ¥487,125 (1열 대비 40% 감소)
  최종 판매가: ¥110,315 (단가 ¥11.03/개)

할인율: 38% OFF
수입원가 감소: ¥1,482,049 → ¥919,289 (38% 감소)
```

### 3-2. 병렬 생산 유무 조건

```
【2열 생산 추천 조건】

1. 효율 75% 이상 → 적극 추천
2. 효율 70-74% → 선택 가능
3. 효율 70% 미만 → 비추천하지 않음

【효율 계산】
2열 필름 폭 ≤ 원반 유효 폭 (590mm: 550mm, 760mm: 720mm)
효율 = (2열 필름 폭 ÷ 원반 유효 폭) × 100%

예: 2열 필름 폭 551mm, 590mm 원반 (유효 폭 550mm)
  효율 = 551 ÷ 550 = 100.2% → 100% 효율, 2열 생산 추천
```

## 6. 기술 구현 가이드라인

### 6-1. 2열 생산 자동 판정 로직

```typescript
/**
 * 2열 생산 가능 자동 판정
 */
function canUseTwoColumnProduction(
  pouchType: string,
  dimensions: PouchDimensions
): { available: boolean; materialWidth: 590 | 760; efficiency: number } {

  const filmWidth2Column = calculateFilmWidth(pouchType, dimensions, 2);
  const maxParallel590 = Math.floor(550 / filmWidth2Column);
  const maxParallel760 = Math.floor(720 / filmWidth2Column);

  if (maxParallel760 >= 2 && maxParallel760 > maxParallel590) {
    const efficiency = (filmWidth2Column * maxParallel760) / 720 * 100;
    if (efficiency >= 0.75) {
      return {
        available: true,
        materialWidth: 760,
        efficiency
      };
    }
  }

  if (maxParallel590 >= 2) {
    const efficiency = (filmWidth2Column * maxParallel590) / 550 * 100;
    if (efficiency >= 0.70) {
      return {
        available: true,
        materialWidth: 590,
        efficiency
      };
    }
  }

  return { available: false, materialWidth: 590, efficiency: 0 };
}
```

### 6-2. 할인율 결정 함수

```typescript
/**
 * 효율에 기반한 할인율 계산
 */
function determineDiscountRate(
  efficiency: number,
  parallelCount: number = 2
): { discount: number; message: string } {

  if (parallelCount === 1) {
    return { discount: 0, message: "1열 생산" };
  }

  if (efficiency >= 0.90) {
    return { discount: 0.40, message: "필름 효율 90% 이상으로 40% OFF" };
  }

  if (efficiency >= 0.80) {
    return { discount: 0.35, message: "필름 효율 80-89%로 35% OFF" };
  }

  if (efficiency >= 0.70) {
    return { discount: 0.30, message: "필름 효율 70-79%로 30% OFF" };
  }

  return { discount: 0.20, message: "필름 효율 60-69%로 20% OFF" };
}
```
