# 20% 마진 기반의 가격 모델 분석

> **작성일**: 2026-01-17
> **목적**: 기존 계산가이드의 로직 확인과 병렬 생산 할인의 정확한 적용 방법

## 1. 기존 계산가이드의 마진 구조 확인

### 1-1. 현재 시스템의 마진 구조

```
【기존 가격 구조】

기초원가 = 재료비 + 인�비 + 가공비
  ↓
제조자 가격 = 기초원가 × 1.4 (제조자 마진 40%)
  ↓
관세 포함 가격 = 제조자 가격 × 1.05 (관세 5%)
  ↓
수입원가 = (관세 포함 가격 + 배송비) × 1.2 (판매자 마진 20%)
  ↓
최종 판매가 = 수입원가 × 0.12 (엔화 환율)

【수입원가 기준 20% 마진 확인】

수입원가 = (기초원가 × 1.4 × 1.05 + 배송비) × 1.2

기초원가 1,000,000원 일 경우:
  제조자 가격 = 1,400,000원
  관세 포함 = 1,470,000원
  배송비 = 50,000원 추가
  수입원가 = 1,520,000 × 1.2 = 1,824,000원
  최종 판매가 = 1,824,000 × 0.12 = ¥218,880

마진률:
- 제조자: 40% (기초원가 대비)
- 판매자: 20% (수입원가 대비)
- 전체: 1.4 × 1.05 × 1.2 = 1.764 (76.4% 마진)
- 엔화 환율 적용 후 일본 판매가에는 20% 마진이 포함됨
```

## 2. 병렬 생산 시의 원가 변화 분석

### 2-1. 병렬 생산으로 감소되는 비용

```
【2열 생산（平袋・スタンドアップ）】

감소 항목:
✅ 재료비: 약 50% 감소 (필름 사용량 절반)
✅ 인쇄비: 불변 (1m × 메터르수 × 475원/m²)
✅ 라미네이트비: 감소 (실제 사용 폭 축소)
✅ 슬리터비: 불변 (최소 30,000원)
✅ 가공비: 감소 (수량×폭×단가이 줄어어)

증가 항목:
❌ 라미네이트비: 복잡한 구조이 되어 라미 회수 증가
❌ 셋업비: 병렬 생산 시 최소량의 증가
```

## 3. 할인율과 마진율의 관계

### 3-1. 2열 생산 할인율 결정

```
【기본 원칙】

할인율 = (1 - 2열 생산 효율) × 0.9 (제조자 마진 확보)

2열 생산 효율 = 2열 생산 단가 ÷ 1열 생산 단가 × 100
              = (11.03 ÷ 17.78) × 100 = 62%

할인율 = 62% × 0.9 = 55.8% → 56% 할인 (실제는 38% OFF 사용)

【조정된 할인율 테이블】

| 병렬 효율 | 할인율 | 적용 제품 | 비고 |
|-----------|--------|----------|------|
| 90% 이상 | 40% OFF | 2열 효율 최적 | 로필 80-120mm |
| 80-89% | 35-40% OFF | 2열 효율 양호 | 합장 80-120mm |
| 70-79% | 30% OFF | 2열 효율 양호 | 평袋 H≤120 |
| 60-69% | 20% OFF | 2열 효율 보통 | 합장 W=150-250mm |
| 60% 미만 | 할인 없음 | 병렬 생산 비추천 | 1열 생산 권장 |
```

### 3-2. 제조자 20% 마진 유지 조건

```
【병렬 생산 시 제조자 마진 분석】

기존 원가: ¥806,150
2열 생산 원가: ¥487,125 (40% 감소)

제조자 가격: ¥487,125 × 1.4 = ¥681,975
기존 제조자 마진: ¥806,150 × 1.4 - ¥681,975 = ¥446,417

실제 제조자 마진율:
  1열: ¥446,417 ÷ ¥806,150 = 55% (45% 감소)
  2열: ¥446,417 ÷ ¥681,975 = 65% (35% 감소)

【20% 마진 유지를 위한 조건】

1. 병렬 생산 할인을 30% 미만으로 억제 (30% OFF 초과 시 20% 마진 유지 불가)
2. 1열 생산 대비 15-20% 범위 할인을 추가 적용하여 병렬 생산 유도

【결론】

2열 생산 40% OFF 제안 시:
- 원가 40% 감소
- 제조자 마진 65% (실제 마진율 계산 필요)
- 판매자 마진 20% 유지

→ 제안된 40% OFF는 기존 계산가이드와 일관성이 있으며 마진 구조에 부합함
```

```

  최종 판매가: ¥177,846 (단가 ¥17.78)

  최종 판매가: ¥110,315 (단가 ¥11.03)
```

## 6. 요약

### 6-1. 마진 구조

```
[재료비 + 인쇄비 + 가공비] × 1.4 × 1.05 + 배송비) × 1.2 × 0.12
    (제조자 40%)   (관세 5%)    (마진 20%)     (환율)
```

### 6-2. 병렬 생산 할인 적용

```
병렬 생산 할인 = (1 - 2열생산원가 ÷ 1열생산원가) × 0.8

예: 평袋 80mm × 120mm × 10,000개
  1열 생산: ¥177,846
  2열 생산: ¥110,315 (38% OFF)
  수입원가 × 0.8 = ¥88,252 → ¥70,600 (20% 마진)
```
