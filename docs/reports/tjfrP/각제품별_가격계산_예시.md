## 2. 평袋 120mm × 180mm × 10,000개 예시

### 2-1. 조건/입력값

| 항목 | 값 |
|------|-----|
| 제품 타입 | 평袋 (3방밀봉) |

| 필름 구조 | PET 12μ / AL 7μ / LLDPE 80μ (3층) |
| 인쇄 | 기본 인쇄 |
| 지퍼 | 있음 |
| 원반 폭 | 590mm 또는 760mm |

### 2-2. 필름 폭 계산

**1열 생산:**
```
필름 폭 = (H × 2) + 41
        = (180 × 2) + 41
        = 360 + 41
        = 401mm
```

**2열 생산:**
```
필름 폭 = (H × 4) + 71
        = (180 × 4) + 71
        = 720 + 71
        = 791mm
```

### 2-3. 1열 생산 vs 2열 생산 비교

#### 1열 생산 상세 계산

**Step 1: 이론 미터 수**
```
피치 (1개당 필요 필름 길이) = W + 15mm = 120 + 15 = 135mm
1m당 생산 수 = 1000mm ÷ 135mm = 7.407개/m
이론 미터 수 = 10,000개 ÷ 7.407개/m = 1,350m
```

**Step 2: 확보량 계산**
```
최소 확보량 (1SKU) = 500m
이론 값 1,350m > 500m이므로 50m 단위로 절상
확보량 = CEILING(1,350 ÷ 50) × 50 = 1,350m
```

**Step 3: 로스 계산**
```
로스 = 400m (고정)
총 필름 = 1,350m + 400m = 1,750m
```

#### 2열 생산 상세 계산

**Step 1: 이론 미터 수**
```
2열 생산 시 1m당 생산 수 = 7.407개/m × 2 = 14.815개/m
이론 미터 수 = 10,000개 ÷ 14.815개/m = 675m
```

**Step 2: 확보량 계산**
```
최소 확보량 (1SKU) = 500m
이론 값 675m > 500m이므로 50m 단위로 절상
확보량 = CEILING(675 ÷ 50) × 50 = 700m
```

**Step 3: 로스 계산**
```
로스 = 400m (고정)
총 필름 = 700m + 400m = 1,100m
```

**Step 4: 필름 폭 검증**
```
계산된 2열 필름 폭 = 791mm
760mm 원반의 유효 폭 = 740mm
791mm > 740mm이므로 2열 생산 불가!
→ 1열 생산만 가능
```

> **중요**: 120mm × 180mm 평袋는 2열 필름 폭이 791mm로 760mm 원반을 초과하므로 2열 생산이 불가능합니다. 1열 생산만 가능합니다.

### 3-2. 필름 폭 계산

**마치 (G) = 50mm 기준**

**1열 생산:**
```
필름 폭 = (H × 2) + G + 35
        = (150 × 2) + 50 + 35

```

```
필름 폭 = (H × 4) + G + 40
```

```
2열 1m당 생산 수 = 8.696개/m × 2 = 17.392개/m
이론 미터 수 = 10,000개 ÷ 17.392개/m = 575m
```

```
확보량 = CEILING(575 ÷ 50) × 50 = 600m
로스 = 400m

```

```
절감율 = (1,550 - 1,000) ÷ 1,550 = 35.5%
```

## 7. 🆕 병렬 생산 할인 정책 업데이트 (2026-01-18)

### 7-1. 할인 기본 룰 (병렬 생산 할인표)

| 병렬 수 | 할인 적용 | 가격 계산식 | 할인율 | 예시 (1본 20만엔) |
|---------|----------|-----------|--------|------------------|

### 7-2. 할인 계산식

```
병렬 생산 가격 = 기준 가격 × [1 + (추가 병렬 수 × 할인 계수)]

할인 계수:
  - 2번째 (2열): 0.6 (40% 할인)
  - 3번째 이상 (3열 이상): 0.3 (70% 할인)

계산 예:
  2본: 기준 가격 × (1 + 0.6) = 기준 가격 × 1.6
  3본: 기준 가격 × (1 + 0.3 + 0.3) = 기준 가격 × 1.6
  4본: 기준 가격 × (1 + 0.3 + 0.3 + 0.3) = 기준 가격 × 2.0
  n본: 기준 가격 × (1 + 0.3 × (n-1)) ※ n≧3
```

### 7-3. 중요사항

#### 7-3-1. 할인 적용 범위

| 적용 대상 | 적용 여부 | 비고 |

### 7-4. 구현 로직 (TypeScript)

```typescript
/**
 * 병렬 생산 할인 계산
 * @param basePrice 기준 가격 (필름 원가 + 인쇄비 + 라미네이트비)
 * @param parallelCount 병렬 수 (1=단독 생산, 2=2열/2본, 3=3열/3본...)
 * @returns 할인 적용 후 가격
 */
function calculateParallelDiscount(
  basePrice: number,
  parallelCount: number
): number {
  if (parallelCount <= 1) return basePrice;

  // 2번째 (2열): 40% 할인 = 60% 가격
  // 3번째 이상: 70% 할인 = 30% 가격
  let discountMultiplier = 1; // 1번째는 100%

  if (parallelCount >= 2) {
    discountMultiplier += 0.6; // 2번째는 60%
  }

  if (parallelCount >= 3) {
    discountMultiplier += 0.3 * (parallelCount - 2); // 3번째 이상은 각 30%
  }

  return basePrice * discountMultiplier;
}
```
