## 1. 分析基準

### 1-2. 価格設定の根拠

```
【並列生産のコスト削減ロジック】

1列生産:
  フィルム使用量 = Q ÷ (1000 ÷ ピッチ) + ロス
  単価 = フィルム原価 ÷ Q

2列生産（平袋・スタンドアップ）:
  フィルム使用量 = 1列生産の約50%
  ただし、設備投資・管理コスト増加を考慮し、
  顧客への還元率は40% OFFに設定

並列生産（ロール・合掌・ボックス）:
  フィルム使用量 = 1/並列本数
  顧客への還元率 = 並列本数に応じて20-40% OFF
```

### 3-4. 数値根拠の詳細

**例: 80mm × 120mm 平袋 × 10,000個**

```
1列生産:
  ピッチ = 80 + 15 = 95mm
  1mあたり生産数 = 1000 ÷ 95 = 10.526個/m
  理論メートル数 = 10000 ÷ 10.526 = 950m
  最小確保量（1 SKU）= 950m → 50m単位で切り上げ = 950m
  ロス = 400m
  総フィルム = 950 + 400 = 1350m

2列生産:
  1mあたり生産数 = 10.526 × 2 = 21.052個/m
  理論メートル数 = 10000 ÷ 21.052 = 475m
  最小確保量（1 SKU）= 500m（理論値475m < 500m）
  ロス = 400m
  総フィルム = 500 + 400 = 900m

コスト比較:
  1列: 1350m使用
  2列: 900m使用（33%削減）
  顧客還元: 40% OFF（製造者は7%利益確保）

なお40% OFFにした理由:
- フィルム原価削減: 33%
- 設備投資・管理コスト: -3%
- 販売促進効果を考慮し40%に設定
```
