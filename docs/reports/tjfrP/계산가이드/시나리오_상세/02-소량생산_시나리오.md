# 소량 생산 시나리오

## 2-1. 500개 생산 (평백 80×120mm)

### 1열 생산

```
【500개 1열 생산 계산】

기본 사양:
- 치수: 80×120mm (평백)
- 횡(W): 120mm = 12cm
- 수량: 500개
- 피치: 120mm
- 1m당 생산 수: 8.33개/m

필름 계산:
- 이론 미터: 60m (500개 ÷ 8.33개/m)
- 확보량: 500m (최소 확보량, 1SKU)
- 로스: 400m (고정)
- 총 필름: 900m
- 필름 폭: 281mm → 590mm 원반 (실제 폭: 0.59m)

【원재료비 계산】(4층 구조: PET 12μ + AL 7μ + PET 12μ + LLDPE 90μ)
- PET (12μ): 0.012 × 0.59 × 900 × 1.40 = 8.92kg × 2층 = 17.84kg
- AL (7μ): 0.007 × 0.59 × 900 × 2.71 = 10.08kg
- LLDPE (90μ): 0.090 × 0.59 × 900 × 0.92 = 43.89kg
- 총 중량: 71.81kg

원재료비 = (17.84kg × 2,800원) + (10.08kg × 7,800원) + (43.89kg × 2,800원)
         = 49,952원 + 78,624원 + 122,892원
         = 251,468원

【인쇄비】(항상 1m로 계산)
= 1m × 900m × 475원/m² = 427,500원

【라미네이트비】(AL 있음: 75원/m, 라미 회수 3회)
= 0.59m × 900m × 75원 × 3회 = 119,475원

【슬리터비】
= MAX(30,000원, 900m × 10원) = 30,000원

【파우치 가공비】(평백 12cm, 최소 200,000원)
= 12cm × 0.4원 × 500개 = 2,400원
= MAX(2,400원, 200,000원) = 200,000원

【기초 원가】
= 251,468 + 427,500 + 119,475 + 30,000 + 200,000
= 1,028,443원

【마진 및 비용】
- 제조자 마진 (40%): 1,028,443 × 0.4 = 411,377원
- 제조자 가격: 1,028,443 + 411,377 = 1,439,820원
- 관세 (5%): 1,439,820 × 0.05 = 71,991원
- 관세 포함 가격: 1,511,811원

【배송비】
- 총 중량: 약 72kg → 3박스 (29kg/박스)
- 배송비: 3 × 127,980원 = 383,940원

【최종 판매가】
- 수입원가: 1,511,811 + 383,940 = 1,895,751원
- 판매자 마진 (20%): 1,895,751 × 0.2 = 379,150원
- 최종 판매가(원화): 2,274,901원
- 최종 판매가(엔화): 2,274,901 × 0.12 = 272,988엔
- 1개당 단가: 546엔
```

### 2열 생산 (1,000개)

```
【1,000개 2열 생산 계산】

기본 사양:
- 치수: 80×120mm (평백)
- 횡(W): 120mm = 12cm
- 수량: 1,000개
- 피치: 120mm
- 1m당 생산 수: 16.67개/m (2열)

필름 계산:
- 이론 미터: 60m (1,000개 ÷ 16.67개/m)
- 확보량: 500m (최소 확보량, 1SKU)
- 로스: 400m (고정)
- 총 필름: 900m
- 필름 폭: 551mm → 590mm 원반 (실제 폭: 0.59m)

【원재료비】(동일: 251,468원)
【인쇄비】(동일: 427,500원)
【라미네이트비】(동일: 119,475원)
【슬리터비】(동일: 30,000원)

【파우치 가공비】(1,000개)
= 12cm × 0.4원 × 1,000개 = 4,800원
= MAX(4,800원, 200,000원) = 200,000원

【기초 원가】(동일)
= 1,028,443원

【마진 및 비용】(동일)
- 최종 판매가(원화): 2,274,901원
- 최종 판매가(엔화): 272,988엔
- 1개당 단가: 273엔

비교:
- 1열 500개: 546엔/개
- 2열 1,000개: 273엔/개 (50% 절감)
```
