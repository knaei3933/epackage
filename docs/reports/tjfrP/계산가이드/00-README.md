## 🚀 クイックスタート

### 基本計算フロー

```
1. パウチサイズ決定 (W × H ± G)
   ↓
2. SKU数決定
   ↓
3. 列数自動判定 (2列可能なら2列採用)
   ↓
4. フィルム幅計算（パウチタイプ・列数に応じて）
   ↓
5. 原反選定（590mm または 760mm）
   ↓
6. 理論メートル数計算
   ↓
7. 確保量計算（SKU数に応じた最小量、50m単位切り上げ）
   ↓
8. ロス追加（400m固定）
   ↓
9. 材料原価計算（ウォン）
   ↓
10. 印刷費用計算（ウォン）
   ↓
11. 後加工費用計算（ウォン）
   ↓
12. 関税適用（5%）
   ↓
13. 配送料計算（29kg単位）
   ↓
14. 円貨換算（× 0.12）
   ↓
15. 1個当たり単価算出
```

---

## 📌 重要ポイント

| 項目 | 設定値 | 説明 |
|------|--------|------|
| **印刷費** | 常に **1m**で計算 | フィルム幅と無関係 |
| **ロス（損失）** | **400m固定** | SKU数や数量に関わらず一定 |
| **1SKU最小確保量** | 500m | 1SKUの場合 |
| **2SKU以上最小確保量** | 各300m | 2SKU以上の場合、各300m |
| **総発注量** | 確保量の合計 + 400m | ロスは400m固定 |
| **ラミ単価（ALあり）** | 75ウォン/m | AL素材使用時 |
| **ラミ単価（ALなし）** | 65ウォン/m | AL素材なしの場合 |
| **配送料** | 127,980ウォン/箱 | 29kg/箱の最大 |

## 🎯 パウチタイプ別フィルム幅計算

| パウチタイプ | 1列公式 | 2列公式 | 備考 |
|-------------|---------|---------|------|
| **平袋（三封）** | `(H × 2) + 41` | `(H × 4) + 71` | 横向き印刷、縦Hが幅を決定 |
| **スタンドアップ** | `(H × 2) + G + 35` | `(H × 4) + G + 40` | 横向き印刷、縦Hが幅を決定 |
| **合掌袋（T封）** | `(W × 2) + 22` | なし | 展開図基準、横Wが幅を決定 |
| **ボックスパウチ（M封）** | `(G + W) × 2 + 32` | なし | 展開図基準、横Wと側面Gが幅を決定 |
| **スパウトパウチ** | スタンドアップ準用 | スタンドアップ準用 | 構造はスタンドアップと同一 |

## 💡 最終計算式

```
最終価格(円) = ((材料原価 + 印刷費 + 後加工費) × 1.4 × 1.05 + 配送料) × 1.2 × 0.12
```

**内訳**:
- `× 1.4` = 製造者マージン40%
- `× 1.05` = 関税5%
- `配送料追加` = マージン計算対象外
- `× 1.2` = 販売者マージン20%
- `× 0.12` = 円貨換算（為替レート）
