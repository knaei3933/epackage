## 1. 完全な価格計算構造

### 1-1. 全体フロー

```
1. 基礎原価 = 原材料費 + 印刷費 + 後加工費
   ↓
2. 製造者価格 = 基礎原価 × 1.4（製造者マージン40%追加）
   ↓
3. 関税込み価格 = 製造者価格 × 1.05（関税5%追加）
   ↓
4. 配送料追加（マージン計算対象外）
   ↓
5. 輸入原価 = 関税込み価格 + 配送料
   ↓
6. 最終販売価格 = 輸入原価 × 1.2（販売者マージン20%追加）
   ↓
7. 円貨換算 = 最終販売価格 × 0.12（為替レート適用）
```

> **最も重要**: 配送料はマージン計算の対象外です。製造者マージン40%と販売者マージン20%の適用範囲には入りません。

### 1-2. 各ステップの詳細

| ステップ | 項目 | 計算式 | 備考 |
|----------|------|----------|------|
| **①** | 原材料費 | 各層の重量 × 単価の合計 | 4層構造の場合、PETが2回使用 |
| **②** | 印刷費 | **1m** × メートル数 × 475ウォン/m² | **重要**: フィルム幅に関係なく、常に1mで計算 |
| **③** | 後加工費 | ラミ + スリッター + 製袋 + オプション | ラミ回数 = 層数 - 1 |
| **④** | 基礎原価 | ① + ② + ③ | マージン計算の基礎 |
| **⑤** | 製造者マージン | ④ × 0.4 | 基礎原価の40% |
| **⑥** | 製造者価格 | ④ + ⑤（または ④ × 1.4） | 関税適用前の価格 |
| **⑦** | 関税込み価格 | ⑥ × 1.05 | 関税5%適用 |
| **⑧** | 配送料 | 箱数 × 127,980ウォン | **マージン対象外** |
| **⑨** | 輸入原価 | ⑦ + ⑧ | 配送料追加後 |
| **⑩** | 販売者マージン | ⑨ × 0.2 | 輸入原価の20% |
| **⑪** | 最終販売価格 | ⑨ + ⑩（または ⑨ × 1.2） | ウォン建て価格 |
| **⑫** | 円貨換算 | ⑪ × 0.12 | 日本円での価格 |

## 2. 関税と配送料

### 2-1. 関税（輸入関税）

```
関税込み価格(ウォン) = 製造者価格 × 1.05
```

> **1.05 = 1 + 0.05（5%関税）**

### 2-2. 配送料計算（韓国国内配送料）

#### 重量ベース配送料（実績 + 8%余裕）

| 種別(kg) | 配送料（ウォン） |
|----------|-----------------|
| 5.0 | 46,440ウォン |
| 10.0 | 64,800ウォン |
| 15.0 | 82,620ウォン |
| 20.0 | 98,820ウォン |
| 25.0 | 115,020ウォン |
| 29.0（1箱最大） | **127,980ウォン** |

> **重要**: 実績価格に8%の余裕を追加した保守的見積もり
> **重要**: 29kgが1箱の最大。1kgは梱包材の余裕分。29kg超過の配送は不可

#### 配送箱数計算

```
配送箱数 = CEILING(総重量 ÷ 29kg)  [切り上げ]
配送料(ウォン) = 配送箱数 × 127,980ウォン/箱
```

> **重要**: 各ロールは29kg以下に制限。超過する場合は分割必要

## 3. 製造者マージンと販売者マージン

### 3-1. マージン率

| マージン | 率 | 適用対象 | 説明 |
|----------|-----|----------|------|
| **製造者マージン** | 40% | 基礎原価 | 韓国製造者の利益 |
| **関税** | 5% | 製造者価格 | 日本輸入時の関税 |
| **販売者マージン** | 20% | 輸入原価 | 日本販売者の利益 |
| **配送料** | - | 重量ベース | **マージン対象外** |

### 3-2. 配送料の位置づけ

```
1. 基礎原価（原材料費 + 印刷費 + 後加工費）
   ↓
2. 製造者価格（基礎原価 × 1.4）
   ↓
3. 関税込み価格（製造者価格 × 1.05）
   ↓
4. 配送料追加（ここで初めて追加）
   ↓
5. 輸入原価（関税込み価格 + 配送料）
   ↓
6. 最終販売価格（輸入原価 × 1.2）
```

> **最も重要**: 配送料はマージン計算の対象外です。製造者マージン40%と販売者マージン20%の適用範囲には入りません。

### 3-3. 配送料の重要なポイント

```
❌ 誤った計算: 配送料にマージンを適用
   (基礎原価 + 配送料) × 1.4 × 1.05 × 1.2

✅ 正しい計算: 配送料はマージン適用後に追加
   ((基礎原価 × 1.4) × 1.05 + 配送料) × 1.2
```

**理由**: 配送料は外部サービス費用であり、製造者や販売者の利益に含まれないため、マージン計算の対象外となります。

## 4. 円貨換算

### 4-1. 為替レート

```
最終価格(円) = 最終販売価格(ウォン) × 0.12
1個当たり単価(円) = 最終価格(円) ÷ 数量
```

> **0.12 = 為替レート（1円 = 約8.33ウォン）**

### 4-2. 計算例

| 最終販売価格(ウォン) | 計算式 | 最終価格(円) |
|-------------------|--------|------------|
| 3,000,000ウォン | 3,000,000 × 0.12 | 360,000円 |
| 3,500,000ウォン | 3,500,000 × 0.12 | 420,000円 |
| 4,000,000ウォン | 4,000,000 × 0.12 | 480,000円 |

## 5. 価格計算のまとめ

### 5-1. 計算の流れ

```
1. 原材料費計算（各層の重量 × 単価の合計）
   ↓
2. 印刷費計算（1m × メートル数 × 475ウォン/m²）
   ↓
3. 後加工費計算（ラミ + スリッター + 製袋 + オプション）
   ↓
4. 基礎原価の算出
   ↓
5. 製造者マージン40%適用 → 製造者価格
   ↓
6. 関税5%適用 → 関税込み価格
   ↓
7. 配送料追加（マージン計算対象外）
   ↓
8. 輸入原価の算出
   ↓
9. 販売者マージン20%適用 → 最終販売価格
   ↓
10. 円貨換算（× 0.12）
```

### 5-2. 原反幅計算のポイント

| 原反幅 | 実際使用幅 | 計算式 |
|--------|-----------|----------|
| **590mm原反** | 0.59m | 原反幅 - 20mm = 実際使用幅 |
| **760mm原反** | 0.76m | 原反幅 - 20mm = 実際使用幅 |

### 5-3. 重要ポイントまとめ

| 項目 | 設定値 | 説明 |
|------|--------|------|
| **印刷費** | 常に **1m**で計算 | フィルム幅と無関係 |
| **ロス（損失）** | **400m固定** | SKU数や数量に関わらず一定 |
| **1SKU最小確保量** | 500m | 1SKUの場合 |
| **2SKU以上最小確保量** | 各300m | 2SKU以上の場合、各300m |
| **総発注量** | 確保量の合計 + 400m | ロスは400m固定 |
| **ラミ単価（ALあり）** | 75ウォン/m | AL素材使用時 |
| **ラミ単価（ALなし）** | 65ウォン/m | AL素材なしの場合 |
| **配送料** | 127,980ウォン/箱 | 29kg/箱の最大 |
| **製造者マージン** | 40% | 基礎原価に対して |
| **関税** | 5% | 製造者価格に対して |
| **販売者マージン** | 20% | 輸入原価に対して |
| **為替レート** | 0.12 | ウォン → 円換算 |

---

## 6. 最終計算式

```
最終価格(円) = ((材料原価 + 印刷費 + 後加工費) × 1.4 × 1.05 + 配送料) × 1.2 × 0.12
```

**内訳**:
- `× 1.4` = 製造者マージン40%
- `× 1.05` = 関税5%
- `配送料追加` = マージン計算対象外
- `× 1.2` = 販売者マージン20%
- `× 0.12` = 円貨換算（為替レート）
