## 1. フィルム原価計算

### 1-1. 基本情報

| 項目 | 説明 |
|------|------|
| 実際供給数量 | 顧客注文数量 (例: 1,000m) |
| ロス含数量 | 実際供給 + ロス (例: 1,400m) |
| ロス(LOSS) | 製造過程の損失率 |

### 1-2. 原反価格計算式（円建て）

```
原反価格(円) = 厚み × 幅 × メートル数(ロス含) × 比重 × 単価(円/kg)
```

### 1-3. 原反別計算例

#### PET 原反

| パラメータ | 値 |
|-----------|-----|
| 厚み | 0.012mm |
| 幅 | 0.76m |
| メートル数(ロス含) | 1,400m |
| 比重 | 1.40 |
| 単価 | 280円/kg |

```
PET 重量 = 0.012 × 0.76 × 1,400 × 1.40 = 17.88kg
PET 価格(円) = 17.88kg × 280円/kg = 5,006円
```

#### AL (アルミニウム) 原反

| パラメータ | 値 |
|-----------|-----|
| 厚み | 0.007mm |
| 幅 | 0.76m |
| メートル数(ロス含) | 1,400m |
| 比重 | 2.71 |
| 単価 | 780円/kg |

```
AL 重量 = 0.007 × 0.76 × 1,400 × 2.71 = 20.19kg
AL 価格(円) = 20.19kg × 780円/kg = 15,748円
```

#### LLDPE 原反

| パラメータ | 値 |
|-----------|-----|
| 厚み | 0.060mm |
| 幅 | 0.76m |
| メートル数(ロス含) | 1,400m |
| 比重 | 0.92 |
| 単価 | 280円/kg |

```
LLDPE 重量 = 0.060 × 0.76 × 1,400 × 0.92 = 58.73kg
LLDPE 価格(円) = 58.73kg × 280円/kg = 16,444円
```

### 1-4. 総材料原価

```
総材料原価(円) = Σ(すべての原反価格)
```

例:
```
総材料原価 = 5,006 + 15,748 + 16,444 = 37,198円
```

### 1-5. 原反単価情報

| 材料 | kg当たり単価(円) | 比重 |
|------|------------------|------|
| AL   | 7,800円          | 2.71 |
| NY   | 5,400円          | 1.16 |
| LLDPE| 2,800円          | 0.92 |
| PET  | 2,800円          | 1.40 |
| VMPET| 3,600円          | 1.40 |

## 2. パウチ加工賃計算

### 2-1. 基本計算式

```
加工賃(円) = パウチ横幅(cm) × 単価(円/cm) × 数量
```

### 2-2. パウチタイプと用語

| 韓国語 | 日本語 | 英語 | 説明 |
|--------|--------|------|------|
| 삼방 (三封) | 平袋 (平らな袋) | Flat Pouch | 3辺シール（底＋両側） |
| T방 (T封) | 合掌袋 | Pillow Pouch | 両側シール、底部が合掌形状 |
| M방 (M封) | BOXパウチ | Box Pouch | 4辺シール、自立可能 |

### 2-3. 各パウチタイプの加工単価

#### 平袋（三封・サンバン）

```
平袋加工賃 = 横幅(cm) × 単価A × 数量
```

| 横幅範囲 | 単価(円/cm) |
|----------|-------------|
| ~10cm    | 〇〇円       |
| 10~15cm  | 〇〇円       |
| 15~20cm  | 〇〇円       |
| 20cm~    | 〇〇円       |

#### 合掌袋（T封・ティーバン）

```
合掌袋加工賃 = 横幅(cm) × 単価B × 数量
```

| 横幅範囲 | 単価(円/cm) |
|----------|-------------|
| ~10cm    | 〇〇円       |
| 10~15cm  | 〇〇円       |
| 15~20cm  | 〇〇円       |
| 20cm~    | 〇〇円       |

#### BOXパウチ（M封・エムバン）

```
BOXパウチ加工賃 = 横幅(cm) × 単価C × 数量
```

| 横幅範囲 | 単価(円/cm) |
|----------|-------------|
| ~10cm    | 〇〇円       |
| 10~15cm  | 〇〇円       |
| 15~20cm  | 〇〇円       |
| 20cm~    | 〇〇円       |

### 2-4. 計算例

#### 例1: 平袋 1,000個

| 項目 | 値 |
|------|-----|
| パウチタイプ | 平袋（三封） |
| 横幅 | 15cm |
| 数量 | 1,000個 |
| 単価 | 5円/cm |

```
加工賃 = 15cm × 5円/cm × 1,000個 = 75,000円
```

#### 例2: 合掌袋 5,000個

| 項目 | 値 |
|------|-----|
| パウチタイプ | 合掌袋（T封） |
| 横幅 | 20cm |
| 数量 | 5,000個 |
| 単価 | 6円/cm |

```
加工賃 = 20cm × 6円/cm × 5,000個 = 600,000円
```

#### 例3: BOXパウチ 2,000個

| 項目 | 値 |
|------|-----|
| パウチタイプ | BOXパウチ（M封） |
| 横幅 | 12cm |
| 数量 | 2,000個 |
| 単価 | 8円/cm |

```
加工賃 = 12cm × 8円/cm × 2,000個 = 192,000円
```

### 2-5. オプション加工費

| オプション | 単価 | 計算式 |
|-----------|------|--------|
| ジッパー追加 | ×円/個 | 数量 × 単価 |
| バルブ追加 | ×円/個 | 数量 × 単価 |
| マット印刷 | ×円/m | メートル数 × 単価 |
| スリット | ×円/個 | 数量 × 単価 |

## 3. 関税適用

### 関税率

| 区分 | 関税率 |
|------|--------|
| 一般材料 | 5% (0.05) |

### 関税込み価格

```
関税込み価格(円) = (材料原価 + 加工賃) × (1 + 関税率)
                = (材料原価 + 加工賃) × 1.05
```

例:
```
(37,198円 + 75,000円) × 1.05 = 117,733円
```

---

## 4. 配送料計算

### 4-1. 重量計算

各原反重量(kg) = 厚み × 幅 × メートル数 × 比重

```
総重量 = Σ(各原反重量)
```

例:
```
PET:   0.012 × 0.76 × 1,400 × 1.40 = 17.88kg
AL:    0.007 × 0.76 × 1,400 × 2.71 = 20.19kg
LLDPE: 0.060 × 0.76 × 1,400 × 0.92 = 58.73kg
───────────────────────────────────────
総重量 = 96.8kg
```

### 4-2. ロール数計算

```
ロール数 = CEILING(総重量 ÷ 30kg)  [切り上げ]
```

例:
```
ロール数 = CEILING(96.8 ÷ 30) = CEILING(3.23) = 4ロール
```

### 4-3. 配送料

```
配送料(円) = ロール数 × 16,800円
```

例:
```
配送料 = 4 × 16,800 = 67,200円
```

## 5. 円貨換算

### 円貨総額

```
円貨総額(円) = 関税込み価格(円) + 配送料(円)
```

例:
```
117,733円 + 67,200円 = 184,933円
```

### 原貨換算

```
最終原価(ウォン) = 円貨総額(円) × 為替レート(ウォン/円)
```

例 (為替レート: 9ウォン/1円):
```
最終原価 = 184,933 × 9 = 1,664,397ウォン
```

## 6. 総合計算式

### 基本式

```
最終原価(ウォン) = ((材料原価(円) + 加工賃(円)) × 1.05 + 配送料(円)) × 為替レート
```

### フィルムのみ（加工なし）

```
最終原価(ウォン) = (材料原価(円) × 1.05 + 配送料(円)) × 為替レート
```

### パウチ製品（加工あり）

```
最終原価(ウォン) = ((材料原価(円) + 横幅×単価×数量) × 1.05 + 配送料(円)) × 為替レート
```

### 1個当たり単価

```
1個当たり単価(円) = 円貨総額(円) ÷ 数量
```

例:
```
184,933円 ÷ 1,000個 = 184.9円/個
```

## 7. 実際の計算例

### 📋 パウチ製品注文情報

| 項目 | 値 |
|------|-----|
| パウチタイプ | 平袋（三封） |
| 横幅 | 15cm |
| 数量 | 10,000個 |
| フィルム構成 | PET/AL/LLDPE 3層 |
| 実際供給数量 | 1,000m |
| ロス含数量 | 1,400m |
| 為替レート | 9ウォン/1円 |

### Step 1: 材料原価計算

| 材料 | 重量計算 | 重量(kg) | 単価(円/kg) | 価格(円) |
|------|----------|----------|-------------|----------|
| PET  | 0.012×0.76×1400×1.40 | 17.88 | 2,800 | 50,064 |
| AL   | 0.007×0.76×1400×2.71  | 20.19 | 7,800 | 157,482 |
| LLDPE| 0.060×0.76×1400×0.92 | 58.73 | 2,800 | 164,444 |
| **合計** | | **96.8** | | **371,990** |

```
材料原価 = 371,990円
```

### Step 2: 加工賃計算

| 項目 | 値 |
|------|-----|
| パウチタイプ | 平袋（三封） |
| 横幅 | 15cm |
| 数量 | 10,000個 |
| 単価 | 5円/cm |

```
加工賃 = 15cm × 5円/cm × 10,000個 = 750,000円
```

### Step 3: 小計（材料 + 加工）

```
小計 = 371,990円 + 750,000円 = 1,121,990円
```

### Step 4: 関税適用

```
関税込み価格 = 1,121,990円 × 1.05 = 1,178,090円
```

### Step 5: 配送料計算

```
ロール数 = CEILING(96.8 ÷ 30) = 4ロール
配送料 = 4 × 16,800円 = 67,200円
```

### Step 6: 円貨総額

```
円貨総額 = 1,178,090円 + 67,200円 = 1,245,290円
```

### Step 7: ウォン換算

```
最終原価 = 1,245,290円 × 9 = 11,207,610ウォン
```

### Step 8: 1個当たり単価

```
1個当たり単価 = 1,245,290円 ÷ 10,000個 = 124.5円/個
```

## 📋 注文処理プロセス

### Step 1: 使用材料選択

顧客注文幅に合わせて540mmまたは740mm材料の中から効率的なものを選択

```
材料幅 ÷ 注文幅 = 分割可能個数 (小数点切り捨て)
```

### Step 2: 最小注文数量計算

```
最小注文数量(m) = 500m × 分割個数
```

### Step 3: 価格計算

#### 📊 計算例表

| 注文幅 | 使用材料 | 分割個数 | 最小注文数量 | 総面積 |
|--------|----------|----------|--------------|---------|
| 740mm  | 740mm    | 1個      | 500m         | 370m²   |
| 540mm  | 540mm    | 1個      | 500m         | 270m²   |
| 370mm  | 740mm    | 2個      | 1,000m       | 370m²   |
| 270mm  | 540mm    | 2個      | 1,000m       | 270m²   |
| 180mm  | 540mm    | 3個      | 1,500m       | 270m²   |
| 120mm  | 740mm    | 6個      | 3,000m       | 360m²   |

> **参考:** 分割個数ごとにSKU（デザイン）を複数にできます。

### ✅ ポイント

- 材料別面積単価が違う → 使用材料により最終価格が変わる
- 分割個数が多いほど → 最小注文数量(メートル)増加
- 同一材料使用時 → 注文幅に関係なく最小金額同一

## まとめ

このフィルム・パウチ価格計算ガイドは次の公式を基盤としています:

### フィルムのみ

```
最終原価(ウォン) = (材料原価(円) × 1.05 + 配送料(円)) × 為替レート
```

### パウチ製品

```
最終原価(ウォン) = ((材料原価(円) + 加工賃(円)) × 1.05 + 配送料(円)) × 為替レート
```

### 加工賃計算

```
加工賃(円) = パウチ横幅(cm) × 単価(円/cm) × 数量
```

この公式を使用して体系的にフィルム・パウチ価格を計算できます！ 🎯
