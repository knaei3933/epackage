## 価格構成（重要）

```
1. 基礎原価 = 原材料費 + 印刷費 + 後加工費
2. 製造者価格 = 基礎原価 × 1.4 (製造者マージン40%)
3. 関税込み価格 = 製造者価格 × 1.05 (関税5%)
4. 配送料追加 = 関税込み価格 + 配送料
5. 輸入原価 = 配送料追加後の価格
6. 最終販売価格 = 輸入原価 × 1.2 (販売者マージン20%)
7. 円貨換算 = 最終販売価格 × 0.12 (為替レート)
```

> **重要**: 配送料はマージン計算の対象外。製造者マージン40%と販売者マージン20%の適用には含まれません。

## 0. ロールフィルム（基礎ロジック）⭐重要

> **重要**: ロールフィルムの製造は、すべてのパッケージ製品の基礎となるロジックです。

### 0-1. 製品情報

| 項目 | 値 |
|------|-----|
| **製品タイプ** | ロールフィルム |
| **サイズ** | 幅=200mm、長さ=500m |
| **数量** | 1本（500m） |
| **フィルム構成** | PET 12μ + AL 7μ + LLDPE 90μ (3層) |
| **印刷** | デジタル印刷 |
| **後加工** | ラミネート、スリッター |
| **梱包** | ダンボール段ボール |

### 0-2. 基礎データ

| 項目 | 値 |
|------|-----|
| **フィルム幅** | 200mm |
| **使用原反** | 590mm (実質原反幅) |
| **印刷可能幅** | 570mm |
| **理論メートル数** | 500m |
| **50m単位切り上げ** | 500m |
| **最小確保量** | 500m (1SKUのため) |
| **ロス** | 400m (固定) |
| **実際発注量** | 500m + 400m = **900m** |

> **重要**: 500m注文でも、ロス400mが固定で追加されるため、実際の発注量は900mになります。

### 0-3. 原材料費計算（590mm原反使用）

| 材料 | 厚さ(μm) | 幅(m) | 長さ(m) | 比重 | 単価(ウォン/kg) | 重量(kg) | 価格(ウォン) |
|------|----------|--------|--------|------|----------------|----------|-------------|
| PET | 12 | 0.59 | 900 | 1.40 | 2,800 | 8.88 | 24,864 |
| AL | 7 | 0.59 | 900 | 2.71 | 7,800 | 10.02 | 78,156 |
| LLDPE | 90 | 0.59 | 900 | 0.92 | 2,800 | 43.74 | 122,472 |
| **合計** | - | - | - | - | - | **62.64kg** | **225,492ウォン** |

> **配送料計算用**: 実使用量500m分の重量を計算
> 実使用重量 = 62.64kg × (500m ÷ 900m) = 34.80kg

### 0-4. 印刷費

```
印刷費 = 1m × 900m × 475ウォン/m² = 427,500ウォン
```

> **重要**: 印刷費は**フィルム幅に関係なく、常に1mで計算**します。

### 0-5. 後加工費

#### ラミネート費 (ALあり: 75ウォン/m)

```
ラミ回数 = 3層 - 1 = 2回
ラミネート費 = 0.57m × 900m × 75ウォン × 2回 = 76,950ウォン
```

#### スリッター費

```
スリッター費 = MAX(30,000ウォン, 900m × 10ウォン) = 30,000ウォン
```

### 0-6. 配送料計算

```
実使用重量 = 62.64kg × (500m ÷ 900m) = 34.80kg
パッケージ数 = CEILING(34.80kg ÷ 29kg) = 2箱
配送料 = 2箱 × 127,980ウォン = 255,960ウォン
```

> **重要**: 配送料は**実使用量500m分の重量**で計算します。ロス400mは製品にならないため配送しません。

### 0-9. 並列生産の仕組み

```
【1本注文（500m）】
使用原反: 590mm
印刷幅: 200mm
原反効率: 200mm ÷ 590mm = 34%

【2本並列注文（1,000m）】
使用原反: 590mm（同じ!）
印刷幅: 200mm × 2 = 400mm
原反効率: 400mm ÷ 590mm = 68%（効率2倍!）

【3本並列注文（1,500m）】
使用原反: 590mm（同じ!）
印刷幅: 200mm × 3 = 600mm（570mm超過で不可）
→ 760mm原反使用
原反効率: 600mm ÷ 760mm = 79%（さらに効率向上!）
```

### 0-10. ロールフィルム価格計算のまとめ

1. **基本ロジック**: フィルム幅、長さ、層構成から重量を計算
2. **ロス固定**: 400mが常に追加される（数量に関わらず）
3. **配送料**: 実使用量分の重量で計算（ロスは配送しない）
4. **並列生産**: 同じ原反で複数本を同時生産可能
5. **単価半額**: 2本並列で単価が50%OFFに

> **結論**: ロールフィルムはすべてのパッケージ製品の基礎となるフィルムを製造する工程です。並列生産を活用することで、大幅なコスト削減が可能です。

## 価格計算フローまとめ

```
1. 原材料費計算
   ↓
2. 印刷費計算
   ↓
3. 後加工費計算（ラミネート、スリッター、製袋など）
   ↓
4. 基礎原価 = ① + ② + ③
   ↓
5. 製造者価格 = 基礎原価 × 1.4 (製造者マージン40%)
   ↓
6. 関税込み価格 = 製造者価格 × 1.05 (関税5%)
   ↓
7. 配送料追加（マージン対象外）
   ↓
8. 輸入原価 = 関税込み価格 + 配送料
   ↓
9. 最終販売価格 = 輸入原価 × 1.2 (販売者マージン20%)
   ↓
10. 円貨換算 = 最終販売価格 × 0.12
```

> **最も重要**: 配送料はマージン計算の対象外です。製造者マージン40%と販売者マージン20%の適用範囲には入りません。
