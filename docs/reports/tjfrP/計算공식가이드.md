## 1. 基本用語

| 用語 | 記号 | 単位 | 説明 |
|------|------|------|------|
| **H** | - | mm | パウチ縦（Height）、フィルム進行方向のサイズ |
| **W** | - | mm | パウチ横（Width）、ピッチ（平袋の場合は進行方向） |
| **G** | - | mm | マチまたは側面（Gusset）、スタンドアップの底部幅 |
| **Q** | - | 個 | 数量（個数またはメートル数） |
| **Q_sku** | - | 個 | SKU数量 |

## 2. フィルム幅計算公式

### 2-1. 三方袋（三封・平袋 / Flat Pouch）

**配向**: 横向きで印刷（縦Hがフィルム幅を決定）

#### 1列（1 Column）

| タイプ | 公式 | 詳細 |
|--------|------|------|
| 一般・ジッパー共通 | `(H × 2) + 41` | `10(余白) + 5 + H + 5 + 5 + H + 5 + 10(ジッパー) + H + 5 + 10 + 1(スリッター)` |

**例**: H=160mmの場合
```
フィルム幅 = (160 × 2) + 41 = 361mm
```

#### 2列（2 Columns）- ジッパー付き

| 公式 | 詳細 |
|------|------|------|
| `(H × 4) + 71` | `10 + 5 + H + 5 + 10(ジッパー) + H + 5 + 5 + H + 5 + 10(ジッパー) + H + 5 + 10(ジッパー) + H + 5 + 10 + 1` |

**例**: H=160mmの場合
```
フィルム幅 = (160 × 4) + 71 = 711mm
```

### 2-2. スタンドアップパウチ（Stand-up Pouch / スタン袋）

**配向**: 横向きで印刷（縦Hがフィルム幅を決定）

#### 1列（1列）- ジッパー付き

| 公式 | 詳細 |
|------|------|------|
| `(H × 2) + G + 35` | `1(スリッター) + 2 + G + 2 + 10(アイマーク) + 5 + H + 5 + 5 + H + 5` |

**例**: H=130mm, G=30mmの場合
```
フィルム幅 = (130 × 2) + 30 + 35 = 325mm
印刷エリア = 325mm
```

#### 2列（2列）- ジッパー付き

| 公式 | 詳細 |
|------|------|------|
| `(H × 4) + G + 40` | `1(スリッター) + 2 + G(70+70) + 2 + 10(アイマーク) + 3 + H + 3...` |

**例**: H=140mm, G=140mmの場合
```
フィルム幅 = (140 × 4) + 140 + 40 = 740mm
印刷エリア = 740mm
```

### 2-3. 合掌袋（T封・T-Bang / Center Seal）

**配向**: 展開図基準（横Wがフィルム幅を決定）

#### 1列（1列）

| 公式 | 詳細 |
|------|------|------|
| `(W × 2) + 22` | `1(スリッター) + 10 + W/2 + W + W/2 + 10 + 1(補正)` |

> **注意**: 画像セル合計は141だが、印刷エリア（イエローボックス）は142。そのため+1mm補正適用（総計22)

**例**: W=60mmの場合
```
フィルム幅 = (60 × 2) + 22 = 142mm
```

### 2-4. ボックスパウチ（M封・M-Box / M-Seal）

**配向**: 展開図基準（横Wと側面Gがフィルム幅を決定）

#### 1列（1列）

| 公式 | 詳細 |
|------|------|------|
| `(G + W) × 2 + 32` | `10(アイマーク) + 5 + G + W + G + W + 5 + 10 + 1(スリッター) + 1(補正)` |

> **注意**: 画像セル合計は311だが、印刷エリア（イエローボックス）は312。そのため+1mm補正適用（総計32)

**例**: G=90mm, W=50mmの場合
```
フィルム幅 = (90 + 50) × 2 + 32 = 312mm
```

#### 2列（2列）- （推定）

| 公式 |
|------|
| `((G + W) × 2 + 15) × 2 + 30` |

> **注意**: 正確なデータ不在のため1列ベース推定値を使用

### 2-5. スパウトパウチ（Spout Pouch）

スパウトパウチは構造的に**スタンドアップパウチ**と基本形式が同一で、上部またはコーナーにスパウト（キャップ）が付きます。フィルム幅計算時には**スタンドアップパウチ公式**を準用します。

| タイプ | 基本公式 |
|--------|----------|
| 1列 | `(H × 2) + G + 35` |
| 2列 | `(H × 4) + G + 40` |

## 3. 列数判定ロジック

### 3-1. 基本アルゴリズム

```
IF (2列必要フィルム幅 ≤ 740mm) THEN
  → 2列採用（740mm原反使用）
ELSE
  → 1列採用（適切な幅の原反使用）
END IF
```

### 3-2. 各パウチタイプの判定条件

| パウチタイプ | 1列フィルム幅 | 2列フィルム幅 | 2列採用条件 |
|-------------|---------------------------------------------|-------------|
| 三方袋 H=100 | 241mm | 471mm | **採用** (471 < 740) |
| 三方袋 H=160 | 361mm | 711mm | **採用** (711 < 740) |
| 三方袋 H=180 | 401mm | 791mm | **不採用** (791 > 740) → 1列 |
| スタンド H=130, G=30 | 325mm | 635mm | **採用** (635 < 740) |
| スタンド H=180, G=40 | 435mm | 760mm | **不採用** (760 > 740) → 1列 |

## 4. 原反選定と効率化

### 4-1. 利用可能な原反幅

| 原反幅 | 印刷可能幅 | 説明 | 最小発注量 |
|--------|-----------|------|-----------|
| 590mm | 570mm | 標準的な小・中型パウチ用 | 500m |
| 760mm | 740mm | 大型パウチまたは2列印刷用 | 500m |

> **重要**: 原材料費計算は実際の原反幅（590mm, 760mm）で計算し、印刷可能幅は両端10mmずつ除外した幅

### 4-2. 原反最小確保量と追加単位

#### 基本ルール

```
ロス（損失） = 400m（固定、SKU数に関わらず一定）

IF SKU数 = 1 THEN
  最小確保量 = 500m
  総発注量 = 必要メートル数 + 400m
ELSE IF SKU数 ≥ 2 THEN
  最小確保量 = 300m × SKU数
  総発注量 = (各SKU必要メートル数の合計) + 400m
END IF

// 50m単位で切り上げ（最小確保量超過時のみ）
実際発注量 = CEILING(必要メートル数, 50m) + 400m
```

> **重要**: ロスはSKU数や数量に関わらず**常に400m固定**

#### SKU数別最小確保量

| SKU数 | 各SKU最小確保量 | ロス（固定） | 総発注量（最小） |
|--------|-----------------|--------------|------------------|
| **1種類** | 500m | 400m | **900m** |
| **2種類** | 各300m | 400m | **1000m** (600m + 400m) |
| **3種類** | 各300m | 400m | **1300m** (900m + 400m) |
| **4種類** | 各300m | 400m | **1600m** (1200m + 400m) |

## 5. メートル数計算

### 5-1. 理論メートル数計算

```
1mあたり生産可能数 = 1,000mm ÷ ピッチ（またはフィルム長/mm）

理論メートル数 = 数量 ÷ 1mあたり生産可能数
```

### 5-2. 各パウチタイプのピッチ（フィルム進行方向）

| パウチタイプ | ピッチ | 説明 |
|-------------|----------|------|
| 平袋（三封） | W | 横方向が進行方向 |
| スタンドアップパウチ | W | 横方向が進行方向 |
| 合掌袋（T封） | W | 展開図基準（横Wがフィルム幅を決定） |
| ボックスパウチ（M封） | G + W | 展開図基準（横Wと側面Gがフィルム幅を決定） |

### 5-3. 実際発注量決定関数

```
各SKU必要メートル数 = 各SKU数量 ÷ (1000mm ÷ フィルム長/mm)

IF SKU数 = 1 THEN
  最小確保量 = 500m
ELSE
  最小確保量 = 300m
END IF

// 各SKUの確保量計算
FOR EACH SKU DO
  IF 必要メートル数 ≤ 最小確保量 THEN
    確保量 = 最小確保量
  ELSE
    // 50m単位で切り上げ
    確保量 = CEILING(必要メートル数, 50m)
  END IF
END FOR

総発注量 = (各SKU確保量の合計) + 400m（ロス固定）
```

## 6. 原材原価計算

### 6-1. 原反重量計算

```
原反重量(g) = 厚み(μm) ÷ 1,000 × 幅(m) × メートル数 × 比重
```

**重要**: 厚さはミクロン(μm)で、60μm = 0.060mm

#### 重量計算例（LLDPE 90μm、幅0.59m、メートル数900m）

```
LLDPE重量 = 90 ÷ 1,000 × 0.59m × 900m × 0.92
         = 0.090 × 0.59 × 900 × 0.92
         = 27.58kg
```

### 6-2. 原反価格計算

```
原反価格(ウォン) = 原反重量(kg) × 単価(ウォン/kg)
```

### 6-3. 原反単価情報（韓国からの輸入価格）

| 材料 | kg当たり単価(ウォン) | 比重 | 備考 |
|------|---------------------|------|------|
| AL | 7,800ウォン | 2.71 | アルミニウム |
| NY | 5,400ウォン | 1.16 | ナイロン |
| LLDPE | 2,800ウォン | 0.92 | 直封用 |
| PET | 2,800ウォン | 1.40 | ポリエステル |
| VMPET | 3,600ウォン | 1.40 | 蒸着PET |
| CPP | 2,700ウォン | 0.91 | ポリプロピレン |

### 6-4. 重要: 原反幅による重量・価格計算

| 原反幅 | 印刷可能幅 | 実際使用幅 | 説明 |
|--------|-----------|-----------|------|
| 590mm | 570mm | 0.59m | **原反幅 - 20mm = 実際使用幅** |
| 760mm | 740mm | 0.76m | **原反幅 - 20mm = 実際使用幅** |

> **超重要**: 原材料費計算は**実際の原反幅**(590mm, 760mm)で計算し、印刷可能幅は両端10mmずつ除外した幅

## 7. 印刷費用計算

### 7-1. 基本印刷料

```
印刷費用(ウォン) = 1m × 使用メートル数 × 475ウォン/m²
```

> **最も重要**: 印刷費は**フィルム幅に関係なく、常に1m**で計算します！

### 7-2. マット印刷（マット仕上げ）

マット仕上げを選択すると、印刷フィルム幅に合わせて**別途マット印刷**を行う必要があります。

#### 計算式

```
マット印刷追加費(ウォン) = 1m × 20ウォン/m × 使用メートル数
```

> **重要**: マット印刷はフィルム幅に比例して費用がかかります。

## 8. 後加工費用計算

### 8-1. ラミネート加工費

```
ラミネート費(ウォン) = 実際使用幅(m) × 使用メートル数 × ラミ単価 × ラミ回数
```

#### ラミ単価（素材別）

| 条件 | 単価 |
|------|------|
| **AL素材あり** | 75ウォン/m |
| **AL素材なし** | 65ウォン/m |

#### ラミ回数決定ルール

```
ラミ回数 = 層数 - 1
```

| 層数 | ラミ回数 |
|------|----------|
| 1層 | 0回（ラミネートなし） |
| 2層 | 1回 |
| 3層 | 2回 |
| 4層 | 3回 |
| 5層 | 4回 |

### 8-2. スリッター加工費

```
スリッター費(ウォン) = MAX(30,000ウォン, 使用メートル数 × 10ウォン)
```

### 8-3. 製袋加工費

```
基本加工費(ウォン) = パウチ横幅(cm) × 単価(ウォン/cm) × 数量
製袋加工費(ウォン) = MAX(基本加工費, 最小単価)
```

#### パウチタイプ別単価と最小単価

| パウチタイプ | 単価(ウォン/cm) | 最小単価（ウォン） |
|--------------|-----------------|-------------------|
| 平袋（三封） | 0.4ウォン | 200,000ウォン |
| 合掌袋（T封） | 1.2ウォン | 440,000ウォン |
| スタンドアップ | 1.2ウォン | 250,000ウォン |
| ジッパー袋 | 1.2ウォン | 250,000ウォン |
| ジッパー付きスタンド | 1.7ウォン | 280,000ウォン |
| ボックスパウチ（M封） | 1.2ウォン | 440,000ウォン |

## 9. 関税と配送料

### 9-1. 関税（輸入関税）

```
関税込み価格(ウォン) = 製造者価格 × 1.05
```

> **1.05 = 1 + 0.05（5%関税）**

### 9-2. 配送料計算（韓国国内配送料）

#### 重量ベース配送料（実績 + 8%余裕）

| 種別(kg) | 配送料（ウォン） |
|----------|-----------------|
| 5.0 | 46,440ウォン |
| 10.0 | 64,800ウォン |
| 15.0 | 82,620ウォン |
| 20.0 | 98,820ウォン |
| 25.0 | 115,020ウォン |
| 29.0（1箱最大） | **127,980ウォン** |

> **重要**: 実績価格に8%の余裕を追加した保守的見積もり
> **重要**: 29kgが1箱の最大。1kgは梱包材の余裕分。29kg超過の配送は不可

#### 配送箱数計算

```
配送箱数 = CEILING(総重量 ÷ 29kg)  [切り上げ]
配送料(ウォン) = 配送箱数 × 127,980ウォン/箱
```

> **重要**: 各ロールは29kg以下に制限。超過する場合は分割必要

## 10. 総合価格計算

### 10-1. 完全な価格計算構造

```
1. 基礎原価 = 原材料費 + 印刷費 + 後加工費
   ↓
2. 製造者価格 = 基礎原価 × 1.4（製造者マージン40%追加）
   ↓
3. 関税込み価格 = 製造者価格 × 1.05（関税5%追加）
   ↓
4. 配送料追加（マージン計算対象外）
   ↓
5. 輸入原価 = 関税込み価格 + 配送料
   ↓
6. 最終販売価格 = 輸入原価 × 1.2（販売者マージン20%追加）
   ↓
7. 円貨換算 = 最終販売価格 × 0.12（為替レート適用）
```

> **最も重要**: 配送料はマージン計算の対象外です。製造者マージン40%と販売者マージン20%の適用範囲には入りません。

### 10-2. 各ステップの詳細

| ステップ | 項目 | 計算式 | 備考 |
|----------|------|----------|------|
| **①** | 原材料費 | 各層の重量 × 単価の合計 | 4層構造の場合、PETが2回使用 |
| **②** | 印刷費 | **1m** × メートル数 × 475ウォン/m² | **重要**: フィルム幅に関係なく、常に1mで計算 |
| **③** | 後加工費 | ラミ + スリッター + 製袋 + オプション | ラミ回数 = 層数 - 1 |
| **④** | 基礎原価 | ① + ② + ③ | マージン計算の基礎 |
| **⑤** | 製造者マージン | ④ × 0.4 | 基礎原価の40% |
| **⑥** | 製造者価格 | ④ + ⑤（または ④ × 1.4） | 関税適用前の価格 |
| **⑦** | 関税込み価格 | ⑥ × 1.05 | 関税5%適用 |
| **⑧** | 配送料 | 箱数 × 127,980ウォン | **マージン対象外** |
| **⑧** | 輸入原価 | ⑦ + ⑧ | 配送料追加後 |
| **⑧** | 販売者マージン | ⑧ × 0.2 | 輸入原価の20% |
| **⑨** | 最終販売価格 | ⑧ + ⑧（または ⑧ × 1.2） | ウォン建て価格 |
| **⑩** | 円貨換算 | ⑨ × 0.12 | 日本円での価格 |

### 10-3. 配送料の重要なポイント

```
❌ 誤った計算: 配送料にマージンを適用
   (基礎原価 + 配送料) × 1.4 × 1.05 × 1.2

✅ 正しい計算: 配送料はマージン適用後に追加
   (基礎原価 × 1.4 × 1.05) + 配送料) × 1.2
```

**理由**: 配送料は外部サービス費用であり、製造者や販売者の利益に含まれないため、マージン計算の対象外となります。

## 11. 価格計算のまとめ

### 11-1. 計算の流れ

```
1. 原材料費計算（各層の重量 × 単価の合計）
   ↓
2. 印刷費計算（1m × メートル数 × 475ウォン/m²）
   ↓
3. 後加工費計算（ラミ + スリッター + 製袋 + オプション）
   ↓
4. 基礎原価の算出
   ↓
5. 製造者マージン40%適用 → 製造者価格
   ↓
6. 関税5%適用 → 関税込み価格
   ↓
7. 配送料追加（マージン計算対象外）
   ↓
8. 輸入原価の算出
   ↓
9. 製売者マージン20%適用 → 最終販売価格
   ↓
10. 円貨換算（× 0.12）
```

### 11-2. 原反幅計算のポイント

| 原反幅 | 実際使用幅 | 計算式 |
|--------|-----------|----------|
| **590mm原反** | 0.59m | 原反幅 - 20mm = 実際使用幅 |
| **760mm原反** | 0.76m | 原反幅 - 20mm = 実際使用幅 |

### 11-3. 重要ポイントまとめ

| 項目 | 設定値 | 説明 |
|------|--------|------|
| **印刷費** | 常に **1m**で計算 | フィルム幅と無関係 |
| **ロス（損失）** | **400m固定** | SKU数や数量に関わらず一定 |
| **1SKU最小確保量** | 500m | 1SKUの場合 |
| **2SKU以上最小確保量** | 各300m | 2SKU以上の場合、各300m |
| **総発注量** | 確保量の合計 + 400m | ロスは400m固定 |
| **ラミ単価（ALあり）** | 75ウォン/m | AL素材使用時 |
| **ラミ単価（ALなし）** | 65ウォン/m | AL素材なしの場合 |
| **配送料** | 127,980ウォン/箱 | 29kg/箱の最大 |

## 12. 製造者マージンと販売者マージン

### 12-1. マージン率

| マージン | 率 | 適用対象 | 説明 |
|----------|-----|----------|------|
| **製造者マージン** | 40% | 基礎原価 | 韓国製造者の利益 |
| **関税** | 5% | 製造者価格 | 日本輸入時の関税 |
| **販売者マージン** | 20% | 輸入原価 | 日本販売者の利益 |
| **配送料** | - | 重量ベース | **マージン対象外** |

### 12-2. 配送料の位置づけ

```
1. 基礎原価（原材料費 + 印刷費 + 後加工費）
   ↓
2. 製造者価格（基礎原価 × 1.4）
   ↓
3. 関税込み価格（製造者価格 × 1.05）
   ↓
4. 配送料追加（ここで初めて追加）
   ↓
5. 輸入原価（関税込み価格 + 配送料）
   ↓
6. 最終販売価格（輸入原価 × 1.2）
```

> **最も重要**: 配送料はマージン計算の対象外です。製造者マージン40%と販売者マージン20%の適用範囲には入りません。

## 13. 計算例：平袋（三封）H=160mm × W=100mm × 10,000個

### 13-1. 製品情報

## 14. 重要ポイントまとめ

### 14-1. 基本ルール

1. **印刷費は常に1mで計算**: フィルム幅과 상관없이 **항상 1m**로 계산합니다.
2. **ロス는 400m 고정**: SKU수나 수량과 무관하게 항상 400m가 추가됩니다.
3. **원재료비는 원반 폭으로 계산**: 590mm → 0.59m, 760mm → 0.76m
4. **배송료는 마진 대상외**: 기초 원가 → 제조사 마진 → 관세 → 배송료 추가 → 판매자 마진

### 14-2. よ型製品別の計算違い

| 製品タイプ | フィルム幅計算 | ピッチ | 主要な後加工 |
|-------------|--------------|------|------------|
| **平袋（三封）** | `(H × 2) + 41` | W | 製袋加工 |
| **スタンドアップ** | `(H × 2) + G + 35` | W | ジッパー取り付け |
| **合掌袋（T封）** | `(W × 2) + 22` | W | 製袋加工 |
| **ボックスパウチ（M封）** | `(G + W) × 2 + 32` | G + W | 製袋加工 |
| **スパウトパウチ** | `(H × 2) + G + 35` | W | スパウト取り付け |

### 14-3. 価格構造の完全なフロー

```
① 原材料費計算
   ↓
② 印刷費計算（常に1mで計算）
   ↓
③ 後加工費計算（ラミ + スリッター + 製袋）
   ↓
④ 基礎原価算出（① + ② + ③）
   ↓
⑤ 製造者価格算出（基礎原価 × 1.4）
   ↓
⑥ 関税込み価格算出（製造者価格 × 1.05）
   ↓
⑦ 配送料算出（重量ベース）
   ↓
⑧ 輸入原価算出（⑦ + 配送料）
   ↓
⑧ 最終販売価格算出（輸入原価 × 1.2）
   ↓
⑧ 円貨換算（最終販売価格 × 0.12）
```
