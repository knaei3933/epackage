# 並列印刷廃止と新価格政策提案

> **作成日**: 2026-01-17
> **目的**: 並列印刷（Parallel Production）を廃止し、顧客に透明で公平な価格体系を提供する

---

## 1. 現状分析

### 1-1. 並列印刷の問題点

**現行の並列印刷モデル**:

| オプション | 並列本数 | 原反幅 | 総フィルム長 | 顧客への単価 | 製造者コスト | 利益 |
|-----------|----------|--------|-------------|-------------|-------------|------|
| 従来 | 1本 | 590mm | 900m | 高い | 高い | 通常 |
| 並列印刷 | 2本 | 590mm | 450m | **高い（同じ）** | **低い（半減）** | **増加** ⭐ |
| 並列印刷 | 3本 | 760mm | 300m | **高い（同じ）** | **更低い（1/3）** | **大幅増加** ⭐⭐ |

**核心的問題**:
```
顧客が支払う金額 = 変わらない
製造者コスト = 並列印刷で大幅に削減
利益 = 並列印刷で製造者が独占
```

### 1-2. 顧客視点の不公平性

**顧客の疑問**:
- 「なぜ2本注文したのに、1本注文と同じ価格なのか？」
- 「3本並列生産のコストメリットは、どこへ行ったのか？」
- 「製造者は並列生産でコスト削減しているのに、価格に反映されない」

**現在の価格決定ロジック**:
```typescript
// 並列生産でも単価は変わらない
estimatedUnitCost = currentUnitPrice;  // 顧客には同じ価格

// 実際の製造コスト
actualCost = baseCostPerMeter / parallelCount;  // 製造者は節約
```

## 2. 新価格政策の提案

### 2-1. 基本理念

```
【原則】並列印刷を廃止し、数量ベースの透明な価格体系へ移行
```

**基本方針**:
1. **並列印刷の完全廃止**: システムから並列生産オプションを削除
2. **数量割引の明確化**: 注文数量に応じた直感的な割引を適用
3. **透明性の確保**: 顧客が価格決定ロジックを理解できるようにする
4. **利益の適正化**: 製造者の利益を適正レベルに維持しつつ、顧客にもメリットを還元

### 2-2. 新価格モデル

#### **数量割引モデル（Quantity-Based Pricing）**

| 注文数量（総数） | 割引率 | 適用理由 |
|-----------------|--------|----------|
| 100-999個 | 0%（基準価格） | 最小発注量 |
| 1,000-4,999個 | 5% | 小ロット |
| 5,000-9,999個 | 10% | 中ロット |
| 10,000-29,999個 | 15% | 大ロット |
| 30,000個以上 | 20% | 特大ロット |

**SKU別数量の集計ルール**:
```
総注文数量 = Σ 各SKUの数量
```

**例**: 3 SKUで各5,000個注文
```
総注文数量 = 5,000 + 5,000 + 5,000 = 15,000個
適用割引 = 15%（大ロット）
```

#### **SKU数による価格調整**

| SKU数 | 調整率 | 適用理由 |
|-------|--------|----------|
| 1 SKU | 0% | 基準 |
| 2-3 SKU | +5% | 複数デザインの管理コスト |
| 4-10 SKU | +10% | 多数デザインの管理コスト |
| 11+ SKU | +15% | 特別対応 |

**論理的根拠**:
- 複数SKUは印刷版交換、品質管理、在庫管理のコストが増加
- 並列印刷廃止により、SKUごとの生産管理が明確化
- 顧客は「SKU数が増えると価格が上がる」ことを直感的に理解

### 2-3. 最終価格計算式

```
基本価格 = 材料費 + 印刷費 + 加工費 + 関税 + 配送料

数量割引後価格 = 基本価格 × (1 - 数量割引率)

SKU調整後価格 = 数量割引後価格 × (1 + SKU調整率)

最終販売価格 = SKU調整後価格 × 販売マージン(1.2) × 為替レート(0.12)
```

**計算例**:

| パラメータ | 値 |
|-----------|-----|
| 基本価格 | 100,000円 |
| 総注文数量 | 15,000個（3 SKU × 5,000個） |
| 数量割引 | 15% |
| SKU数 | 3 SKU |
| SKU調整 | +5% |
| 販売マージン | 1.2 |

```
数量割引後価格 = 100,000 × (1 - 0.15) = 85,000円
SKU調整後価格 = 85,000 × (1 + 0.05) = 89,250円
最終販売価格 = 89,250 × 1.2 = 107,100円
```

### 3-2. UI変更

**削除**:

- 新規注文から新価格体系を適用

**価格履歴の保存**:
```sql

-- 値: 'legacy', 'quantity_based'

ALTER TABLE quotations ADD COLUMN quantity_discount_rate DECIMAL(5,2);
ALTER TABLE quotations ADD COLUMN sku_premium_rate DECIMAL(5,2);
```

## 4. 顧客コミュニケーション

### 4-1. アナウンス文面

```
価格体系の改善についてお知らせ

お客様へ

Epackage Labでは、より透明で公平な価格体系を提供するため、
2026年2月より新しい価格モデルを導入いたします。

【主な変更点】
1. 並列生産オプションの廃止
   わかりやすさ向上のため、専門的なオプションを廃止しました

2. 数量割引の強化
   注文数量に応じた直感的な割引を適用します
   - 1,000個以上: 5%割引
   - 5,000個以上: 10%割引
   - 10,000個以上: 15%割引
   - 30,000個以上: 20%割引

3. SKU価格の明確化
   複数デザイン（SKU）ごとの注文に対応するため、
   少額の管理プレミアムを設定いたします

これにより、お客様はよりシンプルで理解しやすい価格体系で、
パッケージ製品をご注文いただけます。

詳細なお見積もりについては、お気軽にお問い合わせください。
```

### 4-2. よくある質問（FAQ）

**Q1. 並列生産オプションがなくなったので、価格が上がるのですか？**

A: いいえ。新しい数量割引体系により、多くのお客様が
従来よりもお得な価格でご注文いただけます。

**Q2. 複数SKUを注文すると割高になるのですか？**

A: 複数SKUの管理には追加コストがかかるため、
少額のプレミアム（5-15%）を設定しております。
しかし、数量割引と相殺されるケースがほとんどです。

**Q3. 以前よりお得に注文できる方法は？**

A: 複数SKUをまとめて注文することで、総数量が増え、
より高い割引率が適用されます。

**シナリオ2: 中ロット顧客（総数15,000個、3 SKU）**

| 費用項目 | 適正率 | 旧モデル（並列印刷時） | 新モデル |
