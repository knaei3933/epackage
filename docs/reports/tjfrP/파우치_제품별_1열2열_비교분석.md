## 1. 평袋 80mm × 120mm × 10,000개 예시

### 1-1. 조건/입력값

| 항목 | 값 |
|------|-----|
| 제품타입 | 평袋 (3방/Flat Pouch) |
| 사이즈 | **W=80mm × H=120mm** |
| 수량 | 10,000개 |
| 필름구조 | PET12μ/AL7μ/LLDPE90μ (3층) |
| 마치(Gusset) | 없음 |
| 지퍼 | 없음 |

### 1-2. 필름 폭 계산

#### 1열 생산

```
필름 폭 = (H × 2) + 41
        = (120 × 2) + 41
        = 240 + 41
        = 281mm

원단 선택: 590mm (실제 원단 폭)
인쇄 가능 폭: 570mm (양쪽 10mm 여백 제외)
필름 효율: 281mm / 570mm = 49.3%
```

#### 2열 생산

```
필름 폭 = (H × 4) + 71
        = (120 × 4) + 71
        = 480 + 71
        = 551mm

원단 선택: 760mm (실제 원단 폭)
인쇄 가능 폭: 740mm (양쪽 10mm 여백 제외)
필름 효율: 551mm / 740mm = 74.5%
```

**✅ 2열 생산 가능 판정**: 551mm ≤ 740mm → **2열 채택**

### 1-3. 단계별 계산 과정

#### Step 1: 필요 미터 수 계산

```
피치(Pitch) = W = 80mm
1m당 생산 가능 수 = 1000mm ÷ 80mm = 12.5개/m
이론 미터 수 = 10,000개 ÷ 12.5개/m = 800m
50m 단位 올림 = 800m (이미 50의 배수)
최소 확보량 초과 = 800m > 500m ✓
로스(고정) = 400m
실제 발주량 = 800m + 400m = 1,200m
```

## 4. 가격 구조 검증

### 4-1. 마진 구조 확인

```
계산가이드의 마진 구조:
(기초원가 × 1.4 × 1.05 + 배송비) × 1.2 × 0.12

해석:
1. 기초원가 × 1.4 = 제조자 마진 40% 포함
2. × 1.05 = 관세 5% 포함
3. + 배송비 = 배송비 추가 (마진 제외)
4. × 1.2 = 판매자 마진 20% 포함
5. × 0.12 = 엔화 환전
```

### 4-2. 수입원가에서 20% 마진 유지 조건

| 조건 | 계산식 | 결과 |
|------|--------|------|
| **기초 원가** | 재료비 + 인쇄비 + 후가공비 | ① |
| **제조자 마진 40%** | ① × 0.4 | ② |
| **제조자 가격** | ① + ② = ① × 1.4 | ③ |
| **관세 5%** | ③ × 0.05 | ④ |
| **관세 포함 가격** | ③ + ④ = ③ × 1.05 | ⑤ |
| **배송비 추가** | ⑤ + 배송비 | ⑥ |
| **판매자 마진 20%** | ⑥ × 0.2 | ⑦ |
| **최종 판매가** | ⑥ + ⑦ = ⑥ × 1.2 | ⑧ |
| **엔화 환전** | ⑧ × 0.12 | ⑨ |

**✅ 검증 결과**: 마진 구조 정확히 구현됨

### 4-3. 병렬 생산 시 마진율 변화

**💡 핵심 인사이트**:
- **마진율은 항상 20% 고정**: 판매자 마진은 수입 원가의 20%
- **병렬 생산 시 마진율 변화 없음**: 기초 원가가 증가하더라도 마진율은 동일

| 원당 단가 절감 | 배송비 증가 (760mm 원단) |
| 재주문 시 원가 경쟁력 | 1회성 주문에는 총액 비싼 |
| 대량 생산에 적합 | 소량 생산에는 비효율적 |

#### 💡 병렬 생산의 장단점

| 장점 | 단점 |
|------|------|
| 필름 효율 90%+ 달성 | 최소 주문 수량 증가 |
| 단가 20%+ 절감 | 재고 부담 증가 |
| 합장/박스 제품에 필수 | 1회성 주문에는 부적절 |
| 재고 안정 | 선납 비용 증가 |
