# Epackage Lab 홈페이지 E2E 테스트 시나리오

**문서 버전**: 1.0
**작성일**: 2026-01-14
**대상 URL**: http://localhost:3000
**테스트 범위**: Epackage Lab 메인 홈페이지 (/)

---

## 개요 (Overview)

Epackage Lab 홈페이지는 일본어 B2B 패키징 솔루션 웹사이트의 메인 진입점입니다. 본 테스트 시나리오는 홈페이지의 모든 주요 기능, UI 컴포넌트, 사용자 인터랙션, 반응형 디자인, 접근성, 성능을 포괄적으로 검증하기 위해 작성되었습니다.

### 테스트 대상 섹션

1. **헤더 & 네비게이션 (Header & Navigation)**
   - 로고, 메인 네비게이션, 언어 선택기, 인증 UI, 모바일 메뉴
2. **공지사항 배너 (Announcement Banner)**
   - 동적 공지사항 표시 (조건부 렌더링)
3. **히어로 섹션 (Hero Section)**
   - 메인 헤드라인, 통계 수치, CTA 버튼, 신뢰 지표, 배지
4. **제품 쇼케이스 섹션 (Product Showcase Section)**
   - 추천 제품 카드, 제품 그리드, 카탈로그 링크
5. **제조 공정 섹션 (Manufacturing Process Showcase)**
   - 4단계 제조 공정 타임라인, 품질 통계, 인증 정보
6. **CTA 섹션 (Call-to-Action Section)**
   - 4개 CTA 카드 (카탈로그, 가격 계산, 샘플, 문의)
7. **푸터 (Footer)**
   - 회사 정보, 프라이버시 링크, 소셜 미디어, 뉴스레터

### 테스트 도구

- **Playwright**: E2E 테스트 자동화
- **브라우저**: Chromium, Firefox, WebKit
- **디바이스**: Desktop, Tablet, Mobile

---

## 선행 조건 (Prerequisites)

### 환경 설정

1. **개발 서버 실행**
   ```bash
   npm run dev
   ```
   서버는 `http://localhost:3000`에서 실행 중이어야 합니다.

2. **환경 변수 설정**
   - `.env.local` 파일에 필요한 환경 변수 설정
   - Supabase 연결 (공지사항 데이터 로드용)

3. **데이터베이스 상태**
   - `announcements` 테이블에 테스트용 공지사항 데이터 있음 (선택사항)
   - `products` 테이블에 추천 제품 데이터 있음

### 브라우저 설정

- JavaScript 활성화
- 쿠키 허용
- 팝업 차단 해제 (테스트용)
- 화면 해상도: 1920x1080 (Desktop), 375x667 (Mobile)

### TC-HOME-001: 로고 클릭 시 홈페이지로 이동

**설명 (Description)**:
- 일본어: 로고를 클릭하면 홈페이지(루트 경로)로 이동하는지 확인
- 한국어: 로고 클릭 시 홈페이지로의 올바른 네비게이션 검증

- 로고가 표시되어 있음

- URL이 `http://localhost:3000` 또는 `/`로 유지됨

**성공 기준**:

- 페이지가 정상적으로 로드됨

import { test, expect } from '@playwright/test';

test.describe('Homepage - Basic Tests', () => {
  test.beforeEach(async ({ page }) => {
    await page.goto('http://localhost:3000');
    await page.waitForLoadState('domcontentloaded');
  });

  test('TC-HOME-001: Logo navigates to homepage', async ({ page }) => {
    const logo = page.locator('a').filter({ hasText: 'Epackage' }).first();
    await expect(logo).toBeVisible();
    await logo.click();
    expect(page.url()).toContain('http://localhost:3000');
  });

  test('TC-HOME-002: Navigation links work', async ({ page }) => {
    const catalogLink = page.getByRole('link', { name: '製品カタログ' });
    await catalogLink.click();
    expect(page.url()).toContain('/catalog');
  });
});
