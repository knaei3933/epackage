## 2. 서비스 소개

### 2.1 핵심 서비스

Epackage Lab은 고객의 니즈에 맞는 최적의 포장 솔루션을 제공합니다.

#### 제품 라인업

| 제품 | 특징 | 주요 용도 |
|------|------|----------|
| **플랫 파우치** | 가성비 우수 | 식품, 건강기능식품 |
| **스탠드 파우치** | 바닥 지퍼로 진열성 탁월, 산소 차단성 및 광 차단성 우수 | 화장품, 건강기능식품, 스낵 |
| **필로우 파우치** | 액체・분말 제품 대응 | 샴푸, 소스, 분유, 영양제 |
| **거싯 파우치** | 바닥에 지퍼가 있는自立型 파우치, 용량 확보와 진열 효과를 동시에 실현 | 커피 원두 |
| **스퍼트 파우치** | 주입구가 있어 액체 누출 방지 | 화장품, 식품 (일회용 타입) |
| **롤 필름** | 자동 충전기용, 각종 충전기 제조사 대응 | 자동 충전 라인 |

### 2.2 차별화 포인트

#### ① AI 견적 시뮬레이터 (2초 만에 즉시 답변)

기존 3~7일 소요되던 견적을 AI가 **2초** 만에 즉시 가격 제시.

- 견적액: ¥150,000~¥500,000
- 회원가입 없이 24시간 365일 대응 가능
- 복수 조건 시뮬레이션 가능

#### ② 12단계 완전 디지털 워크플로우

견적부터 납품까지 전 프로세스를 온라인에서 완결.

```
견적 의뢰 → AI 가격 제시 → 사양 확정 → 디자인 업로드 →
교정 → 승인 → 제조 → 품질 검사 → 출하 → 배송 → 완료
```

- 진행 상황을 실시간으로 시각화
- 이메일・SMS 자동 알림
- 모두 온라인에서 완결

#### ③ 소량 대응 (500장~)

스타트업, D2C 브랜드, 중소기업에 최적.

- 최소 500장부터 대응 가능
- 재고 리스크 최소화
- 신제품 테스트 마케팅에 활용

#### ④ 한국 파트너와의 품질×코스트 퍼포먼스

한국의 우수한 인쇄 기술과 일본의 품질 관리를 융합.

- 대형 인쇄사보다 **30~50% 저가** 실현
- 일본 식품 안전 규격 완전 대응
- 품질과 비용의 최적 밸런스

## 3. 제품 정보

### 3.1 제품 카탈로그

#### 플랫 파우치 (Flat Pouch)
- 식품・건강기능식품에 최적인 기본 포장재
- 가성비 우수
- 심플한 디자인으로 범용성 높음

#### 스탠드 파우치 (Stand Pouch)
- 바닥 지퍼 타입으로 진열성 탁월
- 산소 차단성, 광 차단성 우수
- 화장품, 건강기능식품, 스낵에 최적

#### 필로우 파우치 (Pillow Pouch)
- 액체・분말 제품에 최적
- 샴푸, 소스, 분유, 영양제 등 대응
- 충전기 대응 가능

#### 거싯 파우치 (Gusset Pouch)
- 커피 원두 등, 바닥에 지퍼가 있는自立형 파우치
- 용량 확보와 진열 효과를 동시에 실현
- 광고 면적이 큼

#### 스퍼트 파우치 (Spout Pouch)
- 주입구가 있어 액체 누출 방지
- 화장품・식품용
- 일회용 타입에 최적

#### 롤 필름 (Roll Film)
- 자동 충전기용
- 각종 충전기 제조사 대응 가능
- 대량 생산 대응

### 3.2 적용 산업

| 산업 | 대응 제품 | 특징 |
|------|----------|------|
| **화장품** | 스탠드 파우치, 스퍼트 파우치 | 고품질 인쇄, 배리어성 |
| **건강기능식품** | 플랫 파우치, 스탠드 파우치 | 식품 위생법 대응 |
| **영양제** | 필로우 파우치, 스탠드 파우치 | 소량 대응 |
| **식품** | 전 제품 | 식품 안전 규격 대응 |
| **펫 푸드** | 거싯 파우치, 스탠드 파우치 | 내구성・보존성 |

## 4. 기술 혁신

#### 24시간 365일 견적 대응

- AI가 2초 만에 가격 제시

- 12단계 워크플로우 시각화
- 이메일・SMS 자동 알림
- 수정 이력 완전 기록

#### 디지털 서명 대응
- Hanko 연계에 의한 전자 서명
- 계약 절차의 완전 온라인화
- 법적으로 유효한 서명

### 4.2 기술 스택

#### 프론트엔드
- **Next.js** 16 - 최신 React 프레임워크
- **React** 19 - UI 라이브러리
- **TypeScript** - 타입 안전 개발
- **Tailwind CSS** - 모던 스타일링

#### 백엔드
- **Supabase** (PostgreSQL) - 데이터베이스
- **Server Actions** - 서버 사이드 처리

#### 인증・보안
- **Supabase Auth** - 인증 기반
- **Hanko** - 전자 서명

#### 문서 생성
- **PDF 생성** - 견적서, 계약서, 사양서 자동 생성
- **Excel 출력** - 데이터 내보내기 기능

### 4.3 품질 관리

### 미션

- ✓ 고객 만족도 최대화
