## 1. 計算の概要（ロール・全体フロー）

グラビア印刷は、原反フィルム（ロール）に**グラビア印刷 → ラミネート**を施し、**ロールまま納品**（袋加工しない）する方式。銅版（동판）製作が必須でセットアップ費が大きい分、**中〜大ロットで有利**。

```
┌─────────────────────────────────────────────────────────────────┐
│        グラビア ロールフィルム 見積計算フロー（ウォン建て）         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  0. 原反幅の決定 = 計算で決定 → 10mm単位で丸め（§3）             │
│                                                                 │
│  1. 製作長の確定 = 注文長 + ロス500m（§4）                       │
│                                                                 │
│  2. 原材料費 = Σ(厚み × 原反幅 × 製作長 × 比重 × 単価)（§5）     │
│     └─ 最終熱シール層は幅 +10mm                                 │
│                                                                 │
│  3. 工程費 = 印刷費 + ラミネート費（§6）                         │
│     ※ ロールは袋加工なし（加工費・列数・パウチタイプは不要）       │
│                                                                 │
│  4. 製造原価 = 原材料費 + 工程費                                 │
│                                                                 │
│  5. 製造単価(m) = 製造原価 ÷ 注文長                              │
│                                                                 │
│  6. 納品単価(m) = 製造単価 × (1 + マージン率)（通常 ×1.2）（§8） │
│                                                                 │
│  7. 銅版費 = 色数 × (銅版幅 × 単価 × 外径)（別途計上）（§9）     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

> **注意**: 元Excelに為替換算・関税は含まれない（ウォン建て）。円貨納品にする場合は §12 の変換を追加。

## 3. 原反幅の決定（10mm単位・最大1,100mm）

**グラビアの原反幅は固定ではなく、製品仕様から計算し、必要な幅に合わせて10mm単位で製作する（無駄を最小化して製作単価を節減）。最大1,100mm。**

### 共通ルール
- **最小 500mm**（小パウチでも最低500mmは使用）
- **最大 1,100mm**
- 10mm単位で丸め
- **最終熱シール層（LLDPE等）は原反幅 + 10mm**（両端5mmずつ長い）
- 銅版幅 = 原反幅 + 100mm

### ロールフィルムの場合

```
原反幅 = ロール仕様幅（顧客指定）+ 端代 → 10mm単位、最大1,100mm
```

### パウチの場合（タイプ別フィルム幅公式）

パウチは**タイプ別の公式**で1列フィルム幅を計算し、1,100mmに収まる列数を選んで原反幅を決定。

#### 1列フィルム幅公式（docs/reports/calcultae/02-필름폭_계산공식.md 準拠）

| パウチタイプ | 1列フィルム幅公式 | 寸法の意味 | 印刷配向 |
|---|---|---|---|
| 平袋（三封）| `(H × 2) + 41` | H=長さ | 横向き |
| **スタンド** | `(H × 2) + (G × 2) + 35` | H=長さ、G=マチ(片面) | 横向き |
| 合掌（T封）| `(W × 2) + 22` | W=幅 | 展開図 |
| ガゼット（M封）| `(G + W) × 2 + 32` | G=側面、W=幅 | 展開図 |
| スパウト（マチなし）| `(H × 2) + 41` | H=長さ | 横向き |
| スパウト（マチあり）| `(H × 2) + (G × 2) + 35` | H=長さ、G=マチ | 横向き |

```
列数 = floor(1,100mm ÷ 1列フィルム幅)
原反幅 = MAX(1列フィルム幅 × 列数, 500mm) → 10mm単位（最小500mm・最大1,100mm）
```

#### マチ（G）の決定（docs/밑지.md 準拠）

スタンドパウチは**パウチ幅に応じて使用可能マチが決まる**（一部）：

| パウチ幅 | 使用可能マチ(mm) |
|---|---|
| R100 | 30 |
| R130 | 30, 35, 37.5, 40, 45 |
| R150 | 30, 35, 40, 42.5, 45 |
| R200 | 30, 42.5, 50, 55, 60 |
| R280 | 50, 60, 62.5, 75 |

### ピッチ（パウチタイプ別）⭐

**加工個数計算の基準（ピッチ）はパウチタイプで異なる**：

| パウチタイプ | ピッチ | 加工単価基準 |
|---|---|---|
| 三方（삼방）/ スタンド | **パウチ幅（W）** | 가로CM（幅）|
| 合掌（T封）/ ガゼット（M封）| **パウチ長さ（H）** | 세roCM（長さ）|

> - 三方・スタンド：横向き印刷 → 長さHがフィルム幅、**幅Wがピッチ**
> - 合掌・ガゼット：展開図基準 → 幅Wがフィルム幅、**長さHがピッチ**

## 4. ロス量（500m込み・納品は5,500m）

グラビアは **6000m製作** が標準ロット。**この6000mの中にロス500mが含まれる**（ロスは「追加」ではなく「製作長に含まれる」）。

```
製作6,000m（標準ロット） = 顧客提供5,500m + ロス500m
```

- **製作6,000m**：原材費・工程費は **6,000m分** で計算
- **顧客提供5,500m**（6,000 − ロス500m）：売上（納品）の基準
- **約10%の誤差**で歩留まりが変動し、納品数が変動する可能性
- ※ デジタル（現システム）が「率0.3〜0.4」に対し、グラビアは**製作長にロス500m込み**（原価は6,000m分、売りは5,500m分）

## 5. 原材料費計算

### 計算式

各原材層（PET / AL / NY / LLD 等）ごとに計算し、合計する。

```
層費用 (₫) = 厚み(mm) × 原反幅(m) × 製作長(m+500) × 比重(g/cm³) × 単価(₫/kg)

原材料費合計 = Σ(各層費用)
```

### 最終熱シール層の幅 +10mm（重要）

```
最後に積層する熱シール層（LLDPE等）は、印刷フィルムより +10mm 幅広
→ 原反幅 + 0.01m で計算
```

> **理由**: 積層（ラミネート）時の位置ずれ・エッジロス補正。裁断後に端が綺麗に揃う。
> **層数に関わらず、最終積層層（通常LLDPE）のみ +10mm**（2層/3層/4層すべて同様）。

### 原材マスター（比重・厚み・単価）

| 原材 | 比重 | 標準厚み(mm) | 単価(₫/kg) |
|---|---|---|---|
| **AL**（アルミ）| 2.71 | 0.006〜0.007 | 7,800 |
| **NY**（ナイロン）| 1.16 | 0.015 | 5,400 |
| 양면NY | 1.16 | 0.015 | 5,400 |
| **LLDPE** | 0.92 | 0.05〜0.09 | 2,800 |
| **PET** | 1.4 | 0.012 | 2,800 |
| **VMPET** | 1.4 | 0.012 | 3,600 |
| **CPP** | 0.91 | — | 2,700 |

## 6. 工程費計算（印刷・ラミネート）

### 6.1 印刷費（인쇄）

```
印刷費 (₫) = 原反幅(m) × 製作長(m+500) × 色数 × 印刷単価(₫/m)
```

| 印刷種類 | 単価 (₫/m) |
|---|---|
| 무광（無光/マット）| 19 |
| 유광（有光/グロス）| 19 |

### 6.2 ラミネート費（D/L）

```
ラミネート費 (₫) = 原反幅(m) × 製作長(m+500) × ラミ回数 × ラミ単価(₫/m)
ラミ回数 = 層数 − 1
```

| ラミネート種類 | 単価 (₫/m) |
|---|---|
| 용제 D/L 2液型ハイ | 75 |
| 용제 D/L 2液型セミ | 65 |
| 라말（無溶剤）| 55 |

### 6.3 加工費

- **ロールフィルム**: 袋加工しないため加工費は **不要**
- **パウチ**: 袋加工費を計算（§6.4 参照）

### 6.4 パウチ加工費（袋加工の場合のみ）

#### 加工個数

```
加工個数 = (製作6,000m − ロス500m) ÷ ピッチ × 列数 × 0.93
         = 5,500m ÷ ピッチ × 列数 × 0.93
```

- **ピッチ**はパウチタイプ別（§3）: 三方/スタンド=**パウチ幅**、合掌/ガゼット=**パウチ長さ**
- **0.93** = 加工歩留まり（袋成形ロス7%。ロス500mとは**別適用**）
- ※ ロス500mは製作長（ロール）のロス、0.93は袋成形時の歩留まり。両方適用。

#### 加工単価（パウチタイプ別）

```
加工単価 = 係数 × 寸法(cm)
```

| パウチタイプ | 係数 | 基準寸法 | 最小加工費(₫) |
|---|---|---|---|
| 삼방파우치（三方）| 0.4 | 가roCM（幅）| 200,000 |
| 스탠드파우치（スタンド）| 1.2 | 가roCM（幅）| 250,000 |
| 지퍼파우치（ジッパー）| 1.2 | 가roCM（幅）| 250,000 |
| 지퍼스탠드（ジッパースタンド）| 1.7 | 가roCM（幅）| 280,000 |
| T방파우치（T方）| 1.2 | 세roCM（長さ）| 440,000 |
| M방파우치（M方）| 1.2 | 세roCM（長さ）| 440,000 |

#### 加工費

```
加工費 = MAX(加工個数 × 加工単価, 最小加工費)
```

#### 追加オプション

- **バルブ加工費** = 加工個数 × バルブ単価
- **ジッパー追加費**（タイプ別: 三方50,000 / スタンド30,000 等）
- **運賃**

## 7. 製造原価・製造単価

```
製造原価 (제조원가) = 原材料費 + 印刷費 + ラミネート費

製造単価(m) = 製造原価 ÷ 顧客提供長(5,500m)
```

> ※ 製造原価は**製作6,000m分**（ロス込）。売りは**顧客提供5,500m分**。ロス分(500m)は原価に含むが売上には含まない。

---

## 8. 納品単価（マージン適用）

```
納品単価(m) (납품단가) = 製造単価(m) × (1 + 마진율)
```

| マージン率 | 乗数 |
|---|---|
| 10% | ×1.10 |
| 15% | ×1.15 |
| **20%（標準）** | **×1.20** |
| 25% | ×1.25 |
| 30% | ×1.30 |

> 「그라비아 마진율 보통 20%로 설정」（グラビアマージン率は通常20%）

## 9. 銅版費（外径計算付き）

銅版は色数分必要。外径は製品ピッチから計算（最小420mm）。

### 9.1 銅版外径の決定

```
銅版円周 = π × 外径
1周あたり個数 = 円周 ÷ 製品ピッチ

機械最小外径 = 420mm（業界標準）
```

#### ピッチ86mmの例

| 外径 | 円周(π×外径) | 1周の個数(円周÷86) |
|---|---|---|
| **420mm（最小）** | 1,319mm | **約15個** |
| 540mm（標準）| 1,696mm | 約19個 |

**決定ロジック**:
1. 1周に並べる個数を決定（レイアウト: 1個/周 or 複数個/周）
2. 外径 = (個数 × ピッチ) ÷ π、ただし **最小420mm下限**
3. 1個/周だと外径 = ピッチ÷π（86÷π ≈ 27mm、小さすぎ→不可）
4. **実務は最小420mm採用が多い**（1周に複数個のピッチを並べる）
5. 既存版の使い回し（수정）なら既存外径を維持

### 9.2 銅版費計算

```
銅版幅 = 原反幅 + 100mm
도당（1色あたり）= 銅版幅 × 単価 × 外径
銅版費 = 色数 × 도당
```

| 銅版種類 | 単価 |
|---|---|
| 신규제작（新規）| 50 |
| 수정（修正）| 30 |

> **別途計上**: 銅版費は初回のみ大きく、リピート時は修正費or無料。**現場確認事項**。

## 11. 計算例：ロールフィルム

**条件**: PET12/AL7/PET12/LLDPE50、3色、製作6,000m（標準ロット）、ピッチ86mm、2液セミ、20%マージン
**原反幅**: 0.74m（740mm、ロール仕様 + 端代の例）

```
製作6,000m = 顧客提供5,500m + ロス500m
```

### 原材料費（製作6,000m、原反幅0.74m、最終層LLDPEは0.75m）

| 層 | 計算 | 金額(₫) |
|---|---|---|
| PET(12µ) | 0.012 × 0.74 × 6000 × 1.4 × 2800 | 208,858 |
| AL(7µ) | 0.007 × 0.74 × 6000 × 2.71 × 7800 | 656,969 |
| PET(12µ) | 0.012 × 0.74 × 6000 × 1.4 × 2800 | 208,858 |
| LLDPE(50µ) | 0.050 × **0.75** × 6000 × 0.92 × 2800 | 579,600 |
| **原材料費計** | | **1,654,284** |

### 工程費（製作6,000m）

| 項目 | 計算 | 金額(₫) |
|---|---|---|
| 印刷費 | 0.74 × 6000 × 3色 × 19 | 253,080 |
| ラミ費 | 0.75 × 6000 × 3回 × 65 | 877,500 |
| **工程費計** | | **1,130,580** |

### 集計

```
製造原価 = 1,654,284 + 1,130,580 = 2,784,864 ₫（製作6,000m分）
製造単価 = 2,784,864 ÷ 5,500m（顧客提供）= 506.34 ₫/m
納品単価 = 506.34 × 1.2 = 607.61 ₫/m
総納品額 = 607.61 × 5,500m = 3,341,837 ₫

銅版費（別途）= 3色 × (幅84cm × 50 × 外径42cm) = 529,200 ₫
  ※ 外径420mm(最小)採用、銅版幅=740+100=840mm
```

## 11B. 計算例：パウチ（袋加工あり）

**条件**: スタンドパウチ H=130 × W=100 × G=30、PET12/AL7/PET12/LLDPE50、3色、製作6,000m、20%マージン

### ステップ1: 原反幅の計算（§3）

```
1列フィルム幅(スタンド) = (H×2)+(G×2)+35 = (130×2)+(30×2)+35 = 355mm
列数 = floor(1,100 ÷ 355) = 3列
原反幅 = 355 × 3 = 1,065mm
最終層LLDPE = 1,065 + 10 = 1,075mm
```

### ステップ2: ピッチ（§3）

```
スタンド → ピッチ = パウチ幅(W) = 100mm = 0.1m
```

### ステップ3: 加工個数（§6.4）

```
加工個数 = ROUND(5,500 ÷ 0.1 × 0.93, -2) × 3列 = 51,100 × 3 = 153,300 個
```

### ステップ4: 原反値（ロールと同一計算、原反幅1,065mm・製作6,000m）

| 項目 | 計算 | 金額(₫) |
|---|---|---|
| 原材料費 | 各層（1,065mm、最終層1,075mm）× 6,000m | 2,377,434 |
| 印刷費 | 1.065 × 6,000 × 3色 × 19 | 364,230 |
| ラミ費 | 1.075 × 6,000 × 3回 × 65 | 1,257,750 |
| **原反値計** | | **3,999,414** |

### ステップ5: 加工費（§6.4）

```
加工単価 = 10.0cm(W100mm÷10) × 1.2(スタンド) = 12.0
加工費 = MAX(153,300 × 12.0, 250,000) = 1,839,600 ₫
```

### ステップ6: パウチ見積

```
パウチ原価 = 原反値 + 加工費 = 3,999,414 + 1,839,600 = 5,839,014 ₫
製造単価/個 = 5,839,014 ÷ 153,300個 = 38.09 ₫/個
納品単価/個 = 38.09 × 1.2 = 45.71 ₫/個
総納品額 = 45.71 × 153,300 = 7,006,817 ₫
```

> ※ バルブ加工・ジッパー追加・運賃は必要に応じて加算。銅版費は別途。

## 13. まとめ

### グラビア ロール見積の要点

1. **原反幅** = 計算で決定、**10mm単位**丸め（**最小500mm・最大1,100mm**、固定ではない）
   - ロール: 仕様幅 + 端代
   - パウチ: タイプ別公式(§3) × 列数
2. **ロス量** = **製作6,000mに500m込み**（顧客提供5,500m）
3. **原材料費** = Σ(厚み × 幅 × 6,000m × 比重 × 単価)、**最終層は幅+10mm**
4. **工程費** = 印刷費 + ラミネート費
   - ロール: 加工なし
   - パウチ: + 加工費（加工個数 × タイプ別単価）
5. **加工個数(パウチ)** = 5,500m ÷ **ピッチ** × 列数 × 0.93
   - ピッチ: 三方/スタンド=**幅**、合掌/ガゼット=**長さ**
6. **納品単価** = 製造原価 × 1.2（マージン20%）÷ 納品量（ロール:m、パウチ:個）
7. **銅版費** = 色数 × (幅 × 単価 × 外径)、外径はピッチ基準・最小420mm

### 計算式の全体像（ロール）

```
製作6,000m（標準ロット）= 顧客提供5,500m + ロス500m

製造原価（6,000m分）= Σ(厚み×幅×6,000×比重×単価)       ← 原材料費
                  + 原反幅 × 6,000 × 色数 × 印刷単価    ← 印刷費
                  + 原反幅 × 6,000 × ラミ回数 × ラミ単価 ← ラミ費

【ロール】
納品単価(m) = (製造原価 ÷ 5,500m[顧客提供]) × 1.2

【パウチ】
加工個数 = 5,500m ÷ ピッチ × 列数 × 0.93
加工費 = MAX(加工個数 × 加工単価, 最小加工費)
パウチ原価 = 製造原価(原反値) + 加工費
納品単価(個) = (パウチ原価 ÷ 加工個数) × 1.2

+ 銅版費（別途・初回）= 色数 × (原反幅+100mm) × 単価 × 外径
```

## 14. デジタル vs グラビア 分岐点分析

### 分岐点の定義

```
デジタル1個原価 = グラビア1個原価 となる数量 Q

デジタル: 小セットアップ(1万₫) + 高単価(475₫/m²/色) → 少量で有利
グラビア: 大固定費(銅版+6000m製作) + 安単価(19₫/m²/色) → 大量で有利

分岐点 Q = (グラビア固定費 - デジタル固定費) ÷ デジタル1個変動費
```

### パウチ別シミュレーション（共通: PET12/AL7/PET12/LLDPE50・3色）

| パウチタイプ | サイズ(W×H×G) | 1個面積 | グラビア列数 | **分岐点** | 傾向 |
|---|---|---|---|---|---|
| 三方（小）| 100×150 | 0.034m² | 3列 | **約54,000個** | — |
| スタンド（小）| 100×130×30 | 0.036m² | 3列 | **約61,000個** | — |
| スタンド（中）| 130×200×40 | 0.067m² | 2列 | **約30,000個** | ↓ |
| スタンド（大）| 160×250×50 | 0.102m² | 1列 | **約12,700個** | ↓↓ |
| 合掌T封 | 60×150 | 0.021m² | 7列 | **約113,000個** | ↑↑ |
| ガゼットM封 | 50×200×90 | 0.062m² | 3列 | **約31,000個** | — |

### 読み取り

1. **パウチが大きいほど分岐点が下がる**（スタンド: 6万→3万→1.3万個）
   - 面積大 → 印刷単価差(475 vs 19)が拡大 → グラビアが早く有利
2. **合掌T封は分岐点が突出して高い**（約11万個）
   - 面積小 + 列数7で効率良 + 銅版費が相対大 → グラビア有利になりにくい
3. **大パウチは1万個台で逆転** → グラビア向き

### 実務の目安

```
小パウチ(100x150程度) : 約5〜6万個でグラビア逆転
中パウチ(130x200程度) : 約3万個でグラビア逆転
大パウチ(160x250程度) : 約1.3万個でグラビア逆転
合掌T(小)             : 約11万個（デジタル優位の期間が長い）
```

### システム実装への示唆（動的レコメンド）

```
分岐点 = (グラビア固定費 - デジタル固定費) ÷ デジタル1個変動費
       = グラビア総原価(6000m製作+銅版) ÷ デジタル1個変動費（概算）

if 注文数量 < 分岐点 → デジタル推奨
if 注文数量 ≥ 分岐点 → グラビア推奨

※ 分岐点は パウチ面積・色数・材質・パウチタイプ から動的計算
※ 色数↑・面積↑ で分岐点は下がる（グラビア有利に）
```

> **注意**: 本分析はモデル計算（ロス率1.4・銅版外径420mm等の前提）。実務単価・条件の変動で分岐点は前後するため、目安として扱い、重要案件は個別精算。
