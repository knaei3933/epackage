# Epackage Lab ホームページ PRD

## 1. Purpose

- **Problem**: 既存の日本の包装資材市場は、小ロット生産における高コスト、長い納期、品質の不安定性という課題に直面している。特に、100〜10,000個規模の注文に対する適正価格での短納期対応が困難。

- **Solution**: 韓国製造業の強みを活かした高品質軟包材を、短期納期・適正価格で日本のものづくり企業に提供する専門プラットフォーム構築。自動見積りシミュレーションシステムにより、顧客が即座に価格を算出・比較できる機能を実装。

- **Business Value**:
  - 年商目標: 1億円（初年度）
  - 顧客獲得コスト削減: 40%（自動見積りによる）
  - 市場シェア: 3年以内に日本軟包材B2B市場の5%獲得
  - 平均顧客単価向上: 25%（アップセル/クロスセルによる）

## 2. Competitive Analysis

### Key Competitors
1. **日本国内包装資材メーカーA**:
   - 強み: 高品質、日本国内生産
   - 弱み: 小ロット対応不可、納期長い（30日以上）

2. **中国製包装資材輸入業者B**:
   - 強み: 低価格
   - 弱み: 品質不安定、コミュニケーションコスト

3. **オンライン包装資材C**:
   - 強み: ウェブ対応
   - 弱み: 見積り手動、カスタマイズ制限

### Our Differentiation Strategy
- **ブリッジ機能**: 日韓バイリンガルによるスムーズなコミュニケーション
- **テクノロジー優位性**: 世界最新鋭設備（HP Indigo 25000, NORDMECCANICA Simplex SL）
- **柔軟性**: 100個から製造可能、10日短期納期対応
- **透明性**: リアルタイム見積り、明確な価格構造

## 3. Risk Assessment

### Technical Risks
- **リスク**: 複雑な価格計算ロジックの実装難易度
  - **緩和策**: brixa-replicaで実証済みの計算エンジン採用、段階的テスト

### Market Risks
- **リスク**: 日本市場における韓国製品への信頼性
  - **緩和策**: 無料サンプル提供、品質保証6ヶ月、金井貿易実績活用

### Operational Risks
- **リスク**: 韓国とのサプライチェーン管理
  - **緩和策**: EPACKAGE社との長期契約、在庫管理システム構築

## 5. User Stories

### User Story 1: 見積り比較
**As a** 健康食品会社の仕入担当者
**I want to** 異なる数量（1,000個、3,000個、5,000個）の見積りを同時に比較したい
**So that** 最適な発注数量とコストを即座に判断できる

**Acceptance Criteria:**
- [ ] 最大5つの数量パターンを同時入力可能
- [ ] 各パターンの単価と総額を明確に表示
- [ ] コストパフォーマンス最適なパターンをハイライト
- [ ] PDF見積書としてダウンロード可能

### User Story 2: 製品選定
**As a** 化粧品ブランドの製品開発担当者
**I want to** 製品の特性（内容物、サイズ、素材）に基づいて最適な包装資材を選定したい
**So that** 品質とコストのバランスが取れた包装ソリューションを見つけられる

**Acceptance Criteria:**
- [ ] 6種類のパッケージタイプから選択可能
- [ ] 素材組み合わせの妥当性自動チェック
- [ ] 各素材の特徴と適用用途を明確に表示
- [ ] カスタムサイズ入力と価格算出

### User Story 3: サンプル請求
**As a** 新規事業の起業家
**I want to** 実際の製品サンプルを無料で請求したい
**So that** 品質と仕様を確認してから本発注を決定できる

**Acceptance Criteria:**
- [ ] オンラインフォームからサンプル請求可能
- [ ] 請求内容に基づいた適切なサンプル選定
- [ ] 発送追跡機能
- [ ] サンプル受領後のフォローアップ自動化
