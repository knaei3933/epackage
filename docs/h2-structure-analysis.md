## 問題点

### 1. IndustryShowcaseコンポーネント
```tsx
// 現状: pタグを使用（SEO的に弱い）
<motion.p className="text-2xl md:text-3xl font-bold text-center text-gray-800 mb-8">
  食品から化粧品、日用品、産業用途まで。多様な業界で信頼される包装ソリューション。
</motion.p>
```

**問題点:**
- 重要なセクション見出しであるにもかかわらず、pタグを使用
- 検索エンジンがこのセクションの重要性を正しく認識できない
- ページ構造の階層が不明確

### 2. ホームページ全体のH2タグの数
- H2タグが不足している可能性
- セクション間の階層関係が不明確

---

## 改善提案

### 提案1: IndustryShowcaseコンポーネントの見出し化

**優先度: 高**

```tsx
// 改善案: h2タグに変更
<motion.h2 className="text-2xl md:text-3xl font-bold text-center text-gray-800 mb-8">
  食品から化粧品、日用品、産業用途まで。<br />
  多様な業界で信頼される包装ソリューション。
</motion.h2>
```

**効果:**
- セクションの重要性を検索エンジンに正しく伝達
- ページ構造の階層化
- 「業界」「用途」といったキーワードの強調

### 提案2: H2タグの一貫性確保

**優先度: 中**

各主要セクションにH2タグを使用し、一貫した構造を確保：

```tsx
// 推奨構造
H1: メインタイトル
├── H2: 業界別実績紹介 (IndustryShowcase)
├── H2: 製品ソリューション (ProductShowcaseSection)
├── H2: 製造工程 (ManufacturingProcessShowcase)
├── H2: 実績紹介 (BeforeAfterSection)
└── H2: お問い合わせ・お見積もり (CTASection)
```

### 提案3: キーワード最適化

**優先度: 中**

H2タグに含めるキーワードの最適化：

| セクション | 現在のH2 | 改善案 | 追加キーワード |
|-----------|---------|--------|--------------|
| IndustryShowcase | (pタグ) | 「多様な業界で信頼される包装ソリューション」 | 業界、信頼、ソリューション |
| ProductShowcase | 「あなたの製品に最適なパッケージソリューション」 | (現状維持) | - |
| ManufacturingProcess | 「製造工程」 | 「最先端の製造工程と品質管理体制」 | 最先端、品質管理 |
| BeforeAfter | (要確認) | 「500社以上の導入実績と成功事例」 | 実績、成功事例 |
