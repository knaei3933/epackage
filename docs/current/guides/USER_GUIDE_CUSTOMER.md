# B2B 시스템 고객 사용자 가이드 (Customer User Guide)

## 시스템 개요

Epackage Lab B2B 시스템을 통해 견적 요청, 주문, 생산 추적까지 온라인으로 처리할 수 있습니다.

---

## 1. 로그인 (Login)

### 로그인 페이지
```
/member/signin 또는 /auth/signin
```

### 로그인 절차
1. 이메일 주소 입력
2. 비밀번호 입력
3. `로그인` 버튼 클릭

### 비밀번호 분실
- `비밀번호 찾기` 클릭
- 등록된 이메일 주소 입력
- 비밀번호 재설정 링크 수신

---

## 2. 대시보드 (Dashboard)

### 접속 경로
```
/member/dashboard
```

### 대시보드 구성

#### 개요 (Overview) 탭
- **전체 주문**: 총 주문 수
- **진행 중 주문**: 현재 처리 중인 주문
- **완료 주문**: 완료된 주문 수
- **견적 요청**: 견적 요청 현황
- **샘플 요청**: 샘플 요청 현황

#### 최근 활동
- 최근 주문 목록
- 최근 견적 목록
- 진척률 표시

---

## 3. 견적 요청 (Quotation Request)

### 새 견적 요청
```
/member/quotations/new
```

### 견적 요청 절차

#### 1. 회사 정보 선택
- 등록된 회사 선택
- (회사가 없는 경우: 먼저 회사 등록)

#### 2. 제품 추가
- `제품 추가` 버튼 클릭
- 다음 항목 입력:
  - **제품명** (필수)
  - **제품 코드** (선택)
  - **카테고리** (선택)
  - **수량** (필수)
  - **단가** (선택, 0 입력 가능)
  - **규격** (선택)
  - **메모** (선택)

#### 3. 여러 제품 요청
- `제품 추가` 버튼으로 여러 제품 추가
- 최대 제한 없음

#### 4. 추가 메모
- 전체 견적에 대한 메모
- 특별 요청 사항 등

#### 5. 저장 및 전송
- **임시 저장**: 나중에 계속 작업
- **견적 요청**: 관리자에게 전송

### 견적 상태
- **임시 (DRAFT)**: 임시 저장됨
- **대기 (PENDING)**: 관리자 검토 중
- **송부 (SENT)**: 견적서 발송됨
- **승인 (APPROVED)**: 승인됨 (주문 가능)
- **변환 (CONVERTED)**: 주문으로 전환됨
- **거부 (REJECTED)**: 거부됨

---

## 4. 견적 확인 및 주문 생성

### 견적 확인
```
/member/quotations
```

### 견적 상세 보기
1. 목록에서 견적 선택
2. 견적 내용 확인
   - 제품 목록
   - 단가
   - 수량
   - 합계 금액
3. PDF 다운로드 가능

### 주문 생성
1. 승인된 견적 선택
2. `주문 생성` 버튼 클릭
3. 주문 번호 자동 생성
4. 주문 상태: `QUOTATION`

---

## 5. 주문 추적 (Order Tracking)

### 주문 목록
```
/member/orders
```

### 주문 상세 보기
1. 목록에서 주문 클릭
2. 전체 타임라인 표시

### 10단계 주문 프로세스

| 단계 | 설명 |
|------|------|
| 1. 견적 | 견적 요청 및 승인 |
| 2. 데이터 입고 | 디자인 데이터 수령 |
| 3. 작업표준서 | 제조 사양서 작성 |
| 4. 계약서 송부 | 계약서 전송 |
| 5. 계약서 서명 | 양측 전자서명 |
| 6. 생산 | 제조 공정 진행 |
| 7. 입고 | 제품 입고 및 품질검사 |
| 8. 출하 | 배송 시작 |
| 9. 배송 완료 | 고객 수령 완료 |

### 진척률 확인
- 상단에 전체 진척률 표시
- 백분율로 표시
- 완료된 단계: 녹색 체크
- 현재 단계: 파랑색 표시
- 미완료 단계: 회색

### 생산 진척률 (생산 중인 경우)
- 현재 공정 표시
- 하위 공정 진척률
- 작업 사진 (있는 경우)
- 최신 작업 메모

### 배송 추적 (출하 후)
- 배송업체
- 운송장 번호
- 추적 링크

---

## 6. 전자서명 (Electronic Signature)

### 계약서 서명 대기
- 대시보드에서 `계약서 서명 대기` 표시
- 알림 표시

### 서명 절차

#### 1. 계약서 열기
- `서명하기` 버튼 클릭
- 또는 주문 상세에서 접속

#### 2. 계약서 내용 확인
- PDF 미리보기로 계약서 확인
- 스크롤하여 전체 내용 확인

#### 3. 이용약관 동의
- 약관 내용 확인
- `동의합니다` 체크박스 클릭

#### 4. 전자서명
- 서명란에 마우스 또는 손가락으로 서명
- 서명 취소: `지우기` 버튼

#### 5. 서명 완료
- `서명 완료` 버튼 클릭
- 서명일시, IP주소 자동 기록

### 서명 후
- 관리자에게 알림 발송
- 관리자 서명 대기
- 양측 서명 완료 후 생산 시작

## 7. 문서 다운로드 (Document Download)

### 다운로드 가능 문서
- **견적서**: 견적 PDF
- **작업표준서**: 제조 사양서 PDF
- **계약서**: 전자계약서 PDF

### 다운로드 방법

#### 주문 상세 페이지에서
1. 주문 상세 페이지 접속
2. 각 단계에서 다운로드 버튼 표시
3. `다운로드` 버튼 클릭

#### 예시
- 작업표준서 단계: `작업표준서 다운로드`
- 계약서 단계: `계약서 다운로드`

## 8. 샘플 요청 (Sample Request)

### 새 샘플 요청
```
/member/samples/new
```

### 샘플 요청 절차

#### 1. 회사 정보 선택
- 등록된 회사 선택

#### 2. 샘플 제품 추가
- 최대 5개까지 가능
- 각 제품별 정보 입력:
  - 제품명
  - SKU/코드
  - 수량
  - 규격

#### 3. 프로젝트 정보
- 프로젝트 명
- 예상 정량
- 예상 납기
- 특이사항

#### 4. 배송 정보
- 수신자명
- 연락처
- 배송 주소

#### 5. 요청 전송
- `샘플 요청` 버튼 클릭

### 샘플 요청 상태
- **대기 (PENDING)**: 검토 대기
- **처리 중 (PROCESSING)**: 샘플 준비 중
- **송부 (SHIPPED)**: 발송됨
- **완료 (DELIVERED)**: 수신 완료

## 주의 사항

### 견적 요청
- 제품명은 정확하게 입력
- 수량은 정확하게 확인
- 특별 사항은 메모에 기재

### 계약서
- 내용을 충분히 확인 후 서명
- 서명은 법적 효력이 있습니다
- 서명 후 변경이 불가능합니다

### 주문 추적
- 진척률은 실시간 업데이트
- 생산 중에는 작업 사진 확인 가능

### 문의 사항
- 견적 관련: sales@epackage-lab.com
- 주문 관련: order@epackage-lab.com
- 기술 지원: support@epackage-lab.com

## 자주 묻는 질문 (FAQ)

### Q: 견적 유효기간은 얼마인가요?
A: 기본 30일입니다. 유효기간 내에 승인되지 않으면 자동 취소될 수 있습니다.

### Q: 주문 후 취소할 수 있나요?
A: 생산 시작 전까지만 가능합니다. 생산 시작 후에는 취소할 수 없습니다.

### Q: 배송 기간은 얼마나 걸리나요?
A: 제품에 따라 다르지만, 일반적으로 7-14일 정도 소요됩니다. 자세한 납기는 견적 시 확인해 주세요.

### Q: 전자서명이 법적으로 효력이 있나요?
A: 네, 일본 전자서명법을 준수하여 법적 효력이 있습니다.

### Q: 작업표준서가 무엇인가요?
A: 제조 시 필요한 모든 사양이 정리된 문서로, 정확한 제품을 생산하기 위한 중요한 문서입니다.

### Q: 샘플은 무료인가요?
A: 샘플 비용은 별도 문의해 주세요. 대량 생산 시 샘플 비용을 차감해 드릴 수 있습니다.

### Q: 주문 수정은 가능한가요?
A: 생산 시작 전까지만 가능합니다. 수정이 필요한 경우 즉시 고객센터에 연락해 주세요.
