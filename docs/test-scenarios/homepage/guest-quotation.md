# 게스트 견적 생성 시나리오

**작성일**: 2026-01-21
**목적**: 게스트 사용자가 홈페이지에서 견적을 생성하는 전체 과정 테스트

---

## 1.1 홈페이지에서 견적 시뮬레이터 시작

**목표**: 홈페이지에서 견적 시뮬레이터를 통해 견적 생성

**전제 조건**:
- 게스트 사용자 (비로그인 상태)
- 브라우저: http://localhost:3002

**테스트 단계**:

```bash
# 1. 홈페이지 접속
[Browser_navigate] http://localhost:3002

# 2. 페이지 로딩 확인
[Browser_wait_for] time: 2
[Browser_snapshot]  # 히어로 섹션, 제품 쇼케이스 확인

# 3. 스크롤하여 견적 시뮬레이터 섹션 찾기
[Browser_evaluate] window.scrollTo(0, 500)

# 4. "견적 시뮬레이터" 버튼/섹션 클릭
[Browser_click] element="お見積り"
```

**예상 결과**:
- 견적 시뮬레이터 위저드가 열림
- 첫 번째 스텝: 제품 유형 선택 화면 표시

**성공 기준**:
- ✅ 견적 위저드가 정상적으로 표시됨
- ✅ 제품 선택 옵션 4개가 모두 보임

---

## 1.2 제품 사양 입력

**목표**: 스탠드 파우치, 대형 사이즈, 스탠다드PET, 스폿트 인쇄로 사양 입력

**테스트 단계**:

```bash
# 1. 제품 유형 선택 (스탠딩파우치)
[Browser_click] element="スタンディングパウチ"

# 2. 생산 수량 선택 (중로트)
[Browser_click] element="中ロット"

# 3. 사이즈 선택 (대형)
[Browser_click] element="大サイズ"

# 4. 다음 단계
[Browser_click] element="次へ進む"
```

**예상 결과**:
- 제품 유형, 수량, 사이즈가 선택됨
- 다음 스텝(상세 설정)으로 이동

**성공 기준**:
- ✅ 모든 사양이 정상 선택됨
- ✅ 다음 버튼이 활성화됨

---

## 1.3 상세 설정 입력

**목표**: 스탠다드PET, 스폿트 인쇄, 표준 납기로 설정

**테스트 단계**:

```bash
# 1. 재질 선택 (스탠다드PET - 인기)
[Browser_click] element="スタンダードPET"

# 2. 인쇄 유형 선택 (스폿트 컬러)
[Browser_click] element="スポットカラー"

# 3. 납기 선택 (표준)
[Browser_click] element="標準"

# 4. 다음 단계 (후가공 스킵)
[Browser_click] element="次へ進む"
```

**예상 결과**:
- 재질, 인쇄, 납기가 선택됨
- 다음 스텝(연락처 정보)으로 이동

**성공 기준**:
- ✅ 모든 상세 설정이 정상 선택됨
- ✅ 다음 버튼이 활성화됨

---

## 1.4 연락처 정보 입력

**목표**: 연락처 정보 입력 후 견적 계산

**테스트 단계**:

```bash
# 1. 이름 입력
[Browser_type] element="お名前" text="テストユーザー"

# 2. 이메일 입력
[Browser_type] element="メールアドレス" text="test@example.com"

# 3. 전화번호 입력
[Browser_type] element="電話番号" text="08012345678"

# 4. 회사명 입력
[Browser_type] element="会社名" text="テスト株式会社"

# 5. 개인정보 동의
[Browser_click] element="個人情報の取り扱いに同意します"

# 6. 견적 계산 제출
[Browser_click] element="見積もり計算"
```

**예상 결과**:
- 견적 결과가 표시됨
- 총 비용, 단가, 절감액 등 ROI 분석 결과 표시

**성공 기준**:
- ✅ 견적 금액이 정상적으로 계산됨
- ✅ 결과 페이지가 표시됨
- ✅ ROI 분석 결과가 보임

---

## 전체 성공 기준

- ✅ 게스트 사용자가 견적을 생성할 수 있음
- ✅ 모든 사양이 정상적으로 선택됨
- ✅ 가격이 정확하게 계산됨
- ✅ 결과 페이지가 정상 표시됨

---

## 오류 처리

**문제**: 제품 유형이 선택되지 않을 때
**해결**:
1. 페이지 새로고침
2. 스크롤하여 견적 시뮬레이터 섹션 확인

**문제**: 연락처 정보 입력 후 제출이 안 될 때
**해결**:
1. 필수 필드 확인 (이름, 이메일, 전화번호)
2. 개인정보 동의 체크박스 확인
