# 샘플 패키지 제조 사양서 (4종 카테고리)

## 개요

- **목적**: 일본 법적 카테고리별 실제 제품 샘플 패키지 제작
- **카테고리**: 식품, 건강식품, 화장품, 의약외품
- **총 수량**: 각 10매 / 총 40매 (4종)
- **납품 기한**: 협의
- **인쇄**: 오프셋 인쇄 / 4~6색 / 실제 제품 디자인
