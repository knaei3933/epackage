# Epackage Lab ユーザーガイド

---

## 1. サービス概要

### 1.1 Epackage Labとは？

Epackage Labは、**B2B向け包装材製造のデジタルプラットフォーム**です。化粧品、健康食品、サプリメントメーカー向けに、小ロットから高品質な包装材をオンラインで注文できます。

#### 主な機能

- **AI見積シミュレーター** - 2秒で即座価格提示
- **製品カタログ** - 6種類の包装材製品
- **12ステージデジタルワークフロー** - 見積から納品までオンライン完結
- **リアルタイム進捗管理** - メール・SMS自動通知

### 1.2 利用可能環境

- **PC**: Chrome, Firefox, Safari, Edge（最新版）
- **タブレット**: iPad, Androidタブレット
- **スマートフォン**: iOS 15+, Android 10+

### 1.3 アカウント種類

| アカウント種類 | 説明 | 主な機能 |
|----------------|------|----------|
| **会員** | 一般顧客 | 見積依頼、注文管理、サンプル申請 |
| **管理者** | スタッフ | 顧客管理、注文管理、システム設定 |
| **デザイナー** | 韓国人デザイナー | デザイン修正、ファイルアップロード |

## 2. 会員登録・ログイン

### 2.1 会員登録

#### 登録手順

1. トップページの「新規登録」をクリック
2. 必要情報を入力
   - メールアドレス
   - パスワード（8文字以上）
   - 会社名
   - お名前
3. メールアドレス認証
4. 登録完了

#### 必要情報

| 項目 | 説明 |
|------|------|
| **メールアドレス** | 企業用メールアドレス推奨 |
| **パスワード** | 8文字以上、英数字記号を含める |
| **会社名** | 登録上の会社名 |
| **お名前** | 担当者のお名前 |

### 2.2 ログイン

#### ログイン方法

1. トップページの「ログイン」をクリック
2. メールアドレスとパスワードを入力
3. 「ログイン」ボタンをクリック

#### パスワードを忘れた場合

1. ログイン画面の「パスワードを忘れた場合」をクリック
2. 登録メールアドレスを入力
3. パスワードリセットメールを受信
4. リンクからパスワードを再設定

## 3. サービス利用ガイド

### 3.1 見積シミュレーター

#### 見積依頼手順

1. **製品選択**
   - 製品カタログから目的の製品を選択
   - 平袋、スタンドパウチ、合掌袋など

2. **仕様入力**
   - サイズ、材質、印刷色数、数量などを入力
   - オプション機能を選択

3. **価格確認**
   - AIが2秒で見積価格を表示
   - 複数条件のシミュレーションが可能

4. **見積依頼**
   - 会員登録で見積依頼を送信
   - メールで見積書を受信

### 3.2 製品カタログ

#### 製品検索

1. 「パウチ製品」メニューをクリック
2. カテゴリーでフィルタリング
3. 製品詳細を確認

#### 製品詳細情報

- 仕様
- 材質
- 寸法
- 特徴
- 用途
- 納期
- 最小注文数
- 価格

### 3.3 サンプル申請

#### サンプル申請手順

1. 製品詳細画面で「サンプル依頼」をクリック
2. 必要情報を入力
3. 送信完了

#### サンプル受領について

- 無料サンプル対応
- 送料はお客様負担
- 受領まで3〜5営業日

### 3.4 注文管理

#### 注文手順

1. 見積を承認
2. デザインファイルをアップロード
3. 校正を確認
4. 承認後に製造開始
5. 出荷・納品

#### デザインアップロード

| 対応フォーマット | 説明 |
|-----------------|------|
| **AI** | Adobe Illustrator（推奨） |
| **PSD** | Adobe Photoshop |
| **PDF** | PDFデータ |
| **JPG** | JPEG画像 |
| **PNG** | PNG画像 |

#### 修正リクエスト

- ダッシュボードから修正依頼を送信
- デザイナーとのコメント機能でやり取り
- 修正履歴はすべて記録

## 8. FAQ

### 8.1 会員登録関連

**Q: 会員登録は無料ですか？**
A: はい、会員登録は無料です。

**Q: 個人でも登録できますか？**
A: 申し訳ございません。法人のお客様のみ対象となります。

**Q: メールアドレスを変更できますか？**
A: ダッシュボードのプロフィール設定から変更できます。

### 8.2 見積・注文関連

**Q: 見積は有料ですか？**
A: いいえ、見積は無料です。

**Q: 最小注文数は？**
A: 500枚から承ります。

**Q: 納期はどのくらい？**
A: 平均21日です。お急ぎの場合は14日から可能です。

**Q: 注文をキャンセルできますか？**
A: 製造開始前であればキャンセル可能です。

### 8.3 決済・配送関連

**Q: 代引きは使えますか？**
A: 申し訳ございません。銀行振込のみ対応しています。

**Q: 離島にも配送しますか？**
A: はい、一部離島を除き配送可能です。

**Q: 海外配送は？**
A: 現在、日本国内のみ対応しています。

### 8.4 技術関連

**Q: 推奨ブラウザは？**
A: Chrome, Firefox, Safari, Edgeの最新版を推奨します。

**Q: デザインファイルのサイズ制限は？**
A: 1ファイルあたり50MBまでです。
