# Epackage Lab User Guide

---

## 1. Service Overview

### 1.1 What is Epackage Lab?

Epackage Lab is a **digital platform for B2B packaging material manufacturing**. We enable cosmetics, health food, and supplement manufacturers to order high-quality packaging materials in small lots online.

#### Key Features

- **AI Quote Simulator** - Instant pricing in 2 seconds
- **Product Catalog** - 6 types of packaging products
- **12-Stage Digital Workflow** - Complete online process from quote to delivery
- **Real-time Progress Management** - Automatic email and SMS notifications

### 1.2 Supported Environments

- **PC**: Chrome, Firefox, Safari, Edge (latest versions)
- **Tablet**: iPad, Android tablets
- **Smartphone**: iOS 15+, Android 10+

### 1.3 Account Types

| Account Type | Description | Main Features |
|--------------|-------------|---------------|
| **Member** | General customers | Quote requests, order management, sample requests |
| **Admin** | Staff | Customer management, order management, system settings |
| **Designer** | Korean designers | Design revisions, file uploads |

## 2. Registration and Login

### 2.1 Member Registration

#### Registration Steps

1. Click "New Registration" on the top page
2. Enter required information
   - Email address
   - Password (8 characters or more)
   - Company name
   - Your name
3. Verify email address
4. Registration complete

#### Required Information

| Item | Description |
|------|-------------|
| **Email Address** | Business email address recommended |
| **Password** | 8+ characters, including letters, numbers, and symbols |
| **Company Name** | Registered company name |
| **Your Name** | Name of the person in charge |

### 2.2 Login

#### How to Login

1. Click "Login" on the top page
2. Enter email address and password
3. Click "Login" button

#### If You Forgot Your Password

1. Click "Forgot Password" on the login screen
2. Enter registered email address
3. Receive password reset email
4. Reset password via the link

## 3. Service Usage Guide

### 3.1 Quote Simulator

#### Quote Request Steps

1. **Product Selection**
   - Select desired product from the product catalog
   - Flat pouches, stand-up pouches, center-seal bags, etc.

2. **Specification Input**
   - Enter size, material, number of print colors, quantity, etc.
   - Select optional features

3. **Price Confirmation**
   - AI displays estimated price in 2 seconds
   - Simulate multiple conditions

4. **Quote Request**
   - Send quote request with member registration
   - Receive quote via email

### 3.2 Product Catalog

#### Product Search

1. Click "Pouch Products" menu
2. Filter by category
3. View product details

#### Product Detail Information

- Specifications
- Material
- Dimensions
- Features
- Applications
- Delivery time
- Minimum order quantity
- Price

### 3.3 Sample Request

#### Sample Request Steps

1. Click "Sample Request" on the product detail screen
2. Enter required information
3. Submit

#### About Sample Receipt

- Free samples available
- Shipping fee is customer's responsibility
- 3-5 business days for delivery

### 3.4 Order Management

#### Order Steps

1. Approve quote
2. Upload design file
3. Review proof
4. Manufacturing begins after approval
5. Shipping and delivery

#### Design Upload

| **AI** | Adobe Illustrator (recommended) |
| **PSD** | Adobe Photoshop |
| **PDF** | PDF data |
| **JPG** | JPEG image |

- Send revision requests from dashboard

## 8. FAQ

### 8.1 Registration Related

**Q: Is membership free?**
A: Yes, membership registration is free.

**Q: Can individuals register?**
A: We apologize, but only corporate customers are eligible.

**Q: Can I change my email address?**
A: You can change it from the profile settings in the dashboard.

### 8.2 Quote and Order Related

**Q: Is there a fee for quotes?**
A: No, quotes are free of charge.

**Q: What is the minimum order quantity?**
A: We accept orders from 500 pieces.

**Q: How long is the delivery time?**
A: Average is 21 days. For urgent orders, 14 days is possible.

**Q: Can I cancel an order?**
A: Cancellation is possible before manufacturing begins.

### 8.3 Payment and Shipping Related

**Q: Is cash on delivery available?**
A: We apologize, but only bank transfer is supported.

**Q: Do you ship to remote islands?**
A: Yes, shipping is available except for some remote islands.

**Q: Do you ship overseas?**
A: Currently, we only serve within Japan.

### 8.4 Technical Related

**Q: Which browsers are recommended?**
A: We recommend the latest versions of Chrome, Firefox, Safari, and Edge.

**Q: What is the file size limit for design files?**
A: Up to 50MB per file.
