# Epackage Lab 사용자 가이드

---

## 1. 서비스 개요

### 1.1 Epackage Lab이란?

Epackage Lab은 **B2B 포장재 제조 디지털 플랫폼**입니다. 화장품, 건강기능식품, 보조제 제조사를 위해 소량부터 고품질 포장재를 온라인으로 주문할 수 있습니다.

#### 주요 기능

- **AI 견적 시뮬레이터** - 2초 만에 즉시 가격 제시
- **제품 카탈로그** - 6종 포장재 제품
- **12단계 디지털 워크플로우** - 견적부터 납품까지 온라인 완결
- **실시간 진행 상황 관리** - 이메일·SMS 자동 알림

### 1.2 이용 가능 환경

- **PC**: Chrome, Firefox, Safari, Edge(최신 버전)
- **태블릿**: iPad, Android 태블릿
- **스마트폰**: iOS 15+, Android 10+

### 1.3 계정 유형

| 계정 유형 | 설명 | 주요 기능 |
|-----------|------|----------|
| **회원** | 일반 고객 | 견적 요청, 주문 관리, 샘플 신청 |
| **관리자** | 스태프 | 고객 관리, 주문 관리, 시스템 설정 |
| **디자이너** | 한국인 디자이너 | 디자인 수정, 파일 업로드 |

## 2. 회원 가입·로그인

### 2.1 회원 가입

#### 가입 절차

1. 홈페이지의 '신규 가입' 클릭
2. 필수 정보 입력
   - 이메일 주소
   - 비밀번호(8자 이상)
   - 회사명
   - 성명
3. 이메일 주소 인증
4. 가입 완료

#### 필수 정보

| 항목 | 설명 |
|------|------|
| **이메일 주소** | 회사용 이메일 주소 권장 |
| **비밀번호** | 8자 이상, 영문숫자기호 포함 |
| **회사명** | 등록상 회사명 |
| **성명** | 담당자 성명 |

### 2.2 로그인

#### 로그인 방법

1. 홈페이지의 '로그인' 클릭
2. 이메일 주소와 비밀번호 입력
3. '로그인' 버튼 클릭

#### 비밀번호를 잊으신 경우

1. 로그인 화면의 '비밀번호 찾기' 클릭
2. 가입 이메일 주소 입력
3. 비밀번호 재설정 이메일 수신
4. 링크에서 비밀번호 재설정

## 3. 서비스 이용 가이드

### 3.1 견적 시뮬레이터

#### 견적 요청 절차

1. **제품 선택**
   - 제품 카탈로그에서 원하는 제품 선택
   - 평팩, 스탠드 파우치, 합장팩 등

2. **규격 입력**
   - 사이즈, 재질, 인쇄 색수, 수량 등 입력
   - 옵션 기능 선택

3. **가격 확인**
   - AI가 2초 만에 견적 가격 표시
   - 여러 조건 시뮬레이션 가능

4. **견적 요청**
   - 회원 가입으로 견적 요청 전송
   - 이메일로 견적서 수신

### 3.2 제품 카탈로그

#### 제품 검색

1. '파우치 제품' 메뉴 클릭
2. 카테고리로 필터링
3. 제품 상세 확인

#### 제품 상세 정보

- 규격
- 재질
- 치수
- 특징
- 용도
- 납기
- 최소 주문 수량
- 가격

### 3.3 샘플 신청

#### 샘플 신청 절차

1. 제품 상세 화면에서 '샘플 요청' 클릭
2. 필수 정보 입력
3. 전송 완료

#### 샘플 수령에 대해

- 무료 샘플 지원
- 배송비는 고객 부담
- 수령까지 3~5 영업일

### 3.4 주문 관리

#### 주문 절차

1. 견적 승인
2. 디자인 파일 업로드
3. 교정 확인
4. 승인 후 제조 시작
5. 출하·납품

#### 디자인 업로드

| 지원 형식 | 설명 |
|-----------|------|
| **AI** | Adobe Illustrator(권장) |
| **PSD** | Adobe Photoshop |
| **PDF** | PDF 데이터 |
| **JPG** | JPEG 이미지 |
| **PNG** | PNG 이미지 |

#### 수정 요청

- 대시보드에서 수정 요청 전송
- 디자이너와 댓글 기능으로 소통
- 수정 이력은 모두 기록

## 4. 회원 대시보드

### 4.1 대시보드 개요

로그인 후 홈 화면에서 다음 정보를 확인할 수 있습니다.

- 최신 주문 현황
- 미처리된 견적 요청
- 알림 목록
- 빠른 작업

### 4.2 주문 목록

#### 주문 상태

| 상태 | 설명 |
|----------|------|
| **견적 중** | 견적 작성 중 |
| **견적 완료** | 견적 완료, 고객 확인 대기 |
| **승인 완료** | 견적 승인, 제조 대기 |
| **제조 중** | 제조 작업 중 |
| **검품 중** | 품질 검사 중 |
| **출하 완료** | 출하 완료 |
| **완료** | 납품 완료 |
| **취소** | 취소 완료 |

### 4.3 주문 상세

주문 상세 화면에서 확인 가능한 정보:

- 주문 내용
- 규격 정보
- 디자인 파일
- 진행 상황
- 댓글 내역
- 수정 내역

### 4.4 견적 관리

#### 견적 목록

- 모든 견적 요청 확인
- 상태로 필터링
- 견적 상세 표시

#### 견적 상세

- 가격 명세
- 규격 상세
- 유효 기간
- 견적서 PDF 다운로드

### 4.5 계정 설정

#### 프로필 편집

- 회사명
- 담당자명
- 이메일 주소
- 전화번호

#### 비밀번호 변경

1. 현재 비밀번호 입력
2. 새 비밀번호 입력
3. 확인용 비밀번호 입력
4. '변경' 클릭

#### 알림 설정

- 이메일 알림 ON/OFF
- SMS 알림 ON/OFF
- 알림 내용 사용자 정의

## 6. 디자이너 포털 가이드

> **대상:** 한국인 디자이너

### 6.1 디자이너 포털이란?

한국인 디자이너를 위한 전용 포털로, 할당된 주문의 디자인 수정·파일 업로드를 수행합니다.

### 6.2 로그인

1. 전용 로그인 페이지 접속
2. 디자이너 ID와 비밀번호 입력
3. 로그인

### 6.3 작업 관리

#### 할당된 주문 목록

- 자신에게 할당된 주문 표시
- 상태로 필터링
- 마감일 확인

#### 주문 상세

- 규격 정보 확인
- 현재 디자인 파일
- 수정 요청 내용
- 고객과의 댓글 내역

### 6.4 파일 업로드

#### 지원 형식

| 형식 | 설명 |
|-------------|------|
| **AI** | Adobe Illustrator(권장) |
| **PSD** | Adobe Photoshop |
| **PDF** | PDF 데이터 |
| **JPG** | JPEG 이미지 |
| **PNG** | PNG 이미지 |

#### 업로드 절차

1. 주문 상세 화면 열기
2. '파일 업로드' 클릭
3. 파일 선택
4. 버전 코멘트 입력
5. 업로드

### 6.5 고객 소통

#### 댓글 기능

- 고객·관리자와 댓글로 소통
- 수정 요청 내용 상세 확인
- 이미지 첨부 가능

#### 수정 내역

- 모든 수정 내역 기록
- 버전 관리
- 변경점 비교

## 8. FAQ

### 8.1 회원 가입 관련

**Q: 회원 가입은 무료인가요?**
A: 네, 회원 가입은 무료입니다.

**Q: 개인도 가입할 수 있나요?**
A: 죄송합니다. 법인 고객만 이용 가능합니다.

**Q: 이메일 주소를 변경할 수 있나요?**
A: 대시보드의 프로필 설정에서 변경할 수 있습니다.

### 8.2 견적·주문 관련

**Q: 견적은 유료인가요?**
A: 아니요, 견적은 무료입니다.

**Q: 최소 주문 수량은?**
A: 500장부터 접수합니다.

**Q: 납기는 얼마나 걸리나요?**
A: 평균 21일입니다. 급한 경우 14일부터 가능합니다.

**Q: 주문을 취소할 수 있나요?**
A: 제조 시작 전이면 취소 가능합니다.

### 8.3 결제·배송 관련

**Q: 대금 수령(代引)을 사용할 수 있나요?**
A: 죄송합니다. 은행 송금만 지원합니다.

**Q: 섬 지역에도 배송하나요?**
A: 네, 일부 섬 지역을 제외하고 배송 가능합니다.

**Q: 해외 배송은?**
A: 현재 일본 국내만 지원합니다.

### 8.4 기술 관련

**Q: 권장 브라우저는?**
A: Chrome, Firefox, Safari, Edge 최신 버전을 권장합니다.

**Q: 디자인 파일 크기 제한은?**
A: 파일당 50MB까지입니다.

### 9.2 자주 발생하는 문제

#### 로그인할 수 없음

- 이메일 주소와 비밀번호 확인
- 브라우저 캐시 삭제
- 비밀번호 재설정 시도

#### 주문을 찾을 수 없음

- 대시보드의 주문 목록 확인
- 상태 필터 초기화
- 검색 기능 사용

#### 파일을 업로드할 수 없음

- 파일 크기가 50MB 이하인지 확인
- 지원 형식인지 확인
- 네트워크 연결 확인

2. 스크린샷 첨부
3. 현상 발생 절차 기술
