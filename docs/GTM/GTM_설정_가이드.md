## 📋 GTM 관리 화면에서 해야 할 작업

### 1단계: 컨테이너 공개 상태 확인 (가장 중요!)

#### 1-1. 워크스페이스 상태 확인
```
GTM 관리 화면 접속
→ 상단에 표시되는 워크스페이스 상태 확인
→ "공개됨" 표시가 있는지 확인
```

#### 1-2. 공개되지 않은 경우 공개하기
```
우상단의 [공개] 버튼 클릭
→ [공개] 버튼 다시 클릭
→ 버전 이름 입력 (예: "v1.0 - 초기 배포")
→ 설명 입력 (예: "GA4, Google Ads 태그 활성화")
→ [공개] 클릭
```

**✅ 확인 방법:**
- 공개 후 URL 변경: `https://www.package-lab.com` 접속
- GTM 미리보기 모드에서 태그 발생(firing) 확인

---

### 2단계: 트리거 설정 확인

#### 2-1. "All Pages" 트리거 확인
```
왼쪽 메뉴 → [트리거]
→ "All Pages" 또는 "페이지 뷰" 트리거 존재 확인
```

**필요한 설정:**
| 항목 | 설정값 |
|------|--------|
| 트리거 유형 | **페이지 뷰 - 페이지 보기** |
| 발생 조건 | **모든 페이지 뷰** 또는 **Page URL** 정규 표현식 `.*` |

#### 2-2. 트리거가 없는 경우 생성
```
[새로 만들기] 클릭
→ 트리거 유형: [페이지 뷰] 선택
→ 트리거 이름: "All Pages"
→ 발생 조건: [일부 페이지 보기] → [Page URL] → [정규 표현식에 일치] → `.*` 입력
→ [저장] 클릭
```

### 4단계: 미리보기 모드로 동작 테스트

#### 4-1. 미리보기 시작
```
우상단의 [미리보기] 버튼 클릭
→ 웹사이트 URL 입력: `https://www.package-lab.com`
→ [연결] 클릭
```

#### 4-2. 태그 발생 확인
```
왼쪽 패널에서 다음 항목 확인:
→ Container Status: Connected (연결됨)
→ Tags Fired: GA4, Google Ads 태그가 초록색 체크 표시
→ gtm.js 이벤트: 데이터 레이어에 표시됨
```

**✅ 성공 기준:**
- Google Tag (G-VBCB77P21T) → **초록색 체크**
- Google Ads (AW-17981675917) → **초록색 체크**
- 에러 메시지 없음

#### 4-3. 에러가 있는 경우
| 에러 메시지 | 원인 | 해결 방법 |
|-----------|------|----------|
| "태그가 발생하지 않음" | 트리거 미설정 또는 미일치 | "All Pages" 트리거 확인 |
| "태그가 일시 중지됨" | 태그가 비활성화됨 | 태그 설정에서 [활성] 체크 |
| "변수를 찾을 수 없음" | 데이터 레이어 미설정 | dataLayer 초기화 코드 확인 |

### 5단계: Google Analytics에서 데이터 수신 확인

#### 5-1. 실시간 보고서 확인
```
Google Analytics 접속
→ [보고서] → [실시간]
→ 다른 브라우저로 https://www.package-lab.com 접속
→ "지난 30분간의 사용자"에 본인 표시 확인
```

**✅ 성공 기준:**
- 실시간 사용자 수: 1명 이상 표시
- 이벤트: page_view 이벤트 기록됨
- 위치: 일본/한국 등 사용자 위치 표시

#### 5-2. 디버그뷰로 데이터 확인
```
GA4 → [구성] → [DebugView]
→ 왼쪽 하단 [장치] 탭에서 이벤트 스트림 확인
→ realtime 이벤트: page_view, first_visit, session_start 등 표시
```

## 🔧 문제 해결 가이드

### 문제 1: "태그에서 데이터 전송이 중지됨" 에러

**원인 1: 컨테이너가 공개되지 않음**
```
해결: [공개] 버튼 → [공개] 클릭하여 배포
```

**원인 2: 태그가 비활성화됨**
```
해결: 태그 설정 → [활성] 체크 → [저장] → [공개]
```

**원인 3: 트리거가 일치하지 않음**
```
해결: "All Pages" 트리거 생성 또는 수정
→ 발생 조건을 [모든 페이지 뷰]로 변경
```

---

### 문제 2: GA4에 데이터가 전송되지 않음

**확인 checklist:**

- [ ] 측정 ID가 정확한가? (G-VBCB77P21T)
- [ ] 트리거가 "All Pages"로 설정되었는가?

### 문제 3: Google Ads 데이터 미전송

**확인 checklist:**
- [ ] Google Ads 태그가 활성화되었는가?
