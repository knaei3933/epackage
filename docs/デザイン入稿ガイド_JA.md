# デザイン入稿ガイド

## 1. 印刷データ作成ガイド（ベストプラクティス）

イパッケージラボでは、ご入稿いただいたデータは当社で編集することができません。
デジタル印刷および軟包装製作時のトラブルを防ぎ、最高品質の結果を得るための必須ガイドラインです。

### カラーモードおよび設定
- **必ずCMYKモード**で作業し、小数点以下の数値は使用しません。
- DICやPANTONEなどの**特色（スポットカラー）**は、印刷時に自動変換されて色の差が生じるため、使用を禁止します。
  - （メインカラーのパントーンナンバーをご提示いただければ、可能な限り近い色で印刷可能です。）
- カラープロファイルは必ず**「Japan Color 2001 Coated」**に設定してください。

### ブラック（Black）の最適化
- **テキストおよびバーコード**: 印刷品質と読み取り精度のため、**K100%単色**の使用を強く推奨します。
- **広い背景面積**: 色の深みを出すために**C20 K100**のブラック設定を推奨します。
  - （CMYKの混色が多くなると印刷品質が低下する可能性があります。）

### テキストおよびオブジェクト処理
- すべての文字は**アウトライン（曲線化）**処理が必須です。
- パターンや効果（影など）を使用した場合、必ず**「分割・拡張（Expand）」**または**「透明度の分割」**を完了してください。

### 画像管理
- 画像は外部リンク方式ではなく、**「ファイルに埋め込み（Embed）」**状態である必要があります。
- 解像度は**300ppi以上**を維持してください。

### レイヤーおよびマスク構成
- デザインデータは必ず**「design」レイヤー**に別途区分して作業します。
- **「クリッピングマスク」**機能を使用して実際の製品サイズに合わせて整理し、不要な余白や空間の放置を禁止します。

### 製作仕様および安全エリア
- 実際のサイズ基準で四方**+3mm（箱型 +5mm）の裁ち落とし（ヌリタシ）**を含める必要があります。
- 重要な情報は断裁線から**最低5mm内側（安全エリア）**に配置し、加工時に切断されるのを防ぎます。

## 2. 白版データ作成ガイド（ベストプラクティス）

透明または金属質の包装材で色を正確に表現するためのガイドラインです。

### 白版の役割
- 無地フィルムに白印刷を先に行い、デザインが透けないようにして本来の色を鮮明に表現します。

### パウチ構成イラストについて
- 白版の使用有無により、透明感やメタリックな質感を生かすことができます。

### レイヤーおよびカラー指定
- デザインレイヤーとは別に**「白版」レイヤー**を作成します。
- 白版印刷が必要な部分は**青色（C100%）**で、不要な部分は白色で区分して作成します。
- 透明層が含まれるデザインは、作業者が混乱しないよう別の色で明確に区別して表示します。

### データ形式
- 白版にはビットマップ画像を使用できず、必ず**ベクター（パス）データ**で作成する必要があります。
- 写真画像を使用する場合も、必ずパスで切り抜いたデータを利用する必要があります。

### 素材別視覚効果の活用
- **アルミ/蒸着素材**: 白版がない部分はメタリックな質感（金/銀の感じ）で表現されます。
- **透明素材**: 白版がない部分は内部の内容物が透ける効果を出すことができます。

### 例外事項
- **白色クラフト紙**を素材として選択した場合に限り、別途の白版データ提出は不要です。
