## 1. Google Adsリマーケティング設定

### 1.1 前提条件

**以下の設定が完了している必要があります:**

- [x] GTMコンテナID: `GTM-T4PL5XMC`
- [x] GA4測定ID: `G-VBCB77P21T`
- [x] Google Adsリンク設定済み
- [x] リマーケティングタグ実装済み（GTM経由）

### 1.2 リマーケティングタグ設定（GTM経由）

**手順1: Google Adsリンク確認**

```
Google Ads > ツールと設定 > リンク済みアカウント
```

| ステップ | 操作 | 詳細 |
|----------|------|------|
| 1 | Google Analytics (GA4)を選択 | リンク済みであることを確認 |
| 2 | リンクステータス確認 | 「有効」であることを確認 |

**手順2: リマーケティングと広告報告機能を有効化**

```
Google Ads > ツールと設定 > データのまとめ > リマーケティングと広告報告機能
```

| 設定項目 | 値 |
|----------|-----|
| リマーケティング | ON |
| 広告報告機能 | ON |
| データの収集 | 「広告個人化設定」に同意 |

**キャンペーン基本設定:**

```yaml

サブタイプ: 標準
```

**配信方法:**

| 設定項目 | 値 |
|----------|-----|
| 配信方法 | 標準 |
| 頻度上限 | 同一ユーザー週3回まで |
| 配信スケジュール | 検索広告と同様（平日9:00-20:00+20%） |

## 2. Yahoo!リターゲティング設定

### 2.1 YDNリターゲティング前提条件

**Yahoo!タグ実装が必要です:**

- Yahoo!タグマネージャーまたは直接実装
- サイトIDの取得

### 2.2 Yahoo!タグ実装

**手順1: Yahoo!タグマネージャー設定**

1. [Yahoo!タグマネージャー](https://tag-manager.yahoo.co.jp/)にアクセス
2. サイトIDを取得

**共通タグ（全ページ）:**

```html
<!-- Yahoo!タグ マネージャー スクリプト -->
<script type="text/javascript">
  (function () {
    var tagjs = document.createElement("script");
    var s = document.getElementsByTagName("script")[0];
    tagjs.async = true;
    tagjs.src = "//s.yjtag.jp/tag.js#site=YJ_SITE_ID";
    s.parentNode.insertBefore(tagjs, s);
  }());
</script>
<noscript>
  <iframe src="//b.yjtag.jp/iframe?c=YJ_SITE_ID" width="1" height="1" frameborder="0" scrolling="no" marginheight="0" marginwidth="0"></iframe>
</noscript>
```

**実装例（Next.js）:**

```tsx
// src/app/layout.tsx
import Script from 'next/script';

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="ja">
      <head>
        {/* Yahoo!タグ */}
        <Script id="yahoo-tag" strategy="afterInteractive">
          {`
            (function () {
              var tagjs = document.createElement("script");
              var s = document.getElementsByTagName("script")[0];
              tagjs.async = true;
              tagjs.src = "//s.yjtag.jp/tag.js#site=${process.env.NEXT_PUBLIC_YAHOO_SITE_ID}";
              s.parentNode.insertBefore(tagjs, s);
            }());
          `}
        </Script>
      </head>
      <body>
        <noscript>
          <iframe
            src={`//b.yjtag.jp/iframe?c=${process.env.NEXT_PUBLIC_YAHOO_SITE_ID}`}
            width="1"
            height="1"
            style={{ display: 'none', visibility: 'hidden' }}
          />
        </noscript>
        {children}
      </body>
    </html>
  );
}
```

## 8. 運用開始後の最適化

### 8.1 週次チェック項目

| 項目 | 確認内容 | 対応基準 |
|------|----------|----------|
| CPA | 目標CPA以内か | 3,000円超過時調整 |
| CVR | 3%以上維持 | 2%未満時広告文改善 |
| フリークエンシー | 過剰配信なし | 週5回以上で頻度調整 |
| CTR | 1%以上 | 0.5%未満時クリエイティブ改善 |

### 8.2 月次最適化タスク

| タスク | 内容 | 頻度 |
|--------|------|------|
| リスト精査 | 低パフォーマンスリスト調整 | 月次 |
| 広告文A/Bテスト | 新規パターン追加 | 月次 |
| 入札単価調整 | CPA目標に基づく調整 | 月次 |
| 除外リスト更新 | 不要セグメント追加 | 月次 |

### 8.3 CPA調整スケジュール

| 月 | 目標CPA | 理由 |
