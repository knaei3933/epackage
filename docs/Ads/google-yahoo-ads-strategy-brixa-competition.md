## 1. 戦略概要

### 1.1 キャンペーン全体の方向性

**ミッション:** 韓国製高品質パッケージの「小ロット・短納期・低価格」を日本市場に浸透させ、Brixa等の競合から市場シェアを獲得する。

**戦略的ポジショニング:**
- **コストリーダーシップ:** 韓国製造の価格優位性を強調
- **スピード競争:** 最短21日納品のスピード感
- **品質保証:** 日本基準準拠の品質管理体制

### 1.2 ターゲット顧客層

| セグメント | 特徴 | 優先度 |
|-----------|------|--------|
| **食品メーカー** | 健康食品、スナック、調味料 | 高 |
| **化粧品ブランド** | スキンケア、ヘアケア、コスメ | 高 |
| **健康食品・サプリメント** | 国産、輸入品問わず | 高 |
| **ペットフードメーカー** | ドライフード、おやつ | 中 |
| **医薬品・医療機器** | 試薬、サンプル、小分け | 中 |
| **スタートアップ・個人企業** | 新規商品開発、少量試作 | 高 |

### 1.3 Brixa競合優位の基本戦略

1. **価格競争力:** 韓国製造による20-30%のコスト削減
2. **小ロット対応:** 500個〜対応（競合は1000個〜が一般的）
3. **納期短縮:** 最短21日（業界平均30-45日）
4. **AI見積もり:** 即時価格シミュレーション
5. **日本サポート:** 日本人スタッフによる完全サポート

#### 1. GTMコンテナ設定

**手順:**

| ステップ | 内容 | 詳細 |
|----------|------|------|
| 1-1 | GTMアカウント作成 | [Google Tag Manager](https://tagmanager.google.com/)で新規アカウント作成 |
| 1-2 | コンテナ作成 | コンテナ名: `epackage-lab-web`、ターゲット: Web |
| 1-3 | GTM ID取得 | `GTM-XXXXXXX`形式のIDを記録 |
| 1-4 | スニペット実装 | `src/app/layout.tsx`にGTMスニペットを追加 |

**GTMスニペット実装コード:**

```tsx
// src/app/layout.tsx に追加
// Head内（<head>タグの直後）
<Script id="gtm-head" strategy="beforeInteractive">
  {`
    (function(w,d,s,l,i){w[l]=w[l]||[];w[l].push({'gtm.start':
    new Date().getTime(),event:'gtm.js'});var f=d.getElementsByTagName(s)[0],
    j=d.createElement(s),dl=l!='dataLayer'?'&l='+l:'';j.async=true;j.src=
    'https://www.googletagmanager.com/gtm.js?id='+i+dl;f.parentNode.insertBefore(j,f);
    })(window,document,'script','dataLayer','GTM-XXXXXXX');
  `}
</Script>

// Body開始タグの直後（noscript）
<noscript>
  <iframe src="https://www.googletagmanager.com/ns.html?id=GTM-XXXXXXX"
    height="0" width="0" style={{display:'none',visibility:'hidden'}}>
  </iframe>
</noscript>
```

**dataLayer設定:**

```tsx
// カスタムイベント用dataLayerプッシュ関数
declare global {
  interface Window {
    dataLayer: Record<string, unknown>[];
  }
}

export const pushToDataLayer = (event: string, data: Record<string, unknown>) => {
  if (typeof window !== 'undefined' && window.dataLayer) {
    window.dataLayer.push({ event, ...data });
  }
};
```

**GA4実装コード:**

```tsx
// src/app/layout.tsx または専用コンポーネント
import Script from 'next/script';

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (

        />

          {`
            window.dataLayer = window.dataLayer || [];
            function gtag(){dataLayer.push(arguments);}
            gtag('js', new Date());
            gtag('config', '${process.env.NEXT_PUBLIC_GA_MEASUREMENT_ID}');
          `}
        </Script>
      </head>
      <body>
        {children}
      </body>
    </html>
  );
}
```

```yaml
# GTMタグ設定
```

## 9. 予算配分案

### 9.1 Google vs Yahooの配分比率

| 期間 | Google | Yahoo! | 理由 |
|------|--------|--------|------|
| **初期（1-2ヶ月）** | 80% | 20% | Googleで学習予算を集中（新規アカウント向け） |
| **中期（3-4ヶ月）** | 60% | 40% | Yahoo!効果測定・拡張 |
| **長期（5ヶ月〜）** | 55% | 45% | 最適配分へ調整 |

**初期80:20の理由:**
- 新規アカウントはGoogleで十分なデータ蓄積が必要
- 高ボリュームのGoogle検索で学習フェーズを短縮
- Yahoo拡張前にGoogleでCPA安定化を図る

### 9.2 キャンペーン別の配分

**月額20万円の場合（初期配分）:**

| キャンペーン | Google | Yahoo! | 合計 |
|-------------|--------|--------|------|
| ブランド | 12,000円 | 3,000円 | 15,000円 |
| 一般キーワード | 64,000円 | 16,000円 | 80,000円 |
| 競合キーワード | 40,000円 | 10,000円 | 50,000円 |
| ロングテール | 28,000円 | 7,000円 | 35,000円 |
| リマーケティング | 16,000円 | 4,000円 | 20,000円 |
| **合計** | **160,000円** | **40,000円** | **200,000円** |

### 9.3 季節性考慮

| 期間 | 予算調整 | 理由 |
|------|----------|------|
| **1-3月** | +20% | 年度始まり需要増 |
| **4-6月** | 標準 | - |
| **7-9月** | +10% | 夏季商材需要 |
| **10-12月** | +30% | 年末商戦・来年度準備 |
