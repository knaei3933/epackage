## 10. 문제 해결

### 10.1 광고 승인 거부

```
증상: 광고가 '승인 거부' 상태

해결:
1. 정책 위반 내용 확인
2. 수정 후 다시 제출
3. 승인 심사 요청 (필요시)
```

### 10.2 전환 기록되지 않음

```
증상: GA4에는 기록되나 Google Ads에는 없음

해결:
1. GA4와 Google Ads 연결 확인
2. 전환 설정에서 '데이터 소스' 확인
3. 24-48시간 대기
```

### 10.3 낮은 CTR (클릭률)

```
증상: CTR 2% 미만

해결:
1. 광고 문구 개선
2. 키워드 관련성 확인
3. 부정 키워드 추가
4. 입찰가 조정
```

### 10.4 높은 CPA

```
증상: 전환당 비용 10,000엔 초과

해결:
1. 키워드 분석
2. 저효율 키워드 제외
3. 광고 문구 최적화
4. 랜딩 페이지 개선
```
