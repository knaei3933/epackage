## 1. Yahoo!広告アカウント開設手順

### 1.1 アカウント作成フロー

| ステップ | 内容 | 詳細手順 | 所要時間 |
|----------|------|----------|----------|
| 1 | Yahoo! JAPANビジネスID取得 | [ビジネスID登録ページ](https://business.yahoo.co.jp/)から登録 | 10分 |
| 2 | Yahoo!広告アカウント作成 | ビジネスIDでログイン後、広告管理ツールにアクセス | 5分 |
| 3 | 支払い方法設定 | クレジットカードまたは口座振込を登録 | 10分 |
| 4 | アカウント承認待ち | 審査完了（通常1-2営業日） | 1-2日 |

### 1.2 詳細設定手順

#### ステップ1: Yahoo! JAPAN ビジネスID登録

1. [Yahoo!広告](https://ads-promo.yahoo.co.jp/)にアクセス
2. 「無料で始める」をクリック
3. 以下の情報を入力:
   - **会社名:** [自社会社名]
   - **担当者名:** [担当者名]
   - **メールアドレス:** business@epackage-lab.com（推奨）
   - **電話番号:** [代表電話番号]
   - **住所:** [本社住所]
4. メール認証完了

#### ステップ2: 広告管理ツール設定

1. [広告管理ツール](https://ads-promo.yahoo.co.jp/)にログイン
2. 以下の初期設定を実施:

```yaml
アカウント設定:
  タイムゾーン: (GMT+09:00) 日本標準時
  通貨: 日本円 (JPY)
  言語: 日本語
```

#### ステップ3: 支払い方法登録

**推奨: クレジットカード（即時開始可能）**

- 対応カード: Visa, Mastercard, JCB, AMEX
- 登録手順:
  1. 管理ツール > アカウント > 支払い方法
  2. 「クレジットカードを追加」を選択
  3. カード情報を入力
  4. 認証コードで承認

**代替: 口座振込（3-5営業日）**

- 振込手数料: 自負
- 初回入金: 10,000円以上

### 1.3 開設前チェックリスト

- [ ] ビジネスID登録完了
- [ ] 広告管理ツールログイン可能
- [ ] 支払い方法登録済み
- [ ] アカウント承認メール受信
- [ ] 初期入金（口座振込の場合）完了

## 2. キャンペーン作成手順

### 2.1 キャンペーン構成

| キャンペーン名 | 目的 | 予算配分 | 入札戦略 |
|---------------|------|----------|----------|
| 検索広告（一般） | リード獲得 | 60% | 目標CPA 5,000円 |
| 検索広告（競合） | 競合奪取 | 25% | 目標ROAS 300% |
| YDN（ディスプレイ） | 認知拡大 | 15% | 目標CPA 8,000円 |

### 2.2 検索広告キャンペーン作成

#### ステップ1: キャンペーン新規作成

1. 広告管理ツール > キャンペーン管理 > 新規キャンペーン
2. キャンペーンタイプ: **検索広告**

#### ステップ2: 基本設定

```yaml
キャンペーン名: Epackage_検索_一般
配信設定: 標準（ accelerated は非推奨）
予算: 日予算 3,000円 〜 （月額ベースで調整）
開始日: 設定当日
終了日: なし（継続配信）
```

#### ステップ3: 入札戦略設定

```yaml
入札戦略: 目標コンバージョン単価
目標CPA: 初期 8,000円 → 6ヶ月目 5,000円
```

**CPA調整スケジュール:**

| 期間 | 目標CPA | 理由 |
|------|---------|------|
| 1-2ヶ月目 | 8,000円 | 学習期間として高めに設定 |
| 3-4ヶ月目 | 6,000円 | データ蓄積後の最適化 |
| 5-6ヶ月目 | 5,000円 | 安定運用時の目標 |

#### ステップ4: ネットワーク設定

```yaml
配信ネットワーク:
  - Yahoo! JAPAN検索: ON
  - ディスプレイネットワーク（検索連動）: OFF
```

#### ステップ5: 開始・終了日設定

```yaml
配信開始日: 即時開始
配信終了日: 設定なし
```

### 7.2 Yahoo!タグ設定手順

#### ステップ1: コンバージョン定義

1. 広告管理ツール > ツール > コンバージョン
2. 「コンバージョンを作成」をクリック
3. 以下を設定:

```yaml
見積もり完了:
  コンバージョン名: 見積もり完了
  価値: 動的価値を使用
  通貨: JPY
  カウント: すべて
  コンバージョン発生までの期間: 30日
  考慮期間: 90日

お問い合わせ送信:
  コンバージョン名: お問い合わせ送信
  価値: 0円
  カウント: すべて
  コンバージョン発生までの期間: 30日
```

#### ステップ2: タグ実装

**共通タグ（全ページ）:**

```html
<!-- Yahoo!タグ マネージャー スクリプト -->
<script type="text/javascript">
  (function () {
    var tagjs = document.createElement("script");
    var s = document.getElementsByTagName("script")[0];
    tagjs.async = true;
    tagjs.src = "//s.yjtag.jp/tag.js#site=YJ_SITE_ID";
    s.parentNode.insertBefore(tagjs, s);
  }());
</script>
<noscript>
  <iframe src="//b.yjtag.jp/iframe?c=YJ_SITE_ID" width="1" height="1" frameborder="0" scrolling="no" marginheight="0" marginwidth="0"></iframe>
</noscript>
```

**コンバージョンタグ（見積もり完了時）:**

```html
<script type="text/javascript">
  (function () {
    var YahooJAPAN = window.YahooJAPAN || {};
    YahooJAPAN.DirectConversion = YahooJAPAN.DirectConversion || {};
    YahooJAPAN.DirectConversion.pageType = "order";
    YahooJAPAN.DirectConversion.price = [見積金額];
  })();
</script>
```

**実装例（Next.js）:**

```tsx
// src/lib/yahoo-tracking.ts
export const pushYahooConversion = (type: 'quote' | 'contact' | 'sample', value?: number) => {
  if (typeof window === 'undefined') return;

  const YahooJAPAN = (window as any).YahooJAPAN || {};
  YahooJAPAN.DirectConversion = YahooJAPAN.DirectConversion || {};

  switch (type) {
    case 'quote':
      YahooJAPAN.DirectConversion.pageType = 'order';
      YahooJAPAN.DirectConversion.price = value;
      break;
    case 'contact':
      YahooJAPAN.DirectConversion.pageType = 'contact';
      break;
    case 'sample':
      YahooJAPAN.DirectConversion.pageType = 'sample';
      break;
  }
};
```
