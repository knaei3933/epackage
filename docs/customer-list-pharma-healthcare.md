## 5. GMP認証取得状況の調査方法

### 公的機関・関連団体

**医薬品関連**
1. **PMDA（医薬品医療機器総合機構）**
   - URL: https://www.pmda.go.jp/
   - GMP適合性調査結果公表ページ
   - 製造所ごとの認証状況を確認

2. **厚生労働省**
   - 薬機法関連製造所許認可リスト
   - 医薬品製造業許可名簿

**健康食品関連**
1. **日本健康・栄養食品協会（JHFA）**
   - JHFA GMP認定工場一覧
   - 認定企業リスト公開

2. **日本GMP研究会**
   - 認証取得企業データベース

### 調査手順
1. 対象業種のリスト化
2. PMDA/JHFAデータベースでの認証確認
3. 企業ウェブサイトでの品質方針確認
4. 製造所情報の収集

## 7. 薬機法対応状況の調査方法

### 確認項目

**医薬品メーカー**
- [ ] 製造販売業許可取得（厚生労働省）
- [ ] GMP適合性調査合格（PMDA）
- [ ] 品質システム導入状況
- [ ] 包装管理区分対応

**健康食品メーカー**
- [ ] JHFA GMP認証取得
- [ ] ISO22000/HACCP認証
- [ ] 機能性表示食品届出状況
- [ ] 包装品質管理システム

### 調査ツール
1. PMDAデータベース検索
2. JHFA認証企業検索
3. 企業決算短信・IR情報
4. 業界ニュース・プレスリリース

## 8. 優先順位付け（規模・GMP対応ニーズ）

### 評価基準

**Aランク（最優先ターゲット）**
- 規模：従業員500名以上、売上高500億円以上
- GMP対応：完了・認証取得済み
- 包装ニーズ：高（医薬品・医療機器製造）
- 該当：大手製薬20社、ジェネリック大手5社

**Bランク（優先ターゲット）**
- 規模：従業員100〜500名、売上高100〜500億円
- GMP対応：完了・認証取得済み
- 包装ニーズ：中〜高
