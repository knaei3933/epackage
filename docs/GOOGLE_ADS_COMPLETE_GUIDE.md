## 8. ランディングページ最適化

### 品質スコア構成要素（2025）

| 要素 | 重要度 |
|------|--------|
| 想定CTR | ⭐⭐⭐ |
| 広告の関連性 | ⭐⭐⭐ |
| **ランディングページの利便性** | ⭐⭐⭐⭐⭐ |

---

### ランディングページ要件

#### 1. ページ読み込み速度

```
目標: 3秒以内（理想的には1.2秒未満）
実装:
- WebP画像圧縮
- CDN導入
- AMP対応（可能な場合）
- 静的リソース最適化

影響: 3秒→1秒で品質スコア21%向上
```

---

#### 2. コンテンツの関連性と深さ（2025年新要件）

```
必須要素:
✅ 詳細な製品仕様
✅ 使用シナリオ説明
✅ 顧客レビュー・事例
✅ コアキーワードをファーストビューに表示

避けるべき:
❌ 画像のみの簡単な説明
❌ 具体性のない一般的表現
```

---

#### 3. 商業的誠実性（2025年新要件）

```
必須表示:
✅ 明確な価格情報
✅ 送料透明化
✅ 返品ポリシー（例: 「60日無料返品」）
✅ 信頼シグナル（ISO認証、実績）

避けるべき:
❌ 曖昧な「お得」表示
❌ 隠れたコスト
```

---

#### 4. モバイルフレンドリー

```
必須要件:
✅ 完璧なモバイル適応
✅ コンバージョンボタン80%以上の可視性
✅ スマホ・タブレットでの円滑ナビゲーション
✅ 指でのタップ誤操作防止
```

---

#### 5. ユーザー体験要素

```
必須要件:
✅ 明確なページレイアウト
✅ 分かりやすいナビゲーション
✅ 3秒で価値提案を理解
✅ 構造化データ（Schema.org）実装
```

---

### 日本市場特有の要件

#### 技術要件

```
✅ 東京/大阪サーバー（高速接続）
✅ モバイル最適化（日本はスマホ利用率高）
```

## 10. KPI・目標値

### 検索広告ベンチマーク

| 指標 | 目標値 | 警告ライン |
|------|--------|-----------|
| **CTR** | 3〜5% | < 2% |
| **品質スコア** | 7〜10 | < 5 |
| **コンバージョン率** | 2〜5% | < 1% |
| **CPA** | 顧客LTVの30%以内 | - |
| **品質リード率** | ≥30% | < 20% |

```
【意決定層】
コンバージョン率: 5〜10%
CPA: 目標範囲内
リード品質スコア: 80点以上
```
