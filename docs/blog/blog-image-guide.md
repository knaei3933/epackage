# ブログ記事画像仕様書

## 画像命名規則

```
blog/{記事番号}-{画像種類}-{連番}.ext

例:
- blog/06-hero-01.jpg
- blog/06-section-01.png
- blog/06-comparison-01.svg
```

## 画像サイズ規格

| 種類 | サイズ | 用途 |
|------|--------|------|
| **Hero (メイン)** | 1920 x 1080px | 記事ヘッダー、OGP用 |
| **Section Header** | 1200 x 630px | セクション見出し用 |
| **Product Shot** | 800 x 800px | 製品紹介 |
| **Comparison** | 1200 x 800px | 比較表、図解 |
| **Lifestyle** | 1200 x 900px | 使用シーン |
| **Icon** | 256 x 256px | アイコン、小画像 |
| **Thumbnail** | 400 x 300px | 記事リスト用 |

## 画像最適化ガイド

### ファイル形式
- 写真: JPG (品質 85%)
- 図解・アイコン: PNG ( transparency 付き )
- インフォグラフィック: SVG または PNG

### 画像圧縮
- Hero: 最大 300KB
- Section: 最大 200KB
- Product: 最大 150KB
- Thumbnail: 最大 50KB

### Alt テキスト
必須: SEO 対策のため、すべての画像に alt テキストを設定

```
例:
alt="スタンドパウチの製品写真、コーヒー豆が入った自立袋"
alt="ガゼットパウチの構造図、マチ部分の説明"
```

## 画像生成ツール設定

### Midjourney 設定
```
--ar 16:9 (Hero用)
--ar 3:2 (Section用)
--ar 1:1 (Product用)
--style raw
--v 6.0
```

### DALL-E 3 設定
```
"Professional product photography, [詳細説明],
studio lighting, white background,
Japanese aesthetic, commercial quality,
high resolution, sharp focus"
```

### Stable Diffusion 設定
```
Steps: 30-50
CFG Scale: 7-9
Sampler: DPM++ 2M Karras
Size: 512x512 (base) → upscale to target
```
