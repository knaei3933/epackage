# ブログ記事画像生成プロンプト（コピー＆ペースト用）

## 使用方法

1. **プロンプト**をコピー
2. nanobanana MCPで実行
3. 生成された画像をファイル名で保存

---

## 共通パラメータ

```json
{
  "quality": "high",
  "candidate_count": 1
}
```

各画像の `aspect_ratio` は以下の通り：
- Hero: `"16:9"`
- Section: `"2:1"`
- Product: `"1:1"`
- Comparison: `"3:2"`
- Lifestyle: `"4:3"`
- Thumbnail: `"4:3"`
