# 画像生成ツール別最適化

## Midjourney 設定

### 共通パラメータ
```
--ar 16:9 (Hero用)
--ar 2:1 (Section Header用)
--ar 1:1 (Product Shot用)
--ar 3:2 (Comparison用)
--ar 4:3 (Lifestyle用)
--style raw (スタイルを生かす)
--v 6.0 (最新バージョン)
--q 2 (高品質)
```

### 推奨ワークフロー
1. まず `--style raw --v 6.0` でベースを生成
2. 気に入らない場合は `--s 250〜750` でスタイライズ度合いを調整
3. 具体的な修正は `Remix` モードでプロンプトを調整

## DALL-E 3 プロンプト最適化

### 構造テンプレート
```
[主題] + [詳細な描写] + [スタイル指定] + [技術仕様] + [ブランド要素]

例:
"Professional product photography of flat pouch filled with coffee beans,
matte finish packaging with elegant typography,
standing on pristine white background with studio lighting,
Japanese minimalist aesthetic, commercial quality,
shot with Canon EOS R5, 100mm macro lens, f/2.8 aperture,
8K resolution, sharp focus"
```

### 最適化ポイント
- 具体的なカメラ・レンズ指定を含める
- ライティングの方向と質を指定
- 背景の色をHEXコードで指定
- 日本的な美学を明示

## Stable Diffusion 設定

### 推奨パラメータ
```
Steps: 30-50
CFG Scale: 7-9
Sampler: DPM++ 2M Karras または DPM++ SDE Karras
Size: 512x512 (base) → upscale to target
VAE: vae-ft-mse-840000
```

### Negative Prompt（必須）
```
low quality, blurry, distorted, deformed, ugly, bad anatomy,
watermark, text, signature, logo, oversaturated, undersaturated,
poor lighting, amateur, snapshot, grainy, noisy
```

### Upscaling
```
Model: Real-ESRGAN 4x+
Target Size: 目標サイズに合わせて
Denoise Strength: 0.3-0.5
```

# 画像最適化ガイド

### ファイル形式
- **写真**: JPG (品質 85%, プログレッシブ)
- **図解・アイコン**: PNG (透明チャンネル付き)
- **インフォグラフィック**: SVG (ベクター) または PNG

### 画像圧縮目標
| 種類 | 最大サイズ | 推奨形式 |
|------|-----------|---------|
| Hero | 300KB | JPG progressive |
| Section | 200KB | PNG-8 または JPG |
| Product | 150KB | JPG |
| Thumbnail | 50KB | JPG |
