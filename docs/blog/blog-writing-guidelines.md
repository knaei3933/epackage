## 記事フォーマット

### タイトル構成

**基本フォーマット:**
```
【カテゴリー】メインキーワード｜サブタイトル
```

**例:**
- `【製品紹介】ガゼットパウチ｜大容量商品を自立させるパッケージの新常識`
- `【印刷知識】バリアブル印刷とは？｜小ロット多品種を実現する印刷技術`
- `【初心者向け】１分で最適なパッケージ見積書が作れる「スマート見積り」の使い方ガイド`

**インタビュー記事のタイトルパターン:**
- `企業名 × キーワード｜魅力的なサブタイトル`
- 例: `高精細印刷×30デザイン。"名刺代わり"の象徴的アイテムにまで昇華した、and/or Granolaのパッケージ革命。`

**製品紹介記事:**
【製品紹介】{製品名}｜{サブタイトル}
例: 【製品紹介】ガゼットパウチ｜大容量商品を自立させるパッケージの新常識

**印刷技術記事:**
【印刷技術】{技術名}｜{サブタイトル}
例: 【印刷技術】バリアブル印刷とは？｜小ロット多品種を実現する印刷技術

**導入事例記事:**
【導入事例】{企業名}×{キーワード}｜{サブタイトル}
例: 【導入事例】健康食品メーカーA社×高精細印刷｜魅力的なパッケージの秘訣

### 日付フォーマット

- **形式**: `YYYY.MM.DD`
- **配置**: 記事タイトルの下、カテゴリータグの横

## コンテンツ構成

### リード文（導入部）

**型:**
```
[読者の状況・悩み] + [現状の課題] + [この記事で得られる価値]
```

**例:**
```
「大容量の商品を包みたいけれど、袋が自立せずに困っていませんか？
従来の平袋では収納性や陳列効果に限界があります。
この記事では、ガゼットパウチの特徴と選び方を分かりやすく解説します。」
```

### 記事構成テンプレート

```markdown
1. リード文（課題提起）
2. 目次（箇条書き形式、アンカーリンク付き）
3. H2: 定義・基本情報
4. H2: メリット・活用シーン
5. H2: 選び方・比較
6. H2: 発注フロー（STEP形式）
7. CTA
8. 関連記事・Ranking
```

### 見出し階層

| レベル | 用途 | フォントサイズ |
|--------|------|---------------|
| H1 | 記事タイトル | 48px |
| H2 | メインセクション | 32px |
| H3 | サブセクション | 26px (カード), 18px (一覧) |

### 段落と読みやすさ

- **短い段落**: 2〜4文で区切る
- **1文1段落**も多用（強調効果）
- **箇条書き**を頻繁に使用して情報を整理

## トーン＆マナー

### 文体

- **です・ます調**：全記事で統一
- **丁寧語**：読者に寄り添う語り口
- **専門用語も平易に説明**：初心者向けの配慮

### 読者への語りかけ方

**問いかけ多用:**
- 「迷っていませんか？」
- 「思ったことはありませんか？」
- 「いかがでしょうか？」

**共感表現:**
- 「という悩みは非常に多くあります」
- 「といった失敗も起こりがちです」

### 専門性と分かりやすさのバランス

**専門性のアピール:**
- 具体的な数値（「10万円以内と国内最安値級」）
- 実績紹介（「Voyce 世界を旅するプロテイン」）

**分かりやすさの工夫:**
- 結論から先に述べる（「結論から言うと、...」）
- 表や箇条書きで整理
- ステップ形式で手順を説明（STEP1〜STEP4）

## 記事タイプ別の構成パターン

| 記事タイプ | 構成の特徴 | トーン |
|-----------|-----------|--------|
| **製品紹介** | 目的→特徴→活用シーン→発注フロー→CTA | 専門的かつ親しみやすく、実用性重視 |
| **インタビュー** | 企業背景→課題→解決策→成果→今後の展望 | 語り口調でリアリティのあるお客様の声 |
| **技術解説** | 定義→仕組み→メリット/デメリット→活用事例→発注方法 | 分かりやすい説明と具体例で信頼性を構築 |

---

## 信頼性構築の要素

1. **具体的な数値やスペックの提示**
2. **実績・事例の紹介**
3. **表・図解での情報整理**
4. **専門用語の平易な説明**
