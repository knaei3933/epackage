## 1. ガゼットパウチとは

ガゼットパウチ（Gusset Pouch）は、袋の側面や底部に「マチ」と呼ばれる折りたたみ部分を設けたパウチのことです。「マチ付きパウチ」や「側面マチパウチ」とも呼ばれます。

### 基本構造

| 部位 | 説明 |
|------|------|
| **前面・背面** | ブランドロゴや商品情報を印刷するメイン面 |
| **ガゼット（マチ）** | 側面または底部の折りたたみ部分 |
| **チャック** | 開閉可能なジッパー（オプション） |
| ** notch（ノッチ）** | 開封用の切り込み（オプション） |

### 主な種類

- **側面マチタイプ**: 両サイドにマチがある最も一般的なタイプ
- **底部マチタイプ**: 底部にマチがあり、底面が広く取れるタイプ
- **四面マチタイプ**: 上下左右にマチがあるボックス型のタイプ

## 2. ガゼットパウチのメリット・活用シーン

### メリット1: 大容量化を実現

マチ部分が広がることで、平袋よりも多くの内容物を充填できます。

- 容積確保: 同じサイズの平袋と比較して1.5〜2倍の容量
- 見た目のボリューム感: 商品の価値を視覚的に訴求

### メリット2: 自立して陳列効果アップ

マチが広がることで袋が自立し、店頭での陳列効果が飛躍的に向上します。

- ゲート型陳列に対応
- 棚からの転倒を防止
- 商品の顔を常に見せられる

### メリット3: 省スペース収納

収納時はマチを折りたたむことで、コンパクトに保管できます。

- 梱包時のスペース節約
- 輸送コストの削減

### 活用シーン

| 商品カテゴリー | 具体例 |
|---------------|--------|
| **スナック菓子** | ポテトチップス、煎餅、ナッツ類 |
| **穀物・穀粉** | 米、麦、パン粉、米粉 |
| **粉もの** | プロテインパウダー、小麦粉、砂糖 |
| **ペットフード** | ドライフード、おやつ |
| **コーヒー豆** | 業務用サイズ、家庭用大容量 |

## 3. 選び方のポイント

### マチ幅の選定

内容物の量と自立性を考慮してマチ幅を決定します。

| マチ幅 | 想定容量 | 自立性 |
|--------|----------|--------|
| 30-50mm | 100-300g | 標準 |
| 50-80mm | 300-500g | 高い |
| 80-100mm | 500g-1kg | 非常に高い |

### 素材の選定

内容物の特性に合わせて適切な素材を選びます。

| 素材 | 特徴 | 適した商品 |
|------|------|-----------|
| **PET/AL/PE** | 高いバリア性、遮光性 | コーヒー豆、乾燥食品 |
| **PET/PE** | 透明性、コストパフォーマンス | スナック菓子 |
| **KRAFT/PE** | 環境対応、ナチュラル感 | オーガニック製品 |

### オプション機能

必要に応じて以下の機能を追加できます。

- **チャック付き**: 開閉可能で保存性向上
- **degassingバルブ**: コーヒー豆のガス抜き対応
- **ハングホール**: フック掛け陳列対応
