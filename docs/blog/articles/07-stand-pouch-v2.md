## 1. スタンドパウチとは

スタンドパウチ（Stand Up Pouch）は、底部が特殊な形状で、内容物を入れると自立する袋のことです。自立性によって、店頭での陳列効果が飛躍的に向上します。

### 基本的な特徴

- **自立性**: 内容物を入れると自立する
- **陳列効果**: 商品の顔を常に見せられる
- **省スペース**: 空の時はコンパクトに収納可能
- **訴求力**: 目立つことで売上に貢献

### 仕組みの詳細

底部のW字型構造が自立を支えています。

1. 底部がW字に折りたたまれている
2. 内容物を充填すると、底部が広がる
3. その反力で袋が自立する
4. 常に正面を消費者に向けることができる

## 2. 特徴と種類

### 基本構造

| 部位 | 役割 |
|------|------|
| **前面** | ブランドロゴ、商品名、主要ビジュアル |
| **背面** | 成分表示、使用方法、会社情報 |
| **底部（ガゼット）** | W字型に折りたたまれ、自立を支える |
| **チャック** | 開閉可能なジッパー（オプション） |
| **notch（ノッチ）** | 開封用の切り込み（オプション） |
| **ハングホール** | フック掛け用の穴（オプション） |

### 主な種類

#### 1. プレーンスタンドパウチ

最も一般的なタイプで、シンプルなデザインが特徴です。

**特徴:**
- コストパフォーマンスが高い
- デザインの自由度が高い
- 最も多くの用途に対応

**適した商品:**
- スナック菓子
- ドライフルーツ
- 粉もの
- 日用品

#### 2. 丸底スタンドパウチ

底部が丸くカットされた、高級感のあるタイプです。

**特徴:**
- 高級感が高い
- 自立性が非常に良い
- 独特の形状で差別化できる

**適した商品:**
- プレミアム食品
- ギフト商品
- 高価なナッツ類

#### 3. フラットボトムパウチ

底部が完全に平らな、安定感のあるタイプです。

**特徴:**
- 非常に安定している
- 大容量にも対応
- ボックス型のような高級感

**適した商品:**
- 大容量商品
- 高級食品
- 贈答品

#### 4. ジッパー付きスタンドパウチ

開閉可能なチャックが付いた、便利なタイプです。

**特徴:**
- 再利用可能
- 保存性が高い
- 消費者に好まれる

**適した商品:**
- 日常的に使う商品
- 大容量の商品
- 残りを保存したい商品

## 3. メリット・デメリット

### メリット1: 陳列効果の飛躍的向上

自立性により、店頭での訴求力が大幅に向上します。

**具体的な効果:**
- 商品が倒れない
- 常に正面を表示できる
- ゲート陳列対応可能
- 複数商品を整列しやすい

### メリット2: ブランドイメージの向上

自立することで、以下のようなブランドイメージの向上が期待できます。

| イメージ | 説明 |
|----------|------|
| **高品質感** | 立ち上がる姿から高級感を感じさせる |
| **信頼感** | 安定した陳列から信頼性を感じさせる |
| **プロフェッショナル** | しっかりとした印象を与える |

### メリット3: 多様なオプション対応

スタンドパウチは、様々な機能を追加できます。

**追加機能:**
- チャック付き：再利用可能に
- ハングホール：フック掛け対応
- デガスバルブ：ガス抜き対応
- 窓付き：商品を部分的に表示

### デメリット

| デメリット | 説明 | 対策 |
|-----------|------|------|
| **コストが高い** | 平袋より高価 | 大量発注で単価を抑える |
| **デザインが難しい** | 立体形状のため設計に注意 | デザイン専門家に相談 |
| **収納スペース** | 空でもある程度のスペースが必要 | コンパクトな形状を選択 |

## 5. 選び方のポイント

### サイズ選定

商品の大きさに合わせたサイズを選びます。

#### サイズの選び方

```
袋の幅 = 商品幅 + 10〜20mm（余白）
袋の高さ = 商品高さ + 50〜80mm（底部シール分 + 頭の部分）
```

#### サイズ別の特徴

| サイズ | 特徴 | 適した商品 |
|--------|------|-----------|
| **小（100×150mm以下）** | コンパクト | 試供品、少量商品 |
| **中（120×200mm程度）** | 標準的な大きさ | 一般的な食品 |
| **大（150×250mm以上）** | 大容量対応 | ファミリーサイズ |

### 素材選び

用途に合わせた素材を選びます。Epackage Labでは4種類のフィルム構成から選択できます。

#### フィルム構成の種類

| 素材構成 | 特徴 | 適した商品 |
|---------|------|-----------|
| **PET AL** | 最強のバリア性、完全遮光 | コーヒー豆、保存食、医療品 |
| **PET VMPET** | 高いバリア性、金属光沢 | 一般的な食品、スナック菓子 |
| **PET LLDPE** | 透明性、ヒートシール性 | ナッツ類、乾燥食品 |
| **PET NY AL** | 高強度、優れたバリア性 | 重量物、鋭利な形状商品 |

#### 厚さ（強度）の選択

各素材構成は4つの厚さから選べます：

| タイプ | 内容重量 | フィルム構成（PET AL） | フィルム構成（PET VMPET） | フィルム構成（PET LLDPE） | フィルム構成（PET NY AL） |
|--------|----------|----------------------|------------------------|------------------------|------------------------|
| **軽量タイプ** | ~100g | PET 12μm / AL 7μm | PET 12μm / VMPET | PET 12μm / LLDPE 50μm | PET 12μm / NY 15μm / AL 7μm |
| **標準タイプ** | ~500g | PET 12μm / AL 9μm | PET 12μm / VMPET | PET 12μm / LLDPE 70μm | PET 12μm / NY 15μm / AL 9μm |
| **高耐久タイプ** | ~800g | PET 12μm / AL 12μm | PET 12μm / VMPET | PET 12μm / LLDPE 90μm | PET 12μm / NY 15μm / AL 12μm |
| **超耐久タイプ** | ~1kg以上 | PET 12μm / AL 15μm | PET 12μm / VMPET | PET 12μm / LLDPE 100μm | PET 12μm / NY 15μm / AL 15μm |

> 💡 **選び方のポイント**: スタンドパウチは自立するため、自立時の底部の負荷を考慮して「標準タイプ」以上を推奨します。

![フィルム構成図](/images/blog/shared/film-structure-diagram.png)

### オプションの選び方

#### チャックの有無

- **必要な場合**: 日常的に使う商品、大容量商品
- **不要な場合**: 一回きりの商品、試供品

#### ハングホールの有無

- **必要な場合**: フック掛け陳列を行う場合
- **不要な場合**: 棚陳列の場合

#### デガスバルブの有無

- **必要な場合**: コーヒー豆、煎った豆類
- **不要な場合**: 一般的な食品

## 6. 発注から納品まで

Epackage Labでのスタンドパウチ発注は、以下のSTEPで進めます。

### STEP1: スマート見積り

サイズ、素材、数量を入力し、即座に見積もりを取得します。

- サイズ: 幅×高さ
- 底部の形状
- 素材構成
- 印刷色数
- 数量
- オプション（チャック、ハングホールなど）

**所要時間: 約1分**

### STEP2: 仕様決定

必要に応じて以下のオプションを決定します。

- チャックの有無
- ハングホールの有無
- notch（ノッチ）の有無
- degassingバルブ（コーヒー用）

### STEP3: デザイン入稿

デザインデータをアップロードします。

#### デザイン仕様

| 項目 | 推奨仕様 |

| **カラーモード** | CMYK |
| **解像度** | 300dpi以上 |
| **サイズ** | 実寸プラス塗り足し3mm |

- 前面に商品の魅力を集中
