## 1. 合掌袋とは

合掌袋（がっしょうぶくろ）は、底部が三角形に折りたたまれた形状のパッケージです。名前の由来は、底部を折りたたんだ時に、まるで両手を合わせたような形状になることからきています。

### 基本構造

| 部位 | 説明 |
|------|------|
| **上部（開口部）** | 商品の出し入れをする部分 |
| **側面** | ブランドロゴやデザインを印刷 |
| **底部（三角部分）** | 三角形に折りたたまれた特徴的な部分 |
| **シール部分** | 上部をシールで閉じる |

### 仕組み

1. 袋を開くと、側面が広がる
2. 底部が三角形に固定されている
3. 内容物を入れると自立する
4. 独特の形状で目立つ

## 2. 特徴と種類

### 形状の特徴

#### 三角形底部

合掌袋の最大の特徴は三角形の底部です。

**特徴:**
- 独自の形状で目立ちやすい
- 自立性が高い
- 空間効率が良い

#### シンプルな側面

側面は平らな形状で、デザインの自由度が高いです。

**特徴:**
- デザインの幅が広い
- 印刷しやすい
- 情報表示がしやすい

### 種類の分類

#### 1. 標準合掌袋

最も一般的なタイプです。

**特徴:**
- 基本的な合掌袋形状
- コストパフォーマンスが高い
- 多くの用途に対応

**適した商品:**
- スナック菓子
- ドライフルーツ
- ナッツ類

#### 2. 大容量合掌袋

サイズが大きいタイプです。

**特徴:**
- 大容量に対応
- 視覚的にボリューム感がある
- 高級感がある

**適した商品:**
- ファミリーサイズ商品
- 贈答品
- 大容量ドライフード

#### 3. 透明合掌袋

素材の透明性を活かしたタイプです。

**特徴:**
- 商品がそのまま見える
- ナチュラルな印象
- 商品の魅力をアピール

**適した商品:**
- ハーブ
- ドライフルーツ
- 天然素材商品

#### 4. オプション付き合掌袋

追加機能付きのタイプです。

| オプション | 効果 | 適した商品 |
|-----------|------|-----------|
| **チャック付き** | 開閉可能 | 日常的に使う商品 |
| **ハングホール** | フック掛け | 陳列スペースの少ない場合 |
| **窓付き** | 商品の一部が見える | 見た目のアピールが必要な場合 |

![合掌袋の種類比較](/images/blog/11-lap-bag/11-products-grid-01.png)

### 材質による分類

Epackage Labでは4種類のフィルム構成から選択できます。

#### フィルム構成の種類

| 素材構成 | 特徴 | 適した商品 |
|---------|------|-----------|
| **PET AL** | 最強のバリア性、完全遮光 | コーヒー豆、保存食、医療品 |
| **PET VMPET** | 高いバリア性、金属光沢 | 一般的な食品、スナック菓子 |
| **PET LLDPE** | 透明性、ヒートシール性 | ナッツ類、乾燥食品 |
| **PET NY AL** | 高強度、優れたバリア性 | 重量物、鋭利な形状商品 |

#### 厚さ（強度）の選択

各素材構成は4つの厚さから選べます：

| タイプ | 内容重量 | フィルム構成（PET AL） | フィルム構成（PET VMPET） | フィルム構成（PET LLDPE） | フィルム構成（PET NY AL） |
|--------|----------|----------------------|------------------------|------------------------|------------------------|
| **軽量タイプ** | ~100g | PET 12μm / AL 7μm | PET 12μm / VMPET | PET 12μm / LLDPE 50μm | PET 12μm / NY 15μm / AL 7μm |
| **標準タイプ** | ~500g | PET 12μm / AL 9μm | PET 12μm / VMPET | PET 12μm / LLDPE 70μm | PET 12μm / NY 15μm / AL 9μm |
| **高耐久タイプ** | ~800g | PET 12μm / AL 12μm | PET 12μm / VMPET | PET 12μm / LLDPE 90μm | PET 12μm / NY 15μm / AL 12μm |
| **超耐久タイプ** | ~1kg以上 | PET 12μm / AL 15μm | PET 12μm / VMPET | PET 12μm / LLDPE 100μm | PET 12μm / NY 15μm / AL 15μm |

> 💡 **選び方のポイント**: 合掌袋は底部が三角形になるため、底部の強度が必要です。「標準タイプ」以上を推奨します。

## 3. メリット・デメリット

### メリット1: 差別化効果抜群

合掌袋の最大のメリットは、独自の形状による差別化効果です。

**差別化の具体例:**

| 効果 | 説明 |
|------|------|
| **目立ちやすい** | 他のパッケージと違う形状で注目を集める |
| **記憶に残りやすい** | 独特の形状で印象に残る |
| **話題性** | 「珍しい形」として口コミになる |

### メリット2: 高級感の演出

三角形の底部が、高級感を演出します。

**高級感の要素:**

| 要素 | 説明 |
|------|------|
| **立体感** | 三角形の形状で立体感が出る |
| **質感** | 素材の質感が際立つ |
| **ブランドイメージ** | ユニークな形状で高級感を感じさせる |

### メリット3: 実用性の高さ

見た目だけでなく、実用的な側面もあります。

**実用性の特徴:**

- **自立性**: 陳列時に自立する
- **省スペース**: 収納時にコンパクト
- **取り出しやすい**: 底部の形状で商品が取り出しやすい

### メリット4: 環境意識の表現

環境対応素材を選ぶことで、環境意識を示せます。

**環境意識の表現方法:**

- **リサイクル素材**: 環境配慮をアピール
- **減量化**: 従来の容器より少ない素材
- **ナチュラル素材**: オーガニックブランドに適合

### デメリット

| デメリット | 説明 | 対策 |
|-----------|------|------|
| **コストが高い** | 通常のパウチより高価 | 大量発注で単価を抑える |
| **デザインが難しい** | 独特の形状のため設計に注意 | 専門家に相談 |
| **製造ロットの制限** | 小ロットではコスパが悪い | 必要最小限の数量を計画 |

## 5. 選び方のポイント

### 形状の選び方

#### サイズ選定

内容物の量に合わせたサイズを選びます。

```
上部の幅 = 商品幅 + 10〜20mm
上部の高さ = 商品高さ + 30〜50mm
底部の三角形 = 上部の幅÷2程度
```

#### 形状のバランス

- **上部と底部のバランス**: 視覚的に美しい比率
- **全体のバランス**: 陳列時の安定感
- **開閉のしやすさ**: 使用時の操作性

### 素材の選び方

#### イメージに合わせた素材

| イメージ | 適した素材 | 具体例 |
|----------|-----------|--------|
| **ナチュラル** | KRAFT/PE | オーガニック商品 |
| **モダン** | PET/PE | 一般的な食品 |
| **高級** | マット素材 | ギフト商品 |
| **伝統** | 和紙素材 | 和菓子、日本茶 |

#### 機能性の考慮

- **遮光性**: 保存が必要な商品
- **耐湿性**: 湿気に弱い商品
- **耐油性**: 油ものの商品

### デザインのポイント

#### デザインの特徴

- **シンプルさ**: 合掌袋の形状を活かす
- **ブランドの強調**: ロゴを前面に配置
- **特徴のアピール**: 独特の形状を活かしたデザイン

#### 色使い

- **色相と商品の関係**: 商品の性質に合わせた色選び
- **ブランドカラー**: 統一感を出す
- **アクセント色**: 目立つ色で注目を集める

### オプションの選び方

#### チャックの有無

- **必要な場合**: 再利用したい商品
- **不要な場合**: 一回きりの商品

#### ハングホール

- **必要な場合**: フック掛け陳列を行う場合
- **不要な場合**: 棚陳列の場合

スマート見積りで以下の情報を入力します。

- サイズ（上部の幅×高さ）
