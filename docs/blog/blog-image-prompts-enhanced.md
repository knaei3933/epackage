## 画像生成プロンプト構造

### 基本テンプレート

```
[主題] + [シーン描写] + [スタイル指定] + [技術要件] + [ブランド要素]
```

### プロンプト要素

1. **主題（Subject）**: メインとなる被写体
2. **シーン描写（Scene）**: 具体的な状況、背景、環境
3. **スタイル指定（Style）**: 写真、イラスト、3D、ミニマルなど
4. **技術要件（Technical）**: ライティング、アングル、焦点
5. **ブランド要素（Brand）**: Epackage Labの雰囲気、色使い

```
✅ 良果的: "white background (#FFFFFF)", "purple accent (#8380FF)"
❌ 避ける: "white or blue background", "purplish background"
```

### ライティング
```
✅ 効果的: "professional studio lighting with soft diffused light from left"
✅ 効果的: "three-point studio lighting: key, fill, rim"
❌ 避ける: "good lighting", "bright lights"
```

✅ 効果的: "shot with Canon EOS R5, 100mm macro lens, f/2.8"
✅ 効果的: "commercial grade quality", "professional product photography"
