# ブログ記事画像生成プロンプト（nanobanana最適化・コピー用）

## 使用方法

1. **プロンプト**をコピー
2. nanobanana MCPで実行
3. 生成された画像を**ファイル名**で保存

---

## 共通パラメータ

```json
{
  "quality": "high",
  "candidate_count": 1
}
```
