# 印刷用語集

> **目的**: パッケージ製造業界で使用される印刷の専門用語を定義
> **更新日**: 2025-04-11
> **バージョン**: 1.0

## 印刷方式基本用語

### 主要印刷方式

| 用語 | 英語表記 | 定義 | 特徴 |
|------|----------|------|------|
| **グラビア印刷** | Gravure Printing | 凹版を使用する印刷方式 | 大量生産に向き、高品質 |
| **凸版印刷** | Letterpress / Relief Printing | 凸部にインキを付着させる印刷方式 | 版の耐久性が高い |
| **オフセット印刷** | Offset Printing | 平版を利用し、ゴムブランケットを経由して印刷 | 多色刷りに適している |
| **フレキソ印刷** | Flexographic Printing | フレキシブルな版を使用する印刷方式 | フィルム印刷に適している |
| **デジタル印刷** | Digital Printing | 版を使用せずに直接印刷 | 小ロット対応、短納期 |

### グラビア印刷詳細

| 用語 | 英語表記 | 定義 | 用例 |
|------|----------|------|------|
| **凹版** | Intaglio Plate / Gravure Cylinder | 凹みにインキが入る版の構造 | グラビア印刷で使用 |
| **シリンダー** | Cylinder | グラビア印刷で使用される円筒状の版 | 版の交換が必要 |
| **セル** | Cell | グラビア版のインキ保持用の凹み | セルの深さで濃淡を表現 |
| **ドクター** | Doctor Blade | 余分なインキをかき取るブレード | インキ層の均一化 |
| **エングレービング** | Engraving | 版の彫刻作業 | 電彫やレーザー彫刻 |

## インキ・色管理用語

### インキ種類

| 用語 | 英語表記 | 定義 | 特徴 |
|------|----------|------|------|
| **溶剤型インキ** | Solvent-based Ink | 溶剤を含むインキ | フィルム印刷に多用 |
| **UVインキ** | UV Curable Ink | 紫外線で硬化するインキ | 即乾性、環境配慮 |
| **水性インキ** | Water-based Ink | 水を溶剤とするインキ | 環境負荷が低い |
| **EBインキ** | Electron Beam Curable Ink | 電子線で硬化するインキ | 高硬度、耐擦性 |

### 色管理

| 用語 | 英語表記 | 定義 | 用例 |
|------|----------|------|------|
| **特色** | Spot Color | 混合インキで再現する色 | ブランドカラーの再現 |
| **プロセスカラー** | Process Color | CMYKの4色を組み合わせて再現する色 | 一般的なフルカラー |
| **CMYK** | Cyan Magenta Yellow Key | プロセスカラーの4色 | シアン、マゼンタ、イエロー、キー(黒) |
| **DIC色** | DIC Color | DICカラーの特色 | 日本の標準色見本 |
| **PANTONE色** | PANTONE Color | PANTONEの標準色 | 国際的な色標準 |

### 色評価

| 用語 | 英語表記 | 定義 | 用例 |
|------|----------|------|------|
| **色差** | Color Difference | 色の差の数値化 | ΔE値による評価 |
| **分光測色計** | Spectrophotometer | 色を数値化する装置 | 精密な色測定 |
| **色見本** | Color Swatch / Color Guide | 色の見本帳 | カラーマッチングの基準 |
| **光源** | Light Source | 色評価を行う照明 | D65光源など |
| **演色評価数** | Color Rendering Index | 光源の演色性評価 | Ra値で表現 |

## 印刷品質管理用語

### 印刷品質評価

| 用語 | 英語表記 | 定義 | 規格・基準 |
|------|----------|------|-----------|
| **濃度測定** | Density Measurement | 印刷濃度を測定すること | 濃度計による評価 |
| **色濃度** | Color Density | 色の濃さ | Dmax値で評価 |
| **網点再現性** | Dot Reproduction | 網点の再現性 | ハイライト、シャドウ |

### 印刷不良

| 用語 | 英語表記 | 定義 | 原因 |
|------|----------|------|------|
| **色ズレ** | Color Misregistration | 色の位置がずれる現象 | 版の位置ズレ |
| **刷りムラ** | Printing Unevenness | 印刷の濃淡ムラ | インキ量、圧の不均一 |
| **ゴースト** | Ghosting | 薄い影が現れる現象 | インキ転写不良 |
| **裏移り** | Set-off | インキが裏面に移る現象 | 乾燥不足 |
| **網点詰まり** | Filling-in | 網点が埋まる現象 | インキ過多、ドクター不良 |
