# 包装用語集

> **目的**: パッケージ製造業界で使用される包装の専門用語を定義
> **更新日**: 2025-04-11
> **バージョン**: 1.0

---

## 包装基本概念

### 包装の定義

| 用語 | 英語表記 | 定義 | 用例 |
|------|----------|------|------|
| **包装** | Packaging | 商品を保護し、輸送・保管・販売を容易にする処置 | 食品包装、工業包装 |
| **個装** | Primary Package / Consumer Unit | 個々の商品を包装した最小単位 | チョコレートの個包装 |
| **内装** | Secondary Package | 個装をまとめた包装 | 箱詰めされた商品 |
| **外装** | Tertiary Package / Transport Package | 輸送・保管を目的とした包装 | 段ボール箱 |
| **商業包装** | Commercial Packaging | 商品価値を高める販売用包装 | 店頭陳列用包装 |
| **工業包装** | Industrial Packaging | 輸送・保管に重点を置いた包装 | 段ボール、パレット |

## 包装機能用語

### 包装の基本的機能

| 用語 | 英語表記 | 定義 | 用例 |
|------|----------|------|------|
| **保護機能** | Protection Function | 商品を外部要因から守る機能 | 酸素、湿気、光からの保護 |
| **保存機能** | Preservation Function | 品質を維持する機能 | 賞味期限の延長 |
| **販売機能** | Sales Function | 商品の魅力を高める機能 | デザイン、ブランド表示 |
| **使いやすさ機能** | Convenience Function | 使用利便性を高める機能 | ジッパー、カットエッジ |
| **情報提供機能** | Information Function | 商品情報を伝える機能 | 表示ラベル、QRコード |
| **環境配慮機能** | Environmental Function | 環境負荷を低減する機能 | 簡易包装、リサイクル性 |

## 包装材料用語

### 基本材料

| 用語 | 英語表記 | 定義 | 特徴 |
|------|----------|------|------|
| **フィルム** | Film | 薄いシート状の素材 | 柔軟性、透明性 |

## 包装方法用語

### 充填方法

| 用語 | 英語表記 | 定義 | 用例 |
|------|----------|------|------|
| **充填** | Filling | 容器に内容物を詰める作業 | 液体充填、粉末充填 |
| **自動充填** | Automatic Filling | 自動機で充填する方法 | 高速充填ライン |
| **手詰め** | Manual Filling | 手作業で充填する方法 | 小ロット生産 |
| **計量充填** | Gravimetric Filling | 重量を計測しながら充填 | 精密な充填量管理 |
| **容積充填** | Volumetric Filling | 容量で充填量を管理 | コストパフォーマンス重視 |

### 密封方法

| 用語 | 英語表記 | 定義 | 用例 |
|------|----------|------|------|
| **シール** | Seal | 密封すること | ヒートシール、誘導シール |
| **ヒートシール** | Heat Seal | 熱で溶着して密封する方法 | パウチの封緘 |
| **誘導シール** | Induction Seal | 電磁誘導で密封する方法 | ボトルのキャップシール |
| **圧着** | Crimping | 圧力で密着させる密封方法 | チューブの密封 |
| **キャッピング** | Capping | キャップで密封する方法 | ボトルの密閉 |
| **栓** | Plug / Stopper | 栓で密閉する方法 | ワインボトル |

## 包装規制用語
