# 診断チャート作成ガイド
# 진단 차트 작성 가이드

> **対象**: ライター、編集者、デザイナー
> **目的**: 記事内で読者が自己診断できるチャートを作成する
> **適用**: 製品紹介記事、選択ガイド記事、技術解説記事
> **言語**: 日本語・韓国語共通

---

## 概要

**診断チャート**とは、読者が記事の内容を自分の状況に当てはめて判断できるよう、記事内に配置される**質問応答形式の意思決定ツール**です。Brixaの「【診断チャートあり】バリア性は何を基準に選べばいいの？」のような記事で効果的に活用されています。

---

## Brixaの事例分析

### 診断チャート付き記事の例

**バリア性選択ガイド**（Brixa）:
```
【診断チャートあり】バリア性は何を基準に選べばいいの？
```

**チャート構造**:
1. 保存期間が3ヶ月以上か？
   - はい → AL/VMPET（湿気に強い）
   - いいえ → LLDPE/PETで十分

2. 内容物の弱点は何か？
   - 湿気 → アルミ箔
   - 光 → アルミ箔またはVMPET
   - 香り → アルミ蒸着または透明蒸着

**特徴**:
- **簡易版**として明記
- 記事本文で詳細説明
- 具体的な数値基準（保存期間）
- イエス/ノーの2分岐

## 診断チャートの価値

### 読者への価値

1. **即座の自己診断**: 記事全文を読まなくても判断可能
2. **理解の促進**: 自分の状況に当てはめながら学習
3. **アクション可能性**: 具体的な次のステップが明確
4. **信頼性向上**: 専門性と客観性の演出

### 記事への価値

1. **CTR向上**: 「診断チャートあり」タグで関心喚起
2. **読了率向上**: 簡易版→詳細版の導線設計
3. **シェア励起**: 実用的なツールとして共有されやすい
4. **SEO効果**: Googleの「方法論」フィーチャードスニペット対応

## 作成ステップ

### ステップ1: 意思決定ポイントの識別

**目的**: 読者が決定する必要のある重要な判断基準を特定

**例**: バリア性選択の場合
1. 保存期間はどのくらいか？
2. 何から守る必要があるか？（湿気・光・酸素・香り）
3. 見た目はどうするか？（透明/不透明）

**ポイント**:
- 意思決定ポイントは3-5個に絞る
- 各ポイントはイエス/ノーで答えられる形式
- 重要度順に並べる

---

### ステップ2: 2分岐ツリーの構築

**基本構造**:
```
質問1: [最も重要な判断基準]
├── はい → [推奨素材A]
└── いいえ → 質問2へ

質問2: [2番目に重要な判断基準]
├── はい → [推奨素材B]
└── いいえ → [推奨素材C]
```

**バリア性選択の例**:
```
保存期間は3ヶ月以上か？
├── はい
│   湿気に弱い？
│   ├── はい → アルミ箔
│   └── いいえ → VMPET
└── いいえ
    透明にしたい？
    ├── はい → 透明蒸着PET
    └── いいえ → LLDPE/PET
```

---

### ステップ3: 視覚的要素の追加

**推奨される視覚要素**:

1. **絵文字/アイコン**
   - ✅ / ❌
   - 👉 / ⬇️
   - 💡 / ⚠️

2. **色分け**
   - 推奨: 👉 青または緑
   - 注意: ⚠️ 黄色またはオレンジ
   - 禁止: ❌ 赤

3. **強調**
   - **太字**で重要なポイント
   - `コードブロック`で技術的な内容

## 記事タイプ別の診断チャート

### 製品紹介記事

**例**: スタンドパウチの選択

```
内容量は500g以上か？
├── はい
│   PE60以上を選択
│   スタンド型推奨
└── いいえ
    平袋でも十分
    PE40で対応可能
```

---

### 技術解説記事

**例**: 印刷方式の選択

```
ロット数は1,000枚以下か？
├── はい
│   デジタル印刷推奨
│   版代がかからない
└── いいえ
    色数は5色以上か？
    ├── はい → グラビア印刷
    └── いいえ → オフセット印刷
```

---

### 選択ガイド記事

**例**: 素材の選択

```
保存期間は1年以上か？
├── はい
│   高バリア必要
│   👉 アルミ箔を選択
└── いいえ
    透明にしたいか？
    ├── はい → 透明蒸着PET
    └── いいえ → アルミ蒸着
```

## タイトル戦略

### 診断チャートありタグの使用

**パターン**:
```
【診断チャートあり】{質問型タイトル}
```

**例**:
- 【診断チャートあり】バリア性は何を基準に選べばいいの？
- 【診断チャートあり】白板は追加費用がかかりますか？
- 【診断チャートあり】どの印刷方式が最適ですか？

## 品質チェックリスト

### コンテンツ

- [ ] 意思決定ポイントが明確である
- [ ] 分岐が論理的である
- [ ] 推奨事項が具体的である
- [ ] 全体の流れが追える
- [ ] 簡易版であることが明記されている

### 視覚的要素

- [ ] 絵文字/アイコンが適切に使われている
- [ ] 階層構造が視覚的にわかりやすい
- [ ] 強調ポイントが明確である
- [ ] 色分けが効果的である（該当する場合）

### 実用性

- [ ] 読者が自分の状況に当てはめられる
- [ ] 結論にアクション可能である
- [ ] 詳細版への導線が設けられている
- [ ] 誤解を招む表現がない

## トラブルシューティング

### 問題1: 分岐が複雑すぎる

**解決策**:
- 最大3階層にする
- 各階層は2択にする
- 複雑な詳細は記事本文に譲る

### 問題2: 診断精度が低い

**解決策**:
- 「簡易版」であることを明記
- 詳細版への誘導を強化
- 注意点を明示する

### 問題3: 視覚的にわかりにくい

**解決策**:
- 絵文字を活用する
- インデントを明確にする
- 階層ごとにセクションを分ける
