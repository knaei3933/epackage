# nanobanana MCP Image Generation Guide
# nanobanana MCP 画像生成ガイド

> **対象**: ライター、開発者
> **目的**: AI画像生成ツールの使用方法
> **適用**: 全言語共通

---

## 概要

nanobanana MCP は、AIによる画像生成を行うツールです。ブログ記事の画像を簡単に作成できます。

---

## 基本的な使用方法

### 画像生成

```bash
mcp__nanobanana__nanobanana_generate \
  prompt="画像の説明" \
  aspect_ratio="16:9" \
  quality="high"
```

### パラメータ

| パラメータ | 説明 | 値 |
|-----------|------|-----|
| `prompt` | 画像の説明（日本語または韓国語） | 文字列 |
| `aspect_ratio` | 画像の比率 | 16:9, 4:3, 1:1, 9:16 |
| `quality` | 画質 | standard, high |
| `output_format` | 出力形式 | png, jpg |
| `candidate_count` | 生成数 | 1〜4 |

## プロンプト作成のベストプラクティス

### 良いプロンプトの条件

1. **具体的である**: 「パッケージ」→「スタンドパウチ、底部がW字型」
2. **雰囲気を指定**: 「プロフェッショナル」「クリエイティブ」「モダン」
3. **スタイルを指定**: 「製品写真風」「イラスト」「インフォグラフィック」
4. **背景を指定**: 「白背景」「淡いグレー」「透明」

### プロンプトの構成

```
[主な被写体] + [詳細な特徴] + [雰囲気/スタイル] + [背景]
```

**例**:
```
スタンドパウチの製品イメージ + 底部がW字型に折りたたまれ、自立する様子 + 清潔でプロフェッショナルな製品写真風 + 背景は白
```

## 言語対応

### 日本語

プロンプトは日本語で記述できます。

```bash
prompt="スタンドパウチの製品イメージ。底部がW字型に折りたたまれ、自立する様子。"
```

### 韓国語

プロンプトは韓国語で記述できます。

```bash
prompt="스탠드파우치 제품 이미지. 바닥이 W자형으로 접혀 있고 서 있는 모습."
```
