# 製造工程

> **目的**: パッケージ製造業界の製造工程を把握するための知識ベース
> **更新日**: 2026-04-11
> **バージョン**: 1.0
> **更新頻度**: 半年次

---

## 製造工程全体像

### パッケージ製造の基本フロー

```
受注 → 仕様確定 → 原材料調達 → 印刷 → ラミネート → スリット → 製袋加工 → 品質検査 → 納品
```

### 主要工程の概要

| 工程 | 内容 | 所要時間 | 品質管理ポイント |
|------|------|----------|------------------|
| **仕様確定** | 規格書作成、デザイン確認 | 1-3日 | 顧客要件の正確な反映 |
| **原材料調達** | フィルム、インク購入 | 3-7日 | 素材検査、在庫管理 |
| **印刷** | グラビア/デジタル印刷 | 1-14日 | 色管理、登録精度 |
| **ラミネート** | 素材接着、バリア性付与 | 1-2日 | 接着強度、ピンホール検査 |
| **スリット** | ロール幅切り出し | 1日 | 幅精度、端面処理 |
| **製袋加工** | パウチ成形、チャック付け | 1-5日 | 寸法精度、チャック機能 |
| **品質検査** | 外観、強度、バリア性検査 | 1日 | 不良品選別、データ記録 |
| **納品** | 包装、出荷 | 1日 | 数量確認、配送管理 |

## 詳細工程解説

### 1. 仕様確定工程

#### 仕様検討項目

| 項目 | 確認内容 | 影響 |
|------|----------|------|
| **製品仕様** | パウチタイプ、寸法、容量 | 製袋工程、機械選定 |
| **素材構成** | 素材種類、構成、厚み | コスト、バリア性 |
| **印刷仕様** | 色数、版数、印刷方式 | 納期、コスト |
| **機能要件** | チャック、窓、デガスバルブ等 | 加工程序、工数 |
| **品質基準** | 強度、バリア性、外観 | 品質検査基準 |
| **納期・数量** | 納期、ロット数、継続性 | 生産計画、在庫 |

#### 仕様確定の重要性

- **トラブル予防**: 事前確認による製造トラブル防止
- **コスト最適化**: 適切な素材・工程選定
- **納期遵守**: 段取り時間の短縮
- **品質保証**: 明確な品質基準設定

---

### 2. 印刷工程

#### グラビア印刷プロセス

```
製版（ロスケ作成） → インク調色 → 印刷開始 → 色合わせ → 本印刷 → 乾燥 → 巻き取り
```

| 工程詳細 | ポイント |
|----------|----------|
| **製版** | 銅ロールにエッチングでセル彫刻、精度±5μm |
| **インク調色** | 色見本との照合、分光測色計使用 |
| **色合わせ** | 見本との色差ΔE<2.0を目標 |
| **本印刷** | 速度50-200m/分、張力管理 |
| **乾燥** | 熱風乾燥、溶剤残留0.5mg/m²以下 |
| **巻き取り** | 巻き皺防止、張力制御 |

#### デジタル印刷プロセス

```
データ受領 → 前処理 → 色変換 → 印刷 → 乾燥 → 巻き取り
```

| 工程詳細 | ポイント |
|----------|----------|
| **データ受領** | PDF、AI形式対応、CMYK指定 |
| **前処理** | 色プロファイル適用、RIP処理 |
| **色変換** | ICCプロファイル使用 |
| **印刷** | インクジェット方式、1200dpi |
| **乾燥** | UVまたは熱風乾燥 |
| **巻き取り** | 張力制御、巻き精度管理 |

#### 印刷工程の品質管理

| 検査項目 | 評価方法 | 合否基準 |
|----------|----------|----------|
| **色再現性** | 分光測色計 | ΔE<2.0 |
| **登録精度** | 顕微鏡測定 | ±0.2mm以内 |
| **印刷濃度** | 濃度計 | 指定値±10% |
| **ピンホール** | 光透過検査 | 3個/m²以下 |
| **キズ・汚れ** | 目視検査 | 視覚不良なし |
| **溶剤残留** | GC-MS分析 | 0.5mg/m²以下 |

### 3. ラミネート工程

#### ラミネート方式

| 方式 | 接着剤 | 対象素材 | 特徴 |
|------|--------|----------|------|
| **ドライラミネート** | 溶剤系接着剤 | PET/AL、PET/PA | 高接着強度 |
| **ウェットラミネート** | 水系接着剤 | 紙/フィルム | 環境配慮 |
| **エキストルージョン** | ポリエチレン溶融 | フィルム/PE | コスト安 |
| **無溶剂ラミネート** | 100%固形分接着剤 | 全般 | 環境配慮、高速化 |

#### ドライラミネートプロセス

```
接着剤塗布 → 乾燥 → 圧着 → エージング → 巻き取り
```

| 工程詳細 | ポイント |
|----------|----------|
| **接着剤塗布** | グラビアコーター、塗布量3-5g/m² |
| **乾燥** | 3-4段階乾燥、溶剤飛ばし |
| **圧着** | ラミネートロール、温度80-100℃ |
| **エージング** | 40℃、24-48時間、接着強度向上 |
| **巻き取り** | 張力制御、巻き皺防止 |

#### ラミネート品質管理

| 検査項目 | 評価方法 | 合否基準 |
|----------|----------|----------|
| **接着強度** | 180°ピール試験 | 300g/15mm以上 |
| **ピンホール** | 光透過検査 | 5個/m²以下 |
| **密着性** | テープ剥離試験 | 剥離なし |
| **外観** | 目視検査 | 異常なし |
| **溶剤残留** | GC-MS分析 | 2mg/m²以下 |

---

### 4. スリット工程

#### スリットプロセス

```
巻出し → スリット（裁断） → 静電除去 → 巻き取り
```

| 工程詳細 | ポイント |
|----------|----------|
| **巻出し** | 張力制御、一定速度供給 |
| **スリット** | ラザーカッター、シャッパーカッター |
| **静電除去** | イオナイザー、帯電防止 |
| **巻き取り** | 幅精度±0.5mm、巻き姿勢管理 |

#### スリット品質管理

| 検査項目 | 評価方法 | 合否基準 |
|----------|----------|----------|
| **幅精度** | 定規測定 | ±0.5mm以内 |
| **端面状態** | 目視検査 | バリなし、平滑 |
| **巻き姿勢** | テーパ測定 | 5mm以内 |
| **長さ** | 長さ計 | ±1%以内 |

### 5. 製袋加工工程

#### 製袋方式

| 方式 | 対象製品 | 特徴 |
|------|----------|------|
| **三方シール** | 平袋、ガゼットパウチ | シンプル、高速 |
| **ピロー** | 枕型包装 | 自動充填包装 |
| **合掌** | 合掌袋 | 自立性なし |
| **スタンドパウチ** | 自立袋 | 底部W字加工 |
| **スパウト付け** | スパウトパウチ | 注ぎ口機能 |
| **ジッパー付け** | 再封可能包装 | 開閉機能 |

#### スタンドパウチ製袋プロセス

```
巻出し → チャック取り付け → 底部形成 → トップシール → サイドシール → クリアランス切断 → 検査 → カウント
```

| 工程詳細 | ポイント |
|----------|----------|
| **巻出し** | フィルム送り、位置決め |
| **チャック取り付け** | チャック溶着、位置精度±1mm |
| **底部形成** | 底部折りたたみ、D字型またはW字型 |
| **トップシール** | ヒートシール、温度140-160℃ |
| **サイドシール** | 左右シール、温度140-160℃ |
| **クリアランス切断** | 余分なフィルム除去 |
| **検査** | 外観、チャック機能、寸法検査 |
| **カウント** | 規定数まとめ、箱詰め準備 |

#### スパウトパウチ製袋プロセス

```
巻出し → スパウト穴開け → スパウト溶着 → チャック取り付け → 底部形成 → シール加工 → 検査 → カウント
```

| 工程詳細 | ポイント |
|----------|----------|
| **巻出し** | フィルム送り、位置決め |
| **スパウト穴開け** | パンチまたはレーザー穴開け |
| **スパウト溶着** | 高周波溶着、気密性確認 |
| **チャック取り付け** | チャック溶着 |
| **底部形成** | 底部折りたたみ |
| **シール加工** | トップ・サイドシール |
| **検査** | スパウト密着性、外観、寸法 |
| **カウント** | 規定数まとめ |

#### 製袋品質管理

| 検査項目 | 評価方法 | 合否基準 |
|----------|----------|----------|
| **寸法精度** | 定規測定 | ±1mm以内 |
| **シール強度** | 引張試験機 | 15N/15mm以上 |
| **チャック機能** | 開閉テスト | スムーズ、確実 |
| **スパウト密着性** | 水圧テスト | 漏れなし |
| **外観** | 目視検査 | キズ、汚れ、異物なし |
| **自立性** | スタンドテスト | 安定自立 |

## 自動包装工程

### 充填・シール工程

#### 充填方式

| 方式 | 対象内容物 | 特徴 |
|------|-----------|------|
| **体積充填** | 顆粒、粉末 | コスト安、高速 |
| **重量充填** | 顆粒、粉末 | 高精度 |
| **液体充填** | 液体、ペースト | ポンプ式 |
| **固形物充填** | 固形物 | 手作業または自動 |

#### 自動包装フロー

```
パウチ供給 → 開袋 → 充填 → 脱気 → シール → 冷却 → 検査 → 排出
```

| 工程詳細 | ポイント |
|----------|----------|
| **パウチ供給** | ジッパー開放、チャック保持 |
| **開袋** | エアノズル、パウチ開口 |
| **充填** | ノズル挿入、内容物充填 |
| **脱気** | 窒素ガス置換、酸素濃度低下 |
| **シール** | ヒートシール、チャック閉鎖 |
| **冷却** | シール部冷却、強度確保 |
| **検査** | 金属探知、X線検査、重量検査 |
| **排出** | 規定数まとめ、箱詰め |

## 品質管理工程

### 品質検査項目

#### 受入検査（原材料）

| 検査項目 | 評価方法 | 頻度 |
|----------|----------|------|
| **外観** | 目視検査 | 全数 |
| **寸法** | 定規、マイクロメーター | ロット毎 |
| **厚み** | 厚み計 | ロット毎 |
| **引張強度** | 引張試験機 | ロット毎 |
| **バリア性** | 酸素透過度、水蒸気透過度 | ロット毎 |

#### 工程内検査

| 工程 | 検査項目 | 頻度 |
|------|----------|------|
| **印刷** | 色、登録、キズ | 1時間毎 |
| **ラミネート** | 接着強度、ピンホール | 2時間毎 |
| **スリット** | 幅、端面、長さ | ロット毎 |
| **製袋** | 寸法、シール、チャック | 30分毎 |
| **充填** | 重量、金属探知、シール | 連続 |

#### 製品検査

| 検査項目 | 評価方法 | AQL |
|----------|----------|-----|
| **外観** | 目視検査 | 2.5 |
| **寸法** | 定規測定 | 1.5 |
| **シール強度** | 引張試験機 | 1.5 |
| **チャック機能** | 開閉テスト | 1.5 |
| **バリア性** | 酸素透過度測定 | 0.65 |

## トラブルシューティング

### よくある製造トラブル

| トラブル | 原因 | 対策 |
|----------|------|------|
| **色ズレ** | 登録ずれ、張力変動 | 張力管理、ガイド調整 |
| **シール不良** | 温度不足、時間不足、異物混入 | 温度・時間調整、清掃 |
| **ピンホール** | 異物、フィルム欠陥 | フィルム検査、清掃 |
| **チャック不良** | 溶着不足、位置ずれ | 温度・圧力調整、位置精度管理 |
| **巻き皺** | 張力不均一、湿度変動 | 張力管理、湿度管理 |
| **印刷ムラ** | インク粘度、ロール偏心 | インク管理、ロールメンテナンス |
