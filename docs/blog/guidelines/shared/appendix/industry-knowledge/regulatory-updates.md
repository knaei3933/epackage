## 日本

### 食品表示法

#### 2024-2025年更新事項

| 項目 | 内容 | 施行日 |
|------|------|--------|
| **アレルゲン表示** | 準特定原材料20品目の表示義務化 | 2023年4月 |
| **栄養成分表示** | ナトリウム→食塩相当量表示移行完了 | 2024年4月 |
| **遺伝子組換え表示** | 「分別」表示の厳格化 | 2024年4月 |

#### 表示必須項目

| 項目 | 詳細 |
|------|------|
| **名称** | 食品の名称を省略なく記載 |
| **原材料名** | 重量の多い順に記載（添加物は最後） |
| **添加物** | 物質名・用途名併記、一部簡便名使用可 |
| **アレルゲン** | 特定27品目（準特定含む）を表示 |
| **栄養成分** | 熱量、たん白質、脂質、炭水化物、食塩相当量 |
| **保存方法** | 必要に応じて記載 |
| **消費期限/賞味期限** | 原則として年月日まで記載 |
| **製造者情報** | 製造者/販売者の名称と住所 |

#### 包装に関するポイント

| 項目 | 内容 |
|------|------|
| **警告表示** | 開封方法、加熱可否、注意事項を明記 |

## 韓国

### 食品等の表示基準

#### 2024-2025年更新事項

| 項目 | 内容 | 施行日 |
|------|------|--------|
| **栄養成分表示** | カフェイン、トランス脂肪酸表示義務化 | 2024年1月 |
| **遺伝子組換え表示** | GMO含有率0.9%超で表示義務 | 2024年1月 |
| **アレルゲン表示** | 卵、牛乳、大豆等20品目表示義務 | 2023年 |

#### 表示必須項目

| 項目 | 詳細 |
|------|------|
| **品名** | 食品の品名 |
| **種類** | 食品の種類 |
| **原材料名及び原材料名** | 重量の多い順に記載 |
| **栄養成分** | 熱量、炭水化物、タンパク質、脂質、ナトリウム |
| **アレルゲン** | 卵、牛乳、大豆等表示 |
| **遺伝子組換え** | GMO含有率表示 |
| **消費期限/流通期限** | 製造日から設定 |

## 欧州

### PPWR（包装・包装廃棄物規則）

#### 主要内容

| 項目 | 内容 | 施行予定 |
|------|------|----------|
| **再利用・リサイクル目標** | 2030年までに65%リサイクル | 2025-2030年段階的 |
| **単一素材化** | 特定包装は単一素材を義務化 | 2030年まで |
| **プラスチック使用削減** | 2030年までに10%削減 | 2025-2030年段階的 |
| **再充填可能包装** | 飲料・テイクアウト食品で義務化 | 2030年まで |
