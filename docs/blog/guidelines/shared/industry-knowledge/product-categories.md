## 製品カテゴリー概要

### カテゴリー体系

Epackage Labでは、柔軟包装袋（フィルムパッケージ）を以下の5つのカテゴリーで展開しています。

```
柔軟包装袋
│
├─ 1. スタンドパウチ（自立袋）
├─ 2. ガゼットパウチ（BOX型）
├─ 3. 平袋（三シール袋）
├─ 4. チューブ袋
└─ 5. ラミネートチューブ
```

### カテゴリー別市場シェア

| カテゴリー | 市場シェア | Epackage Labの強み |
|-----------|-----------|-------------------|
| **スタンドパウチ** | 35% | 小ロット対応、価格競争力 |
| **ガゼットパウチ** | 20% | 大容量対応、カスタマイズ |
| **平袋** | 25% | 低価格、短納期 |
| **チューブ袋** | 15% | 特注対応 |
| **ラミネートチューブ** | 5% | 高級感、機能性 |

## 主要製品ラインナップ

### 1. スタンドパウチ（自立袋）

#### 基本情報

| 項目 | 内容 |
|------|------|
| **別名** | 自立袋、底ガゼット |
| **特徴** | 底部がW字型に折りたたまれ、内容物を入れると自立 |
| **主な用途** | 菓子、健康食品、コーヒー豆、粉末飲料 |
| **サイズ範囲** | 幅80〜250mm × 高さ120〜350mm |
| **最低ロット** | 500枚 |

#### 構造

```
     ┌─────────────┐
     │   チャック   │  ← 開閉可能（オプション）
     ├─────────────┤
     │             │
     │   前面      │  ← ブランドロゴ、デザイン
     │             │
     ├─────────────┤
     │   /   \     │
     │  /     \    │  ← W字型底部（自立構造）
     │ /       \   │
     └───────────┘
```

#### 特徴とメリット

**メリット**:
- **陳列効果**: 自立するため、商品の顔を常に見せられる
- **省スペース**: 空の時はコンパクトに収納可能
- **デザイン性**: 立体的なデザインが可能
- **バリエーション**: チャック、ハングホール等のオプション豊富

**デメリット**:
- 平袋に比べてコストが高い
- 底部の製袋工程が必要

#### 主なサイズラインナップ

| 型番 | サイズ（mm） | 容量目安 | 主な用途 |
|------|-------------|---------|----------|
| **SP-S** | 120×180 | 100〜200g | 詰め合わせ、小分け |
| **SP-M** | 150×230 | 200〜400g | 一般的な菓子 |
| **SP-L** | 180×280 | 400〜700g | 大容量菓子 |
| **SP-LL** | 220×320 | 700g〜1kg | 業務用、コーヒー豆 |

#### オプション

| オプション | 説明 | 追加費用 |
|-----------|------|---------|
| **チャック** | 反復使用可能なジッパー | +¥3〜5/枚 |
| **ハングホール** | 吊り下げ用穴 | +¥1〜2/枚 |
| **エアバルブ** | ガス抜き穴（コーヒー用） | +¥2〜3/枚 |
| **ウインドウ** | 中身が見える透明窓 | +¥2〜4/枚 |

### 2. ガゼットパウチ（BOX型）

#### 基本情報

| 項目 | 内容 |
|------|------|
| **別名** | 箱型パウチ、四方ガゼット |
| **特徴** | 底部と側面がガゼット構造で、箱型に自立 |
| **主な用途** | 大容量菓子、惣菜、ペットフード |
| **サイズ範囲** | 幅150〜300mm × 高さ200〜400mm |
| **最低ロット** | 500枚 |

#### 構造

```
     ┌─────────────┐
     │   チャック   │  ← 開閉可能（オプション）
     ├─────────────┤
     │／          ＼│  ← 側面ガゼット
     │   前面     │  ← ブランドロゴ、デザイン
     │            │
     │  ＼      ／│  ← 側面ガゼット
     ├─────────────┤
     │  ／    ＼  │  ← 底部ガゼット
     │／        ＼│
     └────────────┘
```

#### 特徴とメリット

**メリット**:
- **大容量対応**: 1kg以上の大容量に対応
- **安定性**: 四方にガゼットがあり、非常に安定
- **陳列効果**: 箱型で高級感がある
- **デザイン面**: 前面が広く、デザインの自由度が高い

**デメリット**:
- コストが高い（最も高価）
- 製袋工程が複雑

#### 主なサイズラインナップ

| 型番 | サイズ（mm） | 容量目安 | 主な用途 |
|------|-------------|---------|----------|
| **GP-M** | 180×250×80 | 500〜800g | 一般的な菓子 |
| **GP-L** | 220×300×100 | 800g〜1.5kg | 大容量菓子 |
| **GP-LL** | 250×350×120 | 1.5kg〜3kg | 業務用、ペットフード |

### 3. 平袋（三シール袋）

#### 基本情報

| 項目 | 内容 |
|------|------|
| **別名** | ピローフィルム、三シール袋 |
| **特徴** | 両側と底部の3方向をシールしたシンプルな袋 |
| **主な用途** | 菓子、パン、冷凍食品、衣料品 |
| **サイズ範囲** | 幅70〜200mm × 高さ100〜300mm |
| **最低ロット** | 500枚 |

#### 構造

```
     ┌─────────────┐
     │             │
     │   前面      │  ← ブランドロゴ、デザイン
     │             │
     ├─────────────┤
     │シール│シール│  ← 両側シール
     ├─────────────┤
     │   底部シール │  ← 底部シール
     └─────────────┘
```

#### 特徴とメリット

**メリット**:
- **低価格**: 最もコストパフォーマンスが高い
- **シンプル**: シンプルな構造で使いやすい
- **短納期**: 製造工程がシンプルで短納期対応可能
- **汎用性**: 幅広い用途に対応

**デメリット**:
- 自立しない（陳列効果が低い）
- デザインの制限がある（平面的）

#### 主なサイズラインナップ

| 型番 | サイズ（mm） | 容量目安 | 主な用途 |
|------|-------------|---------|----------|
| **FP-S** | 100×150 | 50〜100g | 小分け、サンプル |
| **FP-M** | 120×180 | 100〜200g | 一般的な菓子 |
| **FP-L** | 150×230 | 200〜400g | 大容量菓子 |

### 4. チューブ袋

#### 基本情報

| 項目 | 内容 |
|------|------|
| **別名** - 枕状袋 |
| **特徴** | 両側をシールし、底部はガゼット構造 |
| **主な用途** | パン、冷冻食品、衣料品 |
| **サイズ範囲** | 直径70〜200mm × 長さ150〜400mm |
| **最低ロット** | 1,000枚 |

#### 構造

```
      (  ) (  )
       │   │
       │   │
       │   │
      ┌───────┐
      │       │
      │ 前面  │  ← ブランドロゴ、デザイン
      │       │
      └───────┘
       │   │
      シール シール  ← 両側シール
```

#### 特徴とメリット

**メリット**:
- **円筒形状**: 独特のデザイン性
- **適度な容量**: 中容量に適している
- **自動包装対応**: 自動包装機での使用に対応

**デメリット**:
- 自立しない
- 設計の制限がある

---

### 5. ラミネートチューブ

#### 基本情報

| 項目 | 内容 |
|------|------|
| **別名** - ラミネートチューブ |
| **特徴** | 円筒状のボディにキャップを付けた容器 |
| **主な用途** | 化粧品、歯磨き粉、食品、医薬品 |
| **サイズ範囲** | 直径16〜50mm × 内容量30〜500ml |
| **最低ロット** | 3,000枚 |

#### 構造

```
         ┌──┐
         │キャップ│  ← ねじ式キャップ
         ├──┤
         │  │  ← チューブ口
      ／    ＼
     │      │  ← チューブ本体
     │      │  ← ラミネートフィルム
      ＼    ／
         │  │
         └──┘  ← 底部シール（または溶着）
```

#### 特徴とメリット

**メリット**:
- **高級感**: 高級感のあるデザイン
- **機能性**: 内容物を最後まで絞り出せる
- **密閉性**: キャップにより高い密閉性

**デメリット**:
- コストが高い
- 最低ロットが3,000枚
- 製造工程が複雑

## 選び方ガイド

### 製品選定のステップ

#### STEP1: 用途を明確に

**質問事項**:
- 何を入れるか？（内容物の形状、サイズ）
- どれくらい入れるか？（容量）
- どこで販売するか？（販売チャネル）
- どのような印象を与えたいか？（ブランドイメージ）

#### STEP2: 必要な機能を確認

**機能チェック**:
- [ ] 自立が必要か？
- [ ] 開閉が必要か？（チャック）
- [ ] 吊り下げが必要か？（ハングホール）
- [ ] 中身が見える必要があるか？（ウインドウ）
- [ ] 鮮度保持が必要か？（バリア性）

#### STEP3: サイズを決定

**サイズ選定のポイント**:
```
袋の幅 = 商品幅 + 10〜20mm（余白）
袋の高さ = 商品高さ + 50〜80mm
袋のマチ = 商品奥行 + 10〜20mm
```

#### STEP4: 素材を選定

**素材選びの基本**:
- 内容物の特性（酸化、湿気、光に弱い等）
- 保存期間
- 法規制（食品衛生法等）

#### STEP5: 予算と納期を確認

**予算と納期の目安**:
| 製品 | 価格目安（税抜） | 納期目安 |
|------|----------------|---------|
| 平袋 | ¥8〜15/枚 | 21日 |
| スタンドパウチ | ¥15〜25/枚 | 28日 |
| ガゼットパウチ | ¥20〜35/枚 | 35日 |
| ラミネートチューブ | ¥30〜50/個 | 42日 |

## ブログ記事での活用ポイント

### 製品紹介記事の構成

#### 導入部

```markdown
「商品が棚に埋もれて目立たない...」「パッケージで商品価値を伝えたい」
そんな悩みを抱えていませんか？

パッケージは商品の「顔」であり、店頭での訴求力を左右する重要な要素です。
従来の平袋では、商品が倒れてしまったり、棚の奥に隠れてしまったりして、
せっかくの商品魅力が伝わらないことも少なくありません。

スタンドパウチ（自立袋）は、底部がW字型に折りたたまれた構造で、
内容物を充填すると自立する袋です。陳列効果を最大化し、商品の価値を
最大限に引き出します。
```

#### 比較表の活用

```markdown
| 特徴 | スタンドパウチ | 平袋 |
|------|---------------|------|
| 自立性 | 自立する | 自立しない |
| 陳列効果 | 高い | 低い |
| コスト | やや高い | 安い |
| 用途 | 菓子、コーヒー豆 | パン、冷凍食品 |
```

#### 選び方のポイント

```markdown
## スタンドパウチを選ぶべき場合

以下のようなニーズにおすすめです。

- 商品を自立させて陳列したい
- 店頭での訴求力を高めたい
- 高級感を演出したい
- ブランドイメージを大切にしたい

## 平袋を選ぶべき場合

以下のようなニーズにおすすめです。

- コストを抑えたい
- 大量に安く作りたい
- シンプルなデザインで十分
- 試作品やテスト販売
```
