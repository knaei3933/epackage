## 製造工程概要

### 製造プロセスの全体像

柔軟包装袋（フィルムパッケージ）の製造は、以下の8つの工程で構成されています。

```
┌─────────────────────────────────────────────────────────────┐
│                   製造工程フロー                              │
└─────────────────────────────────────────────────────────────┘

1. 原反準備
   └─ フィルム原反の確認・準備

2. 製版
   └─ 印刷版（版胴）の作成

3. 印刷
   └─ グラビア印刷によるデザイン印刷

4. ラミネート
   └─ 複数のフィルムを貼り合わせ

5. スリッティング
   └─ 目的の幅に切断

6. 製袋
   └─ 袋形状に加工・成型

7. 検査
   └─ 外観検査・品質チェック

8. 包装・納品
   └─ 梱包・出荷
```

### 工程別所要時間

| 工程 | 所要時間 | 備考 |
|------|---------|------|
| **原反準備** | 1〜2日 | 材料手配を含む |
| **製版** | 3〜5日 | 新規の場合 |
| **印刷** | 2〜3日 | 8〜10色の場合 |
| **ラミネート** | 1〜2日 | 素材数による |
| **スリッティング** | 1日 | - |
| **製袋** | 3〜5日 | 袋形状による |
| **検査** | 1〜2日 | - |
| **包装・納品** | 1〜2日 | - |
| **合計** | **13〜22日** | - |

## 各工程の詳細

### 1. 原反準備

#### 目的

製造に必要なフィルム原反を準備し、仕様を確認します。

#### 作業内容

| 項目 | 内容 |
|------|------|
| **素材確認** | 指定された素材（PET、AL、LLDPE等）の確認 |
| **厚み確認** | 各素材の厚み（12μm、7μm、70μm等）の確認 |
| **幅確認** | 原反の幅（例: 1,020mm、1,280mm）の確認 |
| **長さ確認** | 原反の長さ（例: 3,000m、5,000m）の確認 |
| **品質確認** | 傷、汚れ、端れの有無を確認 |

#### 原反の例

```
【原反ラベル】
素材: PET / 厚み: 12μm / 幅: 1,020mm / 長さ: 3,000m
ロット番号: PET240401-001 / 製造年月日: 2024-04-01
```

---

### 2. 製版

#### 目的

印刷版（版胴）を作成します。これは印刷デザインをフィルムに転写するための「型」です。

#### 作業内容

| 項目 | 内容 |
|------|------|
| **デザイン確認** | 顧客から入稿されたデザインデータの確認 |
| **色分解** | デザインをCMYK + 特色色に分解 |
| **版作成** | シリンダー（版胴）にデザインを彫刻 |
| **試し刷り** | 試し刷りで色味を確認 |
| **修正** | 必要に応じて修正 |

#### 製版方式

**グラビア印刷方式**:
- 凹版（オフセットは凸版）
- デザインが凹んでいる
- インクが凹部に溜まる
- 高精細な印刷が可能

#### 色数と版数

| 色数 | 版数 | 備考 |
|------|------|------|
| **4色** | 4版 | CMYKのみ |
| **8色** | 8版 | CMYK + 4特色 |
| **10色** | 10版 | CMYK + 6特色 |

#### 製版期間

| 状況 | 所要期間 |
|------|---------|
| **新規** | 3〜5日 |
| **再版（既存版使用）** | 1〜2日 |

---

### 3. 印刷

#### 目的

準備した版を使用して、フィルムにデザインを印刷します。

#### 作業内容

| 項目 | 内容 |
|------|------|
| **版セット** | 版胴を印刷機にセット |
| **インキ調合** | 各色のインキを調合 |
| **位置合わせ** | 各色の位置を微調整（見当合わせ） |
| **印刷** | グラビア印刷で連続印刷 |
| **乾燥** | インキを乾燥 |
| **巻き取り** | 印刷済みフィルムを巻き取る |

#### 印刷機の構造

```
┌─────────────────────────────────────────────────────────┐
│                    グラビア印刷機                         │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  原反供給 → 印刷ユニット → 乾燥炉 → 巻き取り            │
│  └─(1)     └─(2)      └─(3)    └─(4)                  │
│                                                         │
│  (1) 原反供給部: 未印刷フィルムを供給                    │
│  (2) 印刷ユニット: 8〜10色の印刷ユニットが直列に配置     │
│  (3) 乾燥炉: インキを乾燥                               │
│  (4) 巻き取り部: 印刷済みフィルムを巻き取る              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 印刷ユニットの例

```
┌─────────────────────────────────────────────────────────┐
│                  8色印刷ユニット                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  原反 → C版 → M版 → Y版 → K版 → 特色1 → 特色2 → 特色3 → 特色4 → 巻き取り
│        ↓     ↓     ↓     ↓      ↓      ↓      ↓      ↓      ↓
│       乾燥  乾燥  乾燥  乾燥   乾燥   乾燥   乾燥   乾燥    乾燥
│                                                         │
│  C: Cyan (シアン)                                       │
│  M: Magenta (マゼンタ)                                  │
│  Y: Yellow (イエロー)                                   │
│  K: Key plate (ブラック)                                │
│  特色: 特殊色（金、銀、蛍光色等）                        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 印刷品質チェック

| チェック項目 | 内容 |
|-----------|------|
| **色味** | 指定された色味になっているか |
| **見当合わせ** | 各色の位置が正確に合っているか |
| **濃度** | インキの濃度が適正か |
| **傷・汚れ** | 傷、汚れ、ゴミがないか |

#### 印刷速度

| 項目 | 数値 |
|------|------|
| **印刷速度** | 150〜200m/分 |
| **1ロット（3,000m）** | 約15〜20分 |

---

### 4. ラミネート

#### 目的

複数のフィルムを貼り合わせ、ラミネートフィルムを作成します。

#### 作業内容

| 項目 | 内容 |
|------|------|
| **フィルム準備** | 印刷済みフィルムとバリア材（AL等）を準備 |
| **接着剤塗布** | 接着剤を塗布 |
| **貼り合わせ** | ドライラミネート方式で貼り合わせ |
| **乾燥・硬化** | 接着剤を乾燥・硬化 |
| **巻き取り** | ラミネート済みフィルムを巻き取る |

#### ラミネート方式

**ドライラミネート方式**:
- 溶剤系接着剤を使用
- 塗布 → 乾燥 → 貼り合わせ → 硬化
- 最も一般的な方式

#### ラミネート構成例

```
┌─────────────────────────────────────────────────────────┐
│              ラミネート構成 (PET/AL/LLDPE)                │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  印刷済みPETフィルム → 接着剤 → AL箔 → 接着剤 → LLDPE    │
│                         ↓                               │
│                    ラミネート加工                         │
│                         ↓                               │
│          PET/AL/LLDPE ラミネートフィルム                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### ラミネート品質チェック

| チェック項目 | 内容 |
|-----------|------|
| **密着性** | 各層が密着しているか |
| **傷・シワ** | 傷、シワがないか |
| **端れ** | 端れがないか |
| **異物** | 異物が混入していないか |

---

### 5. スリッティング

#### 目的

ラミネート済みフィルムを、製袋に適した幅に切断します。

#### 作業内容

| 項目 | 内容 |
|------|------|
| **幅設定** | 製袋サイズに合わせて幅を設定 |
| **切断** | スリッターで切断 |
| **巻き取り** | 切断済みフィルムを巻き取る |

#### スリッティング例

```
【原反】
幅: 1,020mm / 長さ: 3,000m

【スリッティング】
↓ 切断

【製品】
幅: 200mm × 5本（計1,000mm）
  + 20mm（端材）
幅: 150mm × 6本（計900mm）
  + 120mm（端材）
```

#### スリッティング品質チェック

| チェック項目 | 内容 |
|-----------|------|
| **幅精度** | 指定された幅になっているか（±1mm） |
| **切断面** | 切断面がきれいか |
| **巻き取り** | 巻き取りが均一か |

---

### 6. 製袋

#### 目的

スリッティング済みフィルムを、袋形状に加工・成型します。

#### 作業内容

| 項目 | 内容 |
|------|------|
| **フィルムセット** | フィルムを製袋機にセット |
| **成型** | 袋形状に成型（三シール、ガゼット等） |
| **ヒートシール** | シール部を熱圧着 |
| **カット** | 指定された長さにカット |
| **スタッカー** | 製品を集積 |

#### 製袋機の種類

**三シール袋製袋機**:
```
フィルム供給 → 底部シール → 両側シール → カット → 製品
```

**スタンドパウチ製袋機**:
```
フィルム供給 → 底部W折り → 底部シール → 両側シール → カット → 製品
```

**ガゼットパウチ製袋機**:
```
フィルム供給 → 底部ガゼット形成 → 側面ガゼット形成 → 各シール → カット → 製品
```

#### 製袋速度

| 製品 | 速度（目安） |
|------|-------------|
| **平袋** | 150〜200袋/分 |
| **スタンドパウチ** | 80〜120袋/分 |
| **ガゼットパウチ** | 60〜100袋/分 |

#### 製袋品質チェック

| チェック項目 | 内容 |
|-----------|------|
| **シール強度** | シール部の強度は十分か |
| **シール幅** | シール幅は均一か |
| **寸法精度** | 寸法精度は許容範囲内か |
| **外観** | 傷、汚れ、異物がないか |

---

### 7. 検査

#### 目的

製品の品質をチェックし、不良品を除去します。

#### 作業内容

| 項目 | 内容 |
|------|------|
| **外観検査** | 目視検査、または自動検査機 |
| **寸法検査** | 寸法精度の確認 |
| **シール検査** | シール強度の確認 |
| **サンプリング検査** | 抜き取り検査（AQL基準） |

#### 検査基準（AQL: Acceptable Quality Level）

| 検査項目 | AQLレベル |
|---------|----------|
| **外観** | 2.5% |
| **寸法** | 1.5% |
| **シール強度** | 1.0% |

#### 不良品の例

| 不良項目 | 内容 |
|---------|------|
| **印刷不良** | 色ズレ、色ムラ、刷り漏れ |
| **ラミネート不良** | 密着不良、シワ、異物 |
| **製袋不良** | シール不良、寸法不良、成型不良 |
| **汚れ・傷** | 油汚れ、傷、手垢 |

---

### 8. 包装・納品

#### 目的

製品を梱包し、顧客へ納品します。

#### 作業内容

| 項目 | 内容 |
|------|------|
| **カートン詰め** | 段ボール箱に製品を詰める |
| **ラベル貼り** | 品番、数量、ロット番号等を記載 |
| **梱包** | 段ボール箱を梱包 |
| **出荷** | 配送業者へ引き渡し |

#### 包装仕様

| 項目 | 仕様 |
|------|------|
| **内装箱** | 段ボール箱 |
| **数量** | 500〜1,000枚/箱（サイズによる） |
| **ラベル** | 品番、数量、ロット番号、製造年月日 |

## 品質管理

### 品質管理体系

Epackage Labでは、以下の品質管理体系を採用しています。

```
┌─────────────────────────────────────────────────────────┐
│                   品質管理フロー                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  原材料検査 → 工程内検査 → 最終検査 → 出荷検査           │
│  └─(1)      └─(2)       └─(3)      └─(4)              │
│                                                         │
│  (1) 原材料検査: 原反の品質確認                          │
│  (2) 工程内検査: 各工程での品質確認                       │
│  (3) 最終検査: 製品の全数検査または抜き取り検査          │
│  (4) 出荷検査: 出荷前の最終確認                          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 品質記録

| 記録項目 | 内容 |
|---------|------|
| **製造指示書** | 仕様、数量、納期 |
| **原反ロット記録** | 使用した原反のロット番号 |
| **印刷記録** | 色味、見当合わせの確認記録 |
| **ラミネート記録** | 密着性、外観の確認記録 |
| **製袋記録** | 寸法、シール強度の確認記録 |
| **検査記録** | 不良率、不良内容の記録 |

## 納期とコスト

### 納期の目安

| 製品 | 最低ロット | 納期目安 | 備考 |
|------|----------|---------|------|
| **平袋** | 500枚 | 21日 | 最短21日 |
| **スタンドパウチ** | 500枚 | 28日 | 平均28日 |
| **ガゼットパウチ** | 500枚 | 35日 | 製袋工程が複雑 |
| **チューブ袋** | 1,000枚 | 28日 | - |
| **ラミネートチューブ** | 3,000枚 | 42日 | 製造工程が複雑 |

### 納期内訳

| 工程 | 期間（目安） |
|------|-------------|
| **データチェック** | 1〜2日 |
| **製版（新規）** | 3〜5日 |
| **印刷・ラミネート** | 5〜7日 |
| **製袋** | 5〜7日 |
| **検査・包装** | 2〜3日 |
| **配送** | 2〜3日 |
| **合計** | **18〜27日** |

### 納期短縮のポイント

| 方法 | 短縮可能日数 | 備考 |
|------|-------------|------|
| **既存版使用** | 3〜5日 | 製版工程を省略 |
| **簡易検査** | 1〜2日 | 検査工程を簡略化 |
| **優先製造** | 2〜3日 | 別途費用が発生する場合あり |

### コストの内訳

| 費用項目 | 比率 |
|---------|------|
| **原材料費** | 40〜50% |
| **製版費** | 10〜15% |
| **印刷・ラミネート費** | 15〜20% |
| **製袋費** | 15〜20% |
| **検査・包装費** | 5〜10% |
| **その他（管理費等）** | 5〜10% |

## ブログ記事での活用ポイント

### 製造工程解説記事の構成

#### 導入部

```markdown
「パッケージはどのように作られているのだろう？」
「納期がなぜ28日もかかるの？」

そんな疑問を持ったことはありませんか？

柔軟包装袋（フィルムパッケージ）の製造は、8つの工程を経て
完成します。各工程で熟練した技術者が品質管理を行い、
高品質な製品をお届けしています。

この記事では、パッケージの製造プロセスを分かりやすく解説します。
```

#### 工程フローの図解

```markdown
## 製造工程の全体像

```
原反準備 → 製版 → 印刷 → ラミネート → スリッティング → 製袋 → 検査 → 納品
```

各工程で、厳格な品質管理を行っています。
```

#### 納期の説明

```markdown
## 納期が28日かかる理由

製版（3〜5日）+ 印刷・ラミネート（5〜7日）+ 製袋（5〜7日）+ 検査・包装（2〜3日）

各工程で熟練した技術者が品質管理を行うため、
この期間が必要です。

ただし、既存版を使用する場合は、製版工程を省略できるため、
納期を短縮できます（最短21日）。
```
