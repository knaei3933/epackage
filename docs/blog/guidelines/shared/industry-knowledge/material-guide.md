## 素材概要

### 柔軟包装袋の素材構造

柔軟包装袋（フィルムパッケージ）は、複数の素材を貼り合わせた「ラミネートフィルム」で構成されています。

```
┌────────────────────────────────────┐
│         ラミネート構造例             │
├────────────────────────────────────┤
│  外側（印刷面）：PET（ポリエステル） │  ← 強度、印刷適性
│  中間層：AL（アルミ）またはVMPET    │  ← バリア性（遮光、防湿）
│  内側（密封面）：LLDPE（ポリエチレン）│  ← ヒートシール性
└────────────────────────────────────┘
```

### 素材選定の重要性

素材選定は、以下の要素に直接影響します。

| 要素 | 素材の影響 |
|------|----------|
| **保存期間** | バリア性が高いほど長期保存可能 |
| **内容物の品質** | 酸化、湿気、光からの保護 |
| **コスト** | 素材によって価格が異なる |
| **環境負荷** | リサイクル性、バイオマス素材等 |
| **法規制** | 食品衛生法、容器包装リサイクル法等 |

## 基本素材の種類

### 1. PET（ポリエステル）

#### 基本情報

| 項目 | 内容 |
|------|------|
| **正式名称** | ポリエチレンテレフタレート（Polyethylene Terephthalate） |
| **略称** | PET |
| **特徴** | 強度、透明性、印刷適性に優れる |
| **主な用途** | 外装材（印刷面） |

#### 特徴

**メリット**:
- **強度**: 引張強度が高く、破れにくい
- **印刷適性**: グラビア印刷に最適
- **透明性**: 透明性が高く、中身が見える
- **耐熱性**: 耐熱温度が高い（〜150℃）

**デメリット**:
- バリア性が低い（湿気、酸素透過）
- 単体での使用は不向き

#### 厚みと用途

| 厚み | 用途 |
|------|------|
| **12μm** | 一般的な包装 |
| **15μm** | 高強度が必要な場合 |
| **25μm** | 特に強度が必要な場合 |

---

### 2. AL（アルミ箔）

#### 基本情報

| 項目 | 内容 |
|------|------|
| **正式名称** | アルミニウム箔（Aluminum Foil） |
| **略称** | AL |
| **特徴** | 最強のバリア性（完全遮光、防湿、防酸） |
| **主な用途** | 中間層（バリア材） |

#### 特徴

**メリット**:
- **完全遮光**: 光を完全に遮断
- **防湿性**: 湿気を完全にシャットアウト
- **防酸化**: 酸素を完全に遮断
- **香気保持**: 香りを逃がさない

**デメリット**:
- 不透明（中身が見えない）
- コストが高い
- リサイクル性が低い（他素材との分離が困難）

#### 厚みと用途

| 厚み | 用途 |
|------|------|
| **7μm** | 一般的な包装 |
| **9μm** | 高バリア性が必要な場合 |

#### 適した商品

- コーヒー豆
- 香辛料
- 保存食
- 漬物
- 油物

---

### 3. VMPET（蒸着ポリエステル）

#### 基本情報

| 項目 | 内容 |
|------|------|
| **正式名称** | 真空蒸着ポリエステル（Vacuum Metallized PET） |
| **略称** | VMPET |
| **特徴** | PETにアルミニウムを蒸着した素材 |
| **主な用途** | 中間層（バリア材） |

#### 特徴

**メリット**:
- **バリア性**: ALに次ぐ高いバリア性
- **金属光沢**: 高級感のある外観
- **コスト**: ALより安価
- **軽量**: ALより薄く、軽量

**デメリット**:
- ALよりバリア性が劣る
- 光がわずかに透過する

#### 厚みと用途

| 厚み | 用途 |
|------|------|
| **12μm** | 一般的な包装 |
| **15μm** | 高バリア性が必要な場合 |

#### 適した商品

- 一般的な食品
- 菓子
- スナック
- ナッツ類

---

### 4. LLDPE（直鎖状低密度ポリエチレン）

#### 基本情報

| 項目 | 内容 |
|------|------|
| **正式名称** | Linear Low Density Polyethylene |
| **略称** | LLDPE |
| **特徴** | ヒートシール性に優れる |
| **主な用途** | 内装材（密封面） |

#### 特徴

**メリット**:
- **ヒートシール性**: 低温で確実にシール
- **強度**: 引裂強度が高い
- **透明性**: 透明性が高い
- **食品衛生法**: 食品衛生法適合

**デメリット**:
- 耐熱性が低い（〜100℃）
- バリア性が低い

#### 厚みと用途

| 厚み | 用途 |
|------|------|
| **50μm** | 一般的な包装 |
| **70μm** | 高強度が必要な場合 |
| **90μm** | 特に強度が必要な場合 |

---

### 5. CPP（無延伸ポリプロピレン）

#### 基本情報

| 項目 | 内容 |
|------|------|
| **正式名称** | Cast Polypropylene |
| **略称** | CPP |
| **特徴** | 耐熱性、透明性に優れる |
| **主な用途** | 内装材（密封面） |

#### 特徴

**メリット**:
- **耐熱性**: 耐熱温度が高い（〜130℃）
- **透明性**: 高い透明性
- **剛性**: LLDPEより硬い

**デメリット**:
- ヒートシール温度が高い
- 引裂強度が低い

#### 適した商品

- レトルト食品
- 煮沸消毒が必要な商品

---

### 6. NY（ナイロン）

#### 基本情報

| 項目 | 内容 |
|------|------|
| **正式名称** | ナイロン（Nylon） |
| **略称** | NY |
| **特徴** | 強度、バリア性に優れる |
| **主な用途** | 外装材または中間層 |

#### 特徴

**メリット**:
- **強度**: 引張強度、引裂強度が高い
- **バリア性**: 酸素バリア性が高い
- **耐ピンホール性**: ピンホールが発生しにくい

**デメリット**:
- 吸湿性（湿気を吸うとバリア性が低下）
- コストが高い

#### 適した商品

- レトルト食品
- 真空包装
- ボイル袋

## ラミネート構成

### 代表的なラミネート構成

#### 1. PET/AL/LLDPE（最強バリア）

| 層 | 素材 | 役割 |
|----|------|------|

## バリア性と保存性

### バリア性の指標

#### 1. 酸素透過度（Oxygen Transmission Rate: OTR）

**単位**: cc/㎡・day・atm  
**意味**: 1日あたりに透過する酸素量

| 素材 | OTR（cc/㎡・day・atm） | 評価 |
|------|----------------------|------|
| **AL（7μm）** | 0 | 完全遮断 |
| **VMPET（12μm）** | 1〜5 | 極めて高い |
| **NY（15μm）** | 20〜50 | 高い |
| **PET（12μm）** | 50〜100 | 中程度 |
| **LLDPE（70μm）** | 1,000〜2,000 | 低い |

#### 2. 水蒸気透過度（Water Vapor Transmission Rate: WVTR）

**単位**: g/㎡・day  
**意味**: 1日あたりに透過する水蒸気量

| 素材 | WVTR（g/㎡・day） | 評価 |
|------|-------------------|------|
| **AL（7μm）** | 0 | 完全遮断 |
| **VMPET（12μm）** | 1〜3 | 極めて高い |
| **NY（15μm）** | 50〜100 | 高い |
| **PET（12μm）** | 20〜40 | 中程度 |
| **LLDPE（70μm）** | 10〜20 | 中程度 |

### 保存期間の目安

| 素材構成 | 内容物 | 保存期間（目安） |
|---------|--------|----------------|
| **PET/AL/LLDPE** | コーヒー豆 | 12〜18ヶ月 |
| **PET/AL/LLDPE** | 保存食 | 24〜36ヶ月 |
| **PET/VMPET/LLDPE** | 菓子 | 6〜12ヶ月 |
| **PET/LLDPE** | パン | 1〜2週間 |
| **PET/LLDPE** | 冷凍食品 | 6〜12ヶ月 |

## 素材選定ガイド

### 素材選定のステップ

#### STEP1: 内容物の特性を確認

**質問事項**:
- [ ] 酸化しやすいか？（油分、香辛料等）
- [ ] 湿気に弱いか？（乾燥食品等）
- [ ] 光に敏感か？（コーヒー豆、ハーブ等）
- [ ] 香りを保つ必要があるか？
- [ ] 耐熱性が必要か？（レトルト、煮沸等）

#### STEP2: 保存期間を決定

| 保存期間 | 推奨素材構成 |
|---------|-------------|
| **1週間以内** | PET/LLDPE |
| **1ヶ月以内** | PET/LLDPE、PET/VMPET/LLDPE |
| **3〜6ヶ月** | PET/VMPET/LLDPE |
| **6ヶ月〜1年** | PET/AL/LLDPE、PET/VMPET/LLDPE |
| **1年以上** | PET/AL/LLDPE |

#### STEP3: 予算を考慮

| 素材構成 | 価格目安（150×180mm） |
|---------|---------------------|
| **PET/LLDPE** | ¥8〜15/枚 |
| **PET/VMPET/LLDPE** | ¥15〜25/枚 |
| **PET/AL/LLDPE** | ¥20〜35/枚 |
| **PET/AL/CPP** | ¥25〜40/枚 |

#### STEP4: 法規制を確認

**食品衛生法**:
- 食品に直接触れる素材は、食品衛生法に適合している必要があります
- Epackage Labの全素材は食品衛生法適合品を使用しています

**容器包装リサイクル法**:
- プラスチック製容器包装は、リサイクルの対象です
- 素材表示が必要です

### 内容物別素材選定表

| 内容物 | 推奨素材構成 | 理由 |
|--------|-------------|------|
| **コーヒー豆** | PET/AL/LLDPE | 酸化、湿気、光を完全に遮断 |
| **茶葉** | PET/AL/LLDPE | 香り保持、防湿 |
| **香辛料** | PET/AL/LLDPE | 香り保持、防湿 |
| **保存食** | PET/AL/LLDPE | 長期保存 |
| **ナッツ類** | PET/VMPET/LLDPE | 酸化防止 |
| **スナック菓子** | PET/VMPET/LLDPE | 酸化、湿気防止 |
| **クッキー** | PET/VMPET/LLDPE | 湿気防止 |
| **チョコレート** | PET/AL/LLDPE | 酸化、湿気、温度変化対応 |
| **和菓子** | PET/LLDPE | 保存期間が短いため標準でOK |
| **パン** | PET/LLDPE | 保存期間が短いため標準でOK |
| **冷凍食品** | PET/LLDPE | 標準でOK |
| **レトルト食品** | PET/AL/CPP | 耐熱性、高バリア性 |
| **惣菜** | PET/VMPET/LLDPE | 酸化防止 |
| **衣料品** | PET/LLDPE | 標準でOK |
| **化粧品** | PET/AL/LLDPE | 高級感、バリア性 |

### 特別な要件

#### 1. レトルト対応（耐熱性）

**推奨構成**: PET/AL/CPP  
**特徴**:
- 耐熱温度: 121℃（30分）
- 用途: レトルト食品、煮沸消毒品

#### 2. ボイル対応（耐熱性）

**推奨構成**: NY/PE  
**特徴**:
- 耐熱温度: 95〜100℃（30分）
- 用途: ボイル袋、真空包装

#### 3. 超高バリア（長期保存）

**推奨構成**: PET/AL/LLDPE  
**特徴**:
- 保存期間: 2〜3年
- 用途: 保存食、非常食

#### 4. エコ対応（環境負荷低減）

**推奨構成**: 単一素材（PP/PP、PE/PE）  
**特徴**:
- リサイクル性が高い
- コストが高い
- バリア性が低い
