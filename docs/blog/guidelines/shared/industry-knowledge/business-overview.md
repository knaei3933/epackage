## 事業ドメイン

### コア事業

Epackage Labは、以下の3つの柱で事業を展開しています。

#### 1. 柔軟包装袋のOEM製造

**取り扱い製品**:
- スタンドパウチ（自立袋）
- ガゼットパウチ（BOX型）
- 平袋（三シール袋）
- チューブ袋
- ラミネートチューブ

**特徴**:
- 小ロット対応（500枚〜）
- 国内最安値級の価格設定
- 短納期（最短21日、平均28日）
- 完全受注生産（OEM）

#### 2. パッケージコンサルティング

**提供価値**:
- 素材選定のサポート
- 規格サイズの最適化提案
- コストダウン提案
- 法規制対応支援（食品衛生法など）

#### 3. デジタルソリューション

**独自ツール**:
- スマート見積もりシステム（24時間対応）
- AI活用の仕様提案
- デザインテンプレート提供

## 競合優位性

### 4つの強み

#### 1. 小ロット対応（最大の強み）

**従来課題**:
- 既存メーカー：3,000〜5,000枚からの受注が一般的
- 起業・試作には障壁

**Epackage Labの解決**:
- 500枚から製造可能
- 試作・テストマーケティングに最適

**具体例**:
```
従業員: 「500枚からOKというのが決定的でした。
         新商品のテスト販売ができました。」
```

#### 2. 国内最安値級の価格

**価格競争力**:
- 大手メーカー比：20〜30%コストダウン
- 仲介業者を介さない直販モデル
- 効率的な製造プロセス

**価格目安**:
| 製品 | サイズ | 素材 | 価格（税抜） |
|------|--------|------|-------------|
| スタンドパウチ | 120x180mm | PET/AL/LLDPE | ¥15〜25/枚 |
| 平袋 | 100x150mm | PET/LLDPE | ¥8〜15/枚 |

#### 3. 短納期対応

**納期の目安**:
- 最短：21日
- 平均：28日
- 大手：45〜60日が一般的

**納期短縮の仕組み**:
- 効率的な生産スケジュール
- デジタルプロセスの活用
- 柔軟な対応力

#### 4. サポート体制

**充実したサポート**:
- スマート見積もり（24時間対応）
- 素材・規格選定のサポート
- デザインチェック（無料）
- 見本製作（有料）

## ブログ記事での活用ポイント

### 記事執筆時の留意点

#### 1. ターゲット読者を明確に

**良い例**:
```markdown
「健康食品を販売している方へ。
 サプリメントの鮮度を保ち、高級感を演出するパッケージをお探しではありませんか？」
```

**悪い例**:
```markdown
「パッケージをお探しの方へ」
```

#### 2. 具体的な数値を提示

**良い例**:
```markdown
「500枚から製造可能」「最短21日で納品」「業界平均より25%コストダウン」
```

**悪い例**:
```markdown
「小ロット対応」「短納期」「お手頃価格」
```

#### 3. 顧客の声を引用

**事例の提示**:
```markdown
「500枚からOKというのが決定的でした。
  新商品のテスト販売ができ、市場の反応を見ながら
  本格的な販売に踏み切れました。」
```

#### 4. 競合優位性を強調

**強調ポイント**:
- 小ロット対応（500枚〜）
- 国内最安値級（20〜30%コストダウン）
- 短納期（最短21日）
- 充実したサポート
