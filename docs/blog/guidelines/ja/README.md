## 記事テンプレート

`article-templates/` ディレクトリに各記事タイプのテンプレートがあります。

| テンプレート | 説明 | 文字数目安 |
|-------------|------|-----------|
| `product-intro.md` | 製品紹介記事 | 3,000〜5,000文字 |
| `practical-tips.md` | 実践的ノウハウ記事 | 4,000〜7,000文字 |
| `printing-tech.md` | 印刷技術記事 | 2,500〜4,000文字 |
| `customer-stories.md` | 導入事例記事 | 5,000〜10,000文字 |

### テンプレート使用方法

1. 対応するテンプレートをコピー
2. プレースホルダー（`{製品名}`など）を実際の内容に置き換え
3. 日本語の例を参考に内容を調整

## SEO戦略

`seo-strategy/` ディレクトリにGoogle検索最適化のガイドがあります。

| ガイド | 説明 |
|--------|------|
| `title-optimization.md` | タイトル最適化（30〜40文字） |
| `meta-description.md` | メタディスクリプション（120〜160文字） |
| `keyword-research.md` | キーワード戦略（日本市場） |

### タイトル構成の基本型

```markdown
【カテゴリー】メインキーワード｜サブタイトル（魅力を伝える）
```

**例**:
- 【製品紹介】スタンドパウチ｜陳列効果を最大化する自立袋
- 【実践的ノウハウ】パッケージデザインの基本｜売れるパッケージを作る3つのコツ

## 文化的ガイドライン

`cultural-guidelines/` ディレクトリに日本語ライティングのガイドがあります。

| ガイド | 説明 |
|--------|------|
| `tone-and-voice.md` | トーン＆ボイス（丁寧語） |
| `reader-expectations.md` | 日本語読者の期待値 |
| `faq.md` | よくある質問（日本語） |

### トーンの基本

- **丁寧語**: 「です」「ます」調を使用
- **親しみやすさ**: 専門用語は分かりやすく説明
- **具体性**: 実際の例やデータを提示

## 品質チェックリスト

記事公開前に以下のチェックを実施してください。

### コンテンツ品質

- [ ] リード文で読者の悩みに共感している
- [ ] 目次のリンクが正しく機能している
- [ ] 全ての表が正しく表示されている
- [ ] 関連記事のリンクが有効

### SEO品質

- [ ] メインタイトルにキーワードを含んでいる
- [ ] メタディスクリプションを設定済み（120〜160文字）
- [ ] H2見出しに関連キーワードを配置

### 技術品質

- [ ] Markdown形式で記述
- [ ] 画像パスが正しい
- [ ] 画像にalt属性がある
- [ ] CTAリンクが正しい
