  3. 契約の解除：第7条及び第8条に定める契約解除の場合。

  4. その他、本契約の条項に重大な違反があった場合。

  1. 甲乙いずれか一方が本契約を解除する場合は、書面で相手方に通知するものとする。
  2. 解除の通知には、解除の理由及び解除の効力が発生する日を明記するものとする。
  3. 契約の解除により生じた損害は、違反した一方が負担するものとする。
  4. 本契約の解除後、両者は誠実に清算手続きを行うものとする。

第9条 所有権の移転

  1. 対象製品の所有権は、残金の全額支払完了時に甲から乙に移転する。
  2. 検収前に製品に損傷等があった場合、その損害は乙の負担とする。
  3. 所有権が移転するまで、製品に関する一切の危険は甲が負担する。
  4. 乙は、所有権移転前に製品を第三者に譲渡することはできない。

第10条 品質保証

  1. 甲は、対象製品が仕様書に適合し、通常の使用条件下で___ヶ月間品質を保証する。
  2. 保証期間内に欠陥が発見された場合、甲は無償で修理、交換または返金する。
  3. 保証の対象は、材料および製造上の欠陥に限る。過失、改造または不適切な取り扱いによる損害は対象外とする。
  4. 保証期間経過後も、甲は合理的な範囲内で製品の保守・修理に協力するものとする。
  5. 製品の品質に関する紛争は、日本工業規格（JIS）または関連業界規格に準拠して解決する。

第11条 協議事項

  1. 本契約に定めのない事項については、甲乙は誠意をもって協議の上、決定するものとする。
  2. 協議によっても合意に至らない場合は、甲の本店所在地を管轄する地方裁判所を専属的合意管轄裁判所とする。
  3. 本契約の修正または変更は、書面で双方の合意があった場合にのみ有効とする。
  4. 通知は、書面により相手方の住所に送付することで効力を発する。

第12条 管轄裁判所

  1. 本契約に関して紛争が生じた場合、甲乙はまず協議によって解決を図るものとする。
  2. 協議によっても解決しない場合、本契約に係る一切の紛争は、甲の本店所在地を管轄する神戸地方裁判所を専属的合意管轄裁判所とする。
  3. 本契約の準拠法は日本法とする。
  4. 支払遅延、債務不履行、契約違反等に関する一切の請求は、前項の管轄裁判所において行うものとする。

第13条 契約の発行

  1. 本契約書は、正本を2通作成し、甲乙各1通を保有するものとする。
  2. 両者は、各々の正本を保管し、本契約の履行に努めるものとする。
  3. 本契約書の写しまたは電子データは、両者の合意があった場合にのみ証拠として有効とする。
  4. 本契約は、双方の署名捺印により効力を発する。
