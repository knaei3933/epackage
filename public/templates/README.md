# Template Files

This directory contains design template files for download.

## Template Files Structure

- AI files: Adobe Illustrator templates
- EPS files: Vector format templates
- PDF files: Design guidelines and specifications

## Categories

- **flat_3_side**: Three-side seal pouch templates
- **stand_up**: Stand-up pouch templates
- **box**: Box-type pouch templates
- **box_with_valve**: Box pouch with gas valve templates
- **spout_pouch**: Spout pouch templates
- **flat_with_zip**: Flat pouch with zipper templates
- **roll_film**: Roll film templates

Note: Template files should be added here as they become available.
